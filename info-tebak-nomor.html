9. Help!!!Info : itu yg soal nomor 4(angka 2 bukan 4 ya) ​


Jawaban

lim x→∞ √(2x² - 6x + c) - √(2x² - 3x + 4) = -3/4 √2

lim x→∞ (2x - 1) - √(4x² + 7x - 12) = -11/4

Pembahasan

lim x→∞ √(ax² + bx + c) - √(ax² + qx + r)

= (b - q)/(2√a)

lim x→∞ √(2x² - 6x + 5) - √(2x² - 3x + 4)

= (-6 - (-3)) / (2√2)

= -3/(2√2)

= -3/4 √2

lim x→∞ (2x - 1) - √(4x² + 7x - 12)

= lim x→∞ √(2x - 1)² - √(4x² + 7x - 12)

= lim x→∞ √(4x² - 4x + 1) - √(4x² + 7x - 12)

= (-4 - 7) / (2√4)

= -11/4
