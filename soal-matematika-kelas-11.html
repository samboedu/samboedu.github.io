13. soal matematika kelas 11 tentang kombinasi​


Jawaban:

a. n = 7

b. n = 5

c. n = 9

Penjelasan dengan langkah-langkah:

a.nC2 = 21

[tex] \frac{n!}{(n - 2)! \times 2!} = 21 \\ \frac{n \times (n - 1) \times (n - 2)!}{(n - 2)!} = 21 \times 2 \\ n(n - 1) = 42 \\ {n}^{2} - n = 42 \\ {n}^{2} - n - 42 = 0 \\ (n - 7)(n + 6) = 0 \\ n1 = 7 \: n2 = - 6[/tex]

n tidak mungkin negatif, maka n = 7

b. nC2 = 15

[tex] \frac{n!}{(n - 2)! \times 2!} = 15 \\ \frac{n \times (n - 1) \times (n - 2)!}{(n - 2)!} = 15 \times 2 \\ n(n - 1) = 30 \\ {n}^{2} - n = 30 \\ {n}^{2} - n - 30 = 0 \\ (n - 6)(n + 5) = 0 \\ n1 = 6 \: n2 = - 5[/tex]

n tidak mungkin negatif, maka n = 6

c. (n+1)C1 = 10

[tex] \frac{(n + 1)!}{n! \times 1!} = 10 \\ \frac{(n + 1) \times n!}{n!} = 10 \\ n + 1 = 10 \\ n = 10 - 1 \\ n = 9[/tex]

n = 9

8. [tex]\int 3x^2 \sqrt{2x^3+5} [/tex]
[tex]\textit{Let}\,\, u = 2x^3 + 5 [/tex]
[tex] \frac{du}{dx} = 6x^2 [/tex]
[tex] \frac{du}{6x^2} = dx [/tex]
[tex] \int 3x^2 \sqrt{2x^3+5} = \int 3x^2 \sqrt{u}\frac{du}{6x^2} [/tex]
[tex] \int 3x^2 \sqrt{2x^3+5} = \frac{1}{2} \int \sqrt{u}\, du [/tex]
[tex] \int 3x^2 \sqrt{2x^3+5} = \frac{1}{2}\cdot\frac{2}{3} u^{\frac{3}{2}}+C [/tex]
[tex] \int 3x^2 \sqrt{2x^3+5} = \frac{1}{3}\sqrt{(2x^3+5)^3} + C[/tex]

24. Soal matriks matematika peminatan kelas 11


Jawab :

Berdasarkan matriks tersebut, diperoleh persamaan :

• a + 3 = 5 - a
a + a = 5 - 3
2a = 2
a = 2/2
a = 1

• 8 + b = 4 - b
b + b = 4 - 8
2b = -4
b = -4/2
b = -2

• -1 + c = c - c
-1 + c = 0
c = 0 + 1
c = 1

• d + (-9) = -13 - d
d - 9 = -13 - d
d + d = -13 + 9
2d = -4
d = -4/2
d = -2

Maka, nilai dari a+b+c+d adalah

a + b + c + d = 1 + (-2) + 1 + (-2)
a + b + c + d = 1 - 2 + 1 - 2
a + b + c + d = -1 - 1 = -2
