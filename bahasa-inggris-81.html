mtk = 86

b.ingg = ?

rata² = 84

rata² = jumlah semua nilai / frekuensi

84 = (81 + 89 + 78 + 86 + x) / 5

84 x 5 = 334 + x

x = 420 - 334

x = 86

nilai b.inggris = 86

Natsir mendapat nilai 81 untuk IPA, nilai 89 untuk IPS, nilai 78 untuk bahasa Indonesia, dan nilai 86 untuk Matematika. Bila Natsir ingin mendapatkan rata-rata nilainya sebesar 84. Maka nilai yang harus diperoleh untuk pelajaran Bahasa Inggris adalah 86.

Pendahuluan :

[tex] \bf \blacktriangleright Pengertian :[/tex]

Pengolahan Data adalah suatu tindakan memanipulasi data ke dalam bentuk yang informatif agar memudahkan untuk dibaca oleh pengguna data.

Penyajian Data adalah kegiatan yang dilakukan setelah pengolahan data. Penyajian data terdiri dari beberapa jenis, seperti bentuk tabel, diagram batang, diagram lingkaran, diagram garis, dll.

[tex] \\[/tex]

[tex] \bf \blacktriangleright Jenis-jenis~Pengolahan~Data :[/tex]

[tex] \rm 1. Rata-rata (Mean)[/tex]

Rata-rata atau mean adalah sekumpulan data yang mewakili suatu bilangan. Rata-rata dapat dihitung menggunakan rumus berikut :

[tex] \boxed{\bf \overline{X} = \frac{Jumlah~data}{Banyak~data}}[/tex]

Rumus rata-rata gabungan :

[tex] \rm \overline{X}_{Gabungan} = \frac{n_1\overline{X}_1 + n_2\overline{X}_2}{n_1+n_2}[/tex]

dimana :

[tex] \rm \overline{X}[/tex] = rata-rata

[tex] \rm \overline{X}_1[/tex] = rata-rata pertama

[tex] \rm \overline{X}_2[/tex] = rata-rata kedua

[tex] \rm n_1[/tex] = banyak data pertama

[tex] \rm n_2[/tex] = banyak data kedua

[tex] \\[/tex]

[tex] \rm 2. Modus[/tex]

Modus dapat ditentukan dengan melihat data yang frekuensinya paling banyak atau yang paling sering muncul.

[tex] \\[/tex]

[tex] \rm 3. Median[/tex]

Median adalah nilai tengah dari data yang telah diurutkan dari terkecil hingga terbesar.

• Untuk data ganjil :

[tex] \boxed{\bf Median = data~tepat~di~tengah}[/tex]

• Untuk data gemap :

[tex] \boxed{\bf Median = \frac{jumlah~dua~data~di~tengah}{2}}[/tex]

Pembahasan :

Diketahui :

IPA = 81 IPS = 89Bahasa Indonesia = 78 Matematika = 86

Ditanya :

Agar mendapat nilai rata-rata 84, nilai yang harus diperoleh untuk pelajaran Bahasa Inggris?

Jawab :

Misalkan Bahasa Inggris adalah "a" :

[tex] \rm \frac{Jumlah~Data}{Banyak~Data} = Rata-rata[/tex]

[tex] \rm \frac{81+89+78+86+a}{5} = 84[/tex]

[tex] \rm 334+a = 84 \times 5[/tex]

[tex] \rm 334+a = 420[/tex]

[tex] \rm a = 420-334[/tex]

[tex] \bf a = 86[/tex]

Kesimpulan :

Jadi, nilai Bahasa Inggris yang harus diperoleh adalah 86.

   IPA = 81
   IPS = 89
B. Ind = 78
   Mat = 86

Mean = 84


Ditanyakan ⇒ Nilai B. Ing?


Penyelesaian ↓

Misalkan nilai B. Ing = x

  Mean = Jumlah data : Banyak data
      84 = (81 + 89 + 78 + 86 + x) : 5
      84 = (334 + x) : 5
84 × 5 = 334 + x
    420 = 334 + x
        x = 420 - 334
        x = 86

Jadi nilai yang harus diperoleh untuk Bahasa Inggris adalah 86.81 => IPA
89 => IPS
78 => B.indonesia
86 => MTK
a   => B.inggris
banyak plajaran = 5
rata-rata = (IPA +IPS + BINDO + MTK + B.INGGRS)/ 5
84      = ( 81+89+78+86+ a) / 5
84 x 5 = 334 + a
420     = a
420-334 = a
   86    = a


jawabnnya 86
