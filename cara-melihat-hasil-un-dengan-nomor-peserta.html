7. Jika Un = 3n + 3 dan Un = 78 maka nilai n adalah...? Tolong Berserta Caranya. Terima Kasih.


Jawabannya: 25

78=3n + 3
3n=75
n=25dik, Un = 3n + 3
Un = 78
dit, n?
penyelesaian :
3n + 3 = 78
3n + 3 - 3 = 78 - 3
3n = 75
3n × (⅓) = 75 × (⅓)
n = 25
jadi, nilai n = 25

20. Carilah U25 dari barisan dibawah ini beserta caranya: a. 5,10,15,....Un b. 3,6,12,......Un


Un = a + (n - 1) × b

a. 5, 10, 15, ...

U25 = 5 + (25 - 1) × 5

U25 = 5 + (24) × 5

U25 = 5 + 120

U25 = 125

b. 3, 6, 12, ...

U25 = 3 + (25 - 1) × 2

U25 = 3 + (24) × 2

U25 = 3 + 48

U25 = 51

29. tentukan rumus Un dari 1,9,17,25 beserta caranya​


Jawaban:

No. 1, 2 dan 3 semuanya merupkan barisan aritmatika, maka berlaku

suku pertama = U1

beda = b = U_{n+1} -U_{n} = U_{n+2} - U_{n+1} = ......

suku ke-n = Un = U1 + (n - 1) b

1.

U1 = 1

b = 9-1 = 17-9 = ..... = 8

Un = 1 + (n - 1) b = 1 + 8n - 8

Un = 8n - 7

2.

U1 = 6

b = 18-6 = 30-18 = ... = 12

Un = 6 + (n - 1) 12 = 6 + 12n - 12

Un = 12n - 6

3.

U1 = 5

b = 11-5 = 17-11= ... = 6

Un = 5 + (n - 1) 6 = 5 + 6n - 6

Un = 6n - 1
