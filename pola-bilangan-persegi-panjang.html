20. contoh pola bilangan persegi,persegi panjang,dan pola bilangan segitiga​


Jawaban:

Pola bilangan persegi: 1, 4, 9, 16, 25, ..., n²

Pola bilangan persegi panjang: 2, 6, 12, 20, 30, ..., n(n+1)

Pola bilangan segitiga: 1, 3, 6, 10, 15, ..., (n(n+1))/2

Penjelasan dengan langkah-langkah:

Pola bilangan persegi → Un=n²

U1=1²=1

U2=2²=4

U3=3²=9

U4=4²=16

U5=5²=25

--------------------------------------------------------------

Pola bilangan persegi panjang → Un=n(n+1)

U1=1(1+1)=1×2=2

U2=2(2+1)=2×3=6

U3=3(3+1)=3×4=12

U4=4(4+1)=4×5=20

U5=5(5+1)=5×6=30

--------------------------------------------------------------

Pola bilangan segitiga → Un=(n(n+1))/2

(tanda per = "/")

U1=(1(1+1))/2=(1×2)/2=2/2=1

U2=(2(2+1)/2=(2×3)/2=6/2=3

U3=(3(3+1)/2=(3×4)/2=12/2=6

U4=(4(4+1)/2=(4×5)/2=20/2=10

U5=(5(5+1)/2=(5×6)/2=30/2=15
