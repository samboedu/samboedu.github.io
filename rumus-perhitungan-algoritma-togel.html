14. buatlah algoritma dan program untuk menghitung nilai uang dalam bentuk Dollar, Pesso dan Yen. Adapun inputannya adalah Rupiah dengan rumus.


Algoritma dalam pemrograman adalah sebuah susunan langkah-langkah yang diperlukan oleh sebuah komputer untuk memecahkan masalah atau mencapai suatu tujuan. Algoritma terdiri dari logika kita tentang cara memecahkan masalah pemrograman yang akan dibuat.


Algoritma dimulai dari kondisi awal dan input awal, dan berlanjut ke proses komputasi hingga menghasilkan output yang diinginkan.


Contoh algoritma untuk menghitung nilai uang dalam bentuk Dollar, Pesso, dan Yen adalah sebagai berikut

1. Mulai

2. Masukkan nominal rupiah

3. Jika Dollar, maka nilai = rupiah / 10000

4. Jika Pesso, maka nilai = rupiah / 3000

5. Jika Yen, maka nilai = rupiah / 5000

6. Selesai

Contoh program untuk menghitung nilai uang dalam bentuk Dollar, Pesso dan Yen adalah sebagai berikut

(menggunakan bahasa pemrograman C++ )

#include <iostream>

using namespace std;

main(){

float a,b,c,d,e,f,g;

cout<<"masukan Nominal Uang rupiah anda\t="<<" Rp. ";cin>>a;

b=a/10000;

c=a/3000;

d=a/5000;

cout<<"\nRupiah ke Dollar\t\t= "<<b<<endl;

cout<<"\nRupiah ke Pesso\t= "<<c<<endl;

cout<<"\nRupiah ke Yen\t= "<<d<<endl;

}
