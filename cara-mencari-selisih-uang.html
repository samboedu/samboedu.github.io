Misal :
Ahmad = x
Beni = y
Citra = z

x = 2/3y
y = 2/7z
x = 12.000

Masukkan x,
x = 2/3y
12.000 = 2/3y
y = 12.000 : 2/3
y = 12.000 x 3/2
y = 18.000

Masukkan y,
y = 2/7z
18.000 = 2/7z
z = 18.000 : 2/7
z = 18.000 x 7/2
z = 63.000

Sehingga,
z - y = 63.000 - 18.000 = [tex] \boxed{45.000} [/tex]

Jadi, selisih uang Beni dan Citra adalah Rp 45.000
