28. contoh dialog offering help if clause+reminder


Kelas: SMA
Pelajaran: English
Kategori: conditional clause
Kata kunci: if clause, reminder, offering
Pembahasan:

In English grammar there are several function for conditional clause, they are
- as a reminder: If you wanna be succesfull, you'll have to try harder.
- as a advice: if you have a sudden stomachache, you should rub it with cajuput oil
- as an instruction: if you have a problem with it, ask your teacher now.
- to state the general facts: If you put the ice on the heat, it melts.
- to state the possible act: if it doesn't rain today, we'll go to the beach.
