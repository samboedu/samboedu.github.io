= 3 + 11/18
= 3 + 1/(18/11)
= 3 + 1/(1 7/11)
= 3 + 1/(1 + 7/11)
= 3 + 1/(1 + 1/(11/7))
= 3 + 1/(1 + 1/(1 4/7))
= 3 + 1/(1 + 1/(1 + 4/7))
= 3 + 1/(1 + 1/(1 + 1/(7/4)))
= 3 + 1/(x + 1/(y + 1/z))
