4. Cara menghitung 2 log 8


2log8 = 2log2^3 = 3      logaritma adalah sebuah fungsi pencari nilai bilangan pangkat berdasarkan basis tertentu. basis tersebut merupakan bilangan yang dipangkatkan oleh bilangan pangkat dalam ekponensial.

contoh :
jika dalam eksponen,

[tex]a^b=c[/tex]

di logaritma, akan seperti ini

[tex]^2\log c = b[/tex] (berdasarkan rumus eksponensial di atas)

dimana,
- a = bilangan yang dipangkatkan (dalam logaritma disebut basis)
- b = bilangan pangkat
- c = bilangan hasil perpangkatan (dalam logaritma disebut bilangan yang dilogaritmakan dengan basis)

kita kembali ke masalah [tex]^2 \log 8[/tex]
[tex]^2\log 8\\=^2\log 2^3\\=3[/tex]

jadi, hasilnya adalah 3

sekian...

23. Bagaimana caranya menghitung nilai 64 log 4


caranya menghitung nilai 64 log 4

Logaritma itu merupakan kebalikan dari perpangkatan

ᵃlog b = n dalam bentuk perpangkatan → aⁿ = b

sifat" logaritma

1) ᵃlog (b.c) = ᵃlog b + ᵃlog c

2) ᵃlog b x ᵇlog c = ᵃlog c

3) ᵃlog (ᵇ/ₓ) = ᵃlog b - ᵃlog x

4) ᵃlog b = 1/ᵇlog a

5) ᵃlog (b/c) = - ᵃlog (c/b)

6) ᵃlog bⁿ = n. ᵃlog b

7) ᵃ²log b = ¹/₂. ᵃlog b

8) ᵃlog aⁿ = n

9) ᵃlog a = 1

Pembahasan

⁶⁴log 4

= ⁴^³log 4¹

= ¹/₃ ⁴log 4

= ¹/₃

tanda ^ = pangkat
