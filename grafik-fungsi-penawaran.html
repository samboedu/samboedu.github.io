5. pada saat harga barang Rp 1000,00 jumlah barang yang ditawarkan 20 unit, kemudian harga naik menjadi Rp 2.000,00 jumlah penawaran naik menjadi 30 unit. tentukan berikuta. Fungsi penawaranb. grafik fungsi penawaran


Jawaban:

P1 = 1000

P2 = 2000

Q1 = 20

Q2 = 30

(P - P1) / (P2 - P1) = (Q - Q1) / (Q2 - Q1)

(P - 1000) / (2000 - 1000) = (Q - 20) / (30 - 20)

(P - 1000) / 1000 = (Q - 20) / 10

10 x (P - 1000) = 1000 (Q - 20)

10 P - 10.000 = 1000Q - 20.000

1000Q = 10P - 10.000 + 20.000

1000Q = 10P + 10.000

Q = (10P + 10.000) / 1000

Q = 0,01P + 1

Q = 1 + 0,01P

Jadi, fungsi penawarannya Qs = 1 + 0,01P

Diketahui : fungsi penawaran Ps = -75 + 3Q²

Ditanya : gambarkan grafik nya

Penyelesaian :

Dengan membuat tabel koordinat terlebih dahulu

Untuk Q = -3 maka Ps = -75 + 3 (-3)^2 = -75 + 27 = -48 koordinatnya (-3, -48)

Untuk Q = -2 maka Ps = -75 + 3 (-2)^2 = -75 + 12 = -63 koordinatnya (-2, -63)

Untuk Q = -1 maka Ps = -75 + 3 (-1)^2 = -75 + 3 = -72 koordinatnya (-1, -72)

Untuk Q = 0 maka Ps = -75 + 3 (0) = -75 koordinatnya (0, -75)

Untuk Q = 1 maka Ps = -75 + 3 (1)^2 = -75 + 3 = -72 koordinatnya (1, -72)

Untuk Q = 2 maka Ps = -75 + 3 (2)^2 = -75 + 12 = -63 koordinatnya (2, -63)

Untuk Q = 3 maka Ps = -75 + 3 (3)^2 = -75 + 27 = -48 koordinatnya (3, -48)

Untuk Q = 4 maka Ps = -75 + 3 (4)^2 = -75 + 48 = -27 koordinatnya (4, -27)

Untuk Q = 5 maka Ps = -75 + 3 (5)^2 = -75 + 75 = 0 koordinatnya (5, 0)

19. Jika harga spidol 65 maka ada 125 spidol yang ditawarkan dipasar,apabila harganya 75 maka ada 145 spidol.berapa fungsi penawaran?dan buat grafiknya !


Jawaban: Fungsi penawarannya adalah 2P – 5 = Q. Grafik penawaran bisa dilihat dalam lampiran. Fungsi penawaran adalah fungsi yang menunjukkan hubungan antara harga (P) dengan jumlah barang (Q) yang ditawarkan.

Penjelasan:

Bentuk umum fungsi penawaran yaitu:

Qs = a + bPs atau Ps = ¹/bQs – a/b

dimana :

- a & b : konstanta dimana b harus bernilai positif

- Ps : harga barang yang ditawarkan

- Qs : jumlah barang yang ditawarkan

Fungsi penawaran bisa dicari menggunakan rumus :

P – Po / P1 – Po = Q – Qo / Q1 – Qo

Diketahui Po = 65, P1 = 75, Qo = 125, Q1 = 145.

P – Po / P1 – Po = Q – Qo / Q1 – Qo

P – 65 / 75 – 65 = Q – 125 / 145 – 125

P – 65/10 = Q – 125/20

20(P – 65) = 10(Q – 125)

20P – 1.300 = 10Q – 1.250

20P – 1.300 + 1.250 = 10Q

20P – 50 = 10Q ➡ 2P – 5 = Q

Diperoleh fungsi persamaannya adalah 2P – 5 = Q.

24. Diketahui fungsi penawaran Ps=-75+3Q². bagaimana cara menggambar grafiknya?


langkah sketsa grafik nya sama dgn fungsi kuadrat yg y=f(x)

yang perlu diingat adalah kuantitas (Q) bernilai positif yg dipakai di fungsi penawaran
atau grafik di sumbu mendatar atau kuantitas (Q) dimulai dari angka 0 ke bilangan positif

kita mulai ya
perpotongan dgn sumbu mendatar atau Q terjadi saat Ps=0
[tex]ps = - 75 + 3 {q}^{3} \\ 0 = - 75 + 3 {q}^{2} \\ 3 {q}^{2} = 75 \\ {q}^{2} = \frac{75}{3} = 25 \\ q = \sqrt{25} \\ q1 = 5 \\ q2 = - 5[/tex]
mengingat kuantitas barang (Q) tidak bernilai negatif, maka perpotongan dgn sumbu mendatar atau Q pada Q=5

selanjutnya mencari perpotongan dng sumbu vertikal atau P (price), dimana terjadi saat nilai Q=0

[tex]ps = - 75 + 3 {q}^{2} \\ ps = - 75 + 3. {0}^{2} \\ ps = - 75 + 0 = - 75[/tex]
kita dapat perpotongan dgn sumbu vertikal pada P=-75

selanjutnya kita tempatkan kedua titik perpotongan ke grafik dan sketsakan kurva kuadratnya

28. bila persamaan fungsi permintan dan fungsi penawaran masing-masing adakag Qd=140-20P dan Qs=-40+20P dan kedua fungsi tersebut merupakan fungsi permintaan dan penawaran terhadap X maka carilah tingkat harga dan kuantitas keseimbangan dan gambarkan grafiknya.​ gimana dengan grafiknya ?


Jawaban:

Tingkat harga dan kuantitas keseimbangan adalah P = 4,5 dan Q = 50. Dibuat kurva dari fungsi permintaan dan fungsi penawaran. Titik keseimbangan adalah titik potong kedua kurva.

Penjelasan:

Untuk mencari titik keseimbangan, Qd = Qs.

140 - 20P = -40 + 20P

40P = 180

P = 4,5

Substitusi nilai P ke salah satu fungsi, misalnya ke Qd.

Q = 140 - 20(4,5)

Q = 140 - 90

Q = 50

Jadi, pada titik keseimbangan (ekuilibrium) tingkat harga P = 4,5 dan kuantitas Q = 50.
