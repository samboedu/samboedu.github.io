1. Menyederhanakan Pecahan Aljabar 1. Bentuk sederhana x² + 3x : x²– 9 adalah 2. Bentuk sederhana x²+ 3x + 2 : x²– 4.adalah 3. Bentuk sederhana 4x² – 9 : 2x² – x – 3 adalah 4. Bentuk sederhana dari x² – 25 : x² + x – 20 adalah 5. Bentuk sederhana dari x² – 2x – 3 : 2x² + 5x +3 adalah


1). x² + 3x : x²– 9
= x(x+3) / (x-3)(x+3)
(x+3) kita coret maka,
= x / x-3

2). x²+ 3x + 2 : x²– 4
= (x+2)(x+1) / (x-2)(x+2)
(x+2) kita coret, maka
= x+1 / x-2

3). 4x² – 9 : 2x² – x – 3
= (2x-3)(2x+3) / (2x-3)(x+1)
(2x-3) kita coret, maka
= 2x+3 / x+1

4). x² – 25 : x² + x – 20
= (x-5)(x+5) / (x+5)(x-4)
(x+5) kita coret, maka
= x-5 / x-4

5). x² – 2x – 3 : 2x² + 5x +3
= (x-3)(x+1) / (2x+3)(x+1)
(x+1) kita coret maka,
= x-3 / 2x+3

Bilangan berpangkat juga dikenal dengan istilah bilangan eksponen. Suatu bilangan bulat dapat dinyatakan dalam bentuk perpangkatan. Pada dasarnya perpangkatan bilangan adalah perkalian berulang untuk bilangan yang sama.

-

Sifat-sifat perpangkatan untuk untuk a bilangan bulat dan n bilangan bulat positif dinyatakan sebagai berikut.

[tex]a) {a}^{m} \times {a}^{n} = {a}^{m + n} [/tex]

[tex]b) \frac{ {a}^{m} }{ {a}^{n} } = {a}^{m - n} [/tex]

[tex]c) \: ( {a}^{m} {)}^{n} = {a}^{m \times n} [/tex]

[tex]d) \:( a \times b {)}^{m} = {a}^{m} \times {b}^{m} [/tex]

-

Bentuk sederhana dari x⁴ × (x⁶ : x³)

[tex] \sf \: {x}^{4} \times ( {x}^{6} : {x}^{3}) [/tex]

[tex] = \sf \: {x}^{4} \times( {x}^{6 - 3} )[/tex]

[tex] = \sf \: {x}^{4} \times {x}^{3} [/tex]

[tex] = \sf \: {x}^{4 + 3} [/tex]

30. tulisan bentuk w3 x w4 ke dalam bentuk perpangkatan paling sederhana. berapakah hasilnya? Apakah kamu juga dapat menyederhanakan bentuk w3 x n4?


Berdasarkan soal diatas diketahui: w^3, w^4, n^4, dan ditanya :

Tuliskan bentuk w^3 x w^4 ke dalam bentuk perpangkatan paling sederhana. berapakah hasilnya? Apakah kamu juga dapat menyederhanakan bentuk w^3 x n^4?


Jawab:

Untuk soal pertama yakni W³ × W⁴ bisa ditulis dalam bentuk pangkat paling sederhana karena angka induknya nya sama-sama W sehingga ketika dikalikan hasilnya menjadi:

= W^7

Karena angka induk nya sama maka pangkatnya hanya perlu ditambah pangkatnya jika operasinya itu perkalian, dan dikurangi pangkatnya jika ia merupakan operasi pembagian


Sedangkan untuk soal kedua yakni:

W³ × N ⁴ tidak bisa disederhanakan , karena angka induknya tidak sama sehingga kita tidak bisa mengutak-atik persamaan tersebut.
