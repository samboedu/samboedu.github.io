BC2 = 9 + 16

BC2 = 25

BC = √25

BC = 5 cm

K = AB + BC + CD + AD

K = 7 cm + 5 cm + 2 cm + 4 cm

K = 18 cm

L = 2 x luas alas + keliling x tinggi

L = (2 x La) + (K x BF)

L = (2 x 14 cm2) + (18 cm x 10 cm)

L = 28 cm2 + 180 cm2

2 π r t =  40

2 π r (10)=  440

π r = 22

r  =  22x 7/22

r=  7

Luas  tabung =  2 luas alas +  selimut  tabung

= 2 π r ² +  440

= 2 .22/7. 7² + 440

= 308 +  440

= 748 cm²

Luas permukaan tabung

=======================

2 x π x r x t = 440

2 x [tex]\frac{22}{7}[/tex] x r x 10 = 440

[tex]\frac{44}{7}[/tex] x 10 x r = 440

[tex]\frac{440}{7}[/tex] x r = 440

r = 440 : 440/7

r = 7

Luas permukaan

= 2 x π x r x ( r + t )

= 2 x 22/7 x 7 x ( 7 + 10 )

= 44/7 x 7 x 17
