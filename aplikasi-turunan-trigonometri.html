fungsi ini mengandung perkalian fungsi, maka dipecahkan dengsn skema

y' = u'v + uv'

[tex]u = \cot(x) \\ u' = - { \csc }^{2} (x)[/tex]

[tex]v = \csc(x) \\ v' = - \csc(x) \cos(x) [/tex]
maka
[tex]y' = u'v + uv' \\ = - \csc^{2} (x) \times \csc(x) + \cot(x) \times ( - \csc(x) \times \cos(x) ) \\ = - \csc^{3} (x) - \cot(x) \times ( \csc(x) \times \cos(x) \\ = - \csc^{3} (x) - \frac{ \cos(x) }{ \sin(x) } \times \frac{1}{ \sin(x) } \times \cos(x) \\ = - \csc^{3} (x) - \ \cot^{2} (x) [/tex]
NO. 2

[tex]y = \tan( \sqrt{ {x}^{2} + 8x} ) [/tex]
turunan fungsi ini menggunakan turunan bertingkat dengan skema
[tex] \frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dv} \times \frac{dv}{dx} [/tex]
[tex] \frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dv} \times \frac{dv}{dx}[/tex]

misal
[tex]v = {x}^{2} + 8x \\ \frac{dv}{dx} = 2x + 8[/tex]
dan
[tex]u = \sqrt{v} = {v}^{ \frac{1}{2} } \\ \frac{du}{dv} = \frac{1}{2} {v}^{ - \frac{1}{2} } [/tex]

dan
[tex]y = \tan(u) \\ \frac{dy}{du} = \sec^{2} u[/tex]
dengan skema di atas,substitusikan
[tex] \frac{dy}{dx} = \sec^{2} u \times \frac{1}{2} {v}^{ - \frac{1}{2} } \times (2x + 8) \\ = \sec^{2} \sqrt{ {x}^{2} + 8x} \times \frac{1}{2 \sqrt{ {x}^{2} + 8x} } \times (2x + 8) \\ = \frac{x + 4}{ \sqrt{ {x}^{2} + 8x} } \times \sec^{2} \sqrt{ {x}^{2} + 8x} [/tex]

22. Pengertian turunan aljabar, turunan trigonometri, turunan fungsi apa?


a. Turunan aljabar adalah perluasan dari materi limit fungsi. 

b. Turunan trigonometri adalah persamaan turunan yang melibatkan fungsi-fungsi trigonometri seperti sin, cos, tan, cot, sec dan cosec.

c. Turunan fungsi adalah fungsi lain dari suatu fungsi sebelumnya, misalnya fungsi f menjadi f' yang mempunyai nilai tidak beraturan.

28. contoh soal aplikasi trigonometri


Jika α, β, dan γ adalah sudut-sudut dalam segitiga ABC, tunjukkanlah bahwa : a. sin (β + γ) = sin α b. cos (β + γ) = -cos α c. tan (β + γ) = -tan α Pembahasan Ingat bahwa dalam segitiga jumlah sudutnya sam dengan 180o, sehingga berlaku : α + β + γ = 180o , → β + γ = 180o - α. sin (β + γ) = sin α ⇒ sin (180o - α) = sin α ⇒ sin α = sin α Terbukti. cos (β + γ) = -cos α ⇒ cos (180o - α) = -cos α ⇒ -cos α = -cos α Terbukti. tan (β + γ) = -tan α ⇒ tan (180o - α) = -tan α ⇒ -tan α = -tan α Terbukti.
