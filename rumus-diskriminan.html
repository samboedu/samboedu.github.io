11. Quiz Nilai Diskriminan : → 9x² + 5x - 2 Nb : Pake Cara + Rumus​


Jawaban:

9x² + 5x - 2 = 0

a = 9

b = 5

c = (-2)

Diskriminan : b² - 4ac

= b² - 4ac

= 5² - 4(9)(-2)

= (5 x 5) - (4 x 9 x (-2))

= 25 - (36 x (-2)

= 25 - (-72)

= 25 + 72

= 97

Rumus dari Diskriminan :

= b² - 4ac

= (b x b) - 4(a)(c)

= (b x b) - (4 x a x c)

= ... - ...

= Hasil Diskriminan
