3x² + 7x + 2 = 0

[tex]3 {x}^{2} + 6x + x + 2 = 0 \\ \\ 3x(x + 2) + (x + 2) = 0 \\ \\ (3x + 1)(x + 2) = 0 \\ \\ 3x + 1 = 0 \: \: \: atau \: \: \: x + 2 = 0 \\ \\ x = - \frac{1}{3} \: \: \: atau \: \: \: x = - 2 \\ \\ x_{1} = - \frac{1}{3} \: \: \: atau \: \: \: x_{2} = - 2 \\ \\ \\ \\ [/tex]

11. Diketahui persamaan kuadrat x²-x+3=0 dengan akar x1 dan x2 Bentuklah persamaan kuadrat baru yang akar-akarnya kuadrat dari akar-akar persamaan kuadrat di atas.


Persamaan kuadrat x2-x+3 = 0 mempunyai akar" x1 dan x2. Persamaan kuadrat baru yang akar akarnya 3x1+1 dan 3x2+1 adalah ax²+bx+c = 0 nilai 2a+b+c adalah 38

Pembahasan

Persamaan kuadrat adalah persamaan yang memiliki 1 variabel dengan pangkat tertinggi adalah 2

Bentuk umum

ax² + bx + c = 0

Rumus-rumus dalam persamaan kuadrat

x1 + x2 = -b/a

x1 . x2 = c/a

Persamaan baru

x² - (x1 + x2)x + (x1 . x2)

••••••••••••••

Diketahui

x² - x + 3 = 0

a = 1

b = -1

c = 3

Akar persamaan baru

3x1 + 1

3x2 + 1

Ditanya

Nilai 2a + b + c pada persamaan baru

Langkah cara

Untuk membuat persamaan baru, kita bisa menggunakan rumus yang sudah kita bahas diatas sebelumnya. Ada dua cara untuk mendapatkan persamaan baru, yaitu

Dengan mencari nilai x1 dan x2 persamaan pertama

Dengan mencari nilai-nilai yang dibutuhkan seperti x1+x2 dan x1.x2

Kita akan menggunakan cara kedua.

kita cari nilai x1 + x2, tapi kita masukkan dulu nilai x1 dan x2 persamaan baru seperti yang diketahui

x1 + x2 = (3x1 + 1) + (3x2 + 1)

x1 + x2 = 3x1 + 3x2 + 1 + 1

x1 + x2 = 3(x1 + x2) + 2

x1 + x2 = 3(1/a) + 2

x1 + x2 = 3(1/1) +2

x1 + x2 = 3 + 2

x1 + x2 = 5

Jadi, nilai x1 + x2 pada persamaan baru adalah 5

Next, kita bakalan mencari nilai x1 . x2 dengan rumus yang sudah kita bahas.

Nilai x1 . x2 persaaman baru

= (3x1 + 1)(3x2 + 1)

= 9x1.x2 + 3x1 + 3x2 + 1

= 9(x1 . x2) + 3(x1 + x2) + 1

= 9(c/a) + 3(-b/a) + 1

= 9(3/1) + 3(1/1) + 1

= 27 + 3 + 1

= 41

Jadi, nilai x1 . x2 persamaan baru adalah 41

Di langkah terakhir ini, kita tinggal memasukkan nilai yang sudah diketahui kedalam rumus untuk mencari persamaan baru.Persamaan baru

= x² - (x1 + x2)x + (x1 . x2)

= x² - (5)x + 41

= x² - 5x + 41

Yey! Sudah diketahui persamaan barunya, yaitu x² - 5x + 41 dengan

a = 1

b = -5

c = 41

The last step

Kita belum selesai gaes, karena yang ditanyakan adalah nilai 2a + b + c. Ayok kita kerjakan!

2a + b + c = 2(1) - 5 + 41

2a + b + c = 2 - 5 + 41

2a + b + c = - 3 + 41

2a + b + c = 38

Jadi, nilai 2a + b + c = 38

16. Suatu persamaan kuadrat memiliki akar akar persamaan 2 dan -1. Persamaan kuadrat baru yang setiap akar akar nya dua kali dari akar akar persamaan kuadrat semula adalah .​


persamaan kuadrat

akar2 persamaan kuadrat baru

m = 2 x1 = 2 × 2 = 4

n = 2 x2 = 2 × (-1) = -2

PKB

x² - (m + n)x + (m × n) = 0

x² - 2x - 8 = 0

Penjelasan dengan langkah-langkah:

misalkan akar-akar dari persamaan kuadrat baru adalah p dan q, dan akar-akar nya 2 kali dari akar-akar persamaan kuadrat semula(x1 = 2 dan x2 = -1), maka :

p = 2x1 = 2(2) = 4

q = 2x2 = 2(-1) = -2

Jadi, persamaan kuadrat barunya adalah

x² - (p + q)x + p . q = 0

x² - (4 - 2)x + 4 . (-2) = 0

x² - 2x - 8 = 0

Adapun untuk Rumus Menghitung Persamaan Kuadrat sering disamakan untuk menentukan Akar – Akar Persamaan Kuadrat dan Rumus Mencari Persamaan Kuadrat ini sering muncul didalam Ujian Sekolah maupun Ujian Nasional ditingkat Sekolah Menengah Atas (SMA) sehingga pembahasan tentang Persamaan Kuadrat ini sangatlah penting sekali bagi Siswa Siswi tingkat SMA.

Lalu sekali lagi Bentuk Umum Persamaan Kuadrat y = ax² + bx + c dengan a ≠ 0 dan penjelasan tentang huruf a, b dan c didalam Bentuk Umum Persamaan Kuadrad diatas disebut dengan Koefisien yang terbagi menjadi Koefisien Kuadrat a ialah Koefisien dari x², Koefisien Linier b ialah Koefisien dari x dan Koefisien c ialah Koefisien Konstan atau disebut juga dengan Suku Bebas. Sebagai tambahan saja bahwa Rumus Persamaan Kuadrat dengan Rumus Akar Persamaan Kuadrat itu sama sehingga kalian harus benar – benar mengetahui dan pahami dengan seksama artian tentang huruf a, b dan c didalam Rumus Persamaan Kuadrad tersebut.
