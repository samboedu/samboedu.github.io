Tanpa menggambarkan diagram, tentukan posisi titik-titik berikut terletak di dalam, di luar, atau pada lingkaran X² + Y² = 50.

Kedudukan titik terhadap lingkaran ada 3

Di dalam lingkaran

Pada lingkaran

Di luar lingkaran

Pada Persamaan Lingkaran

X² + Y² = r²

Jika

X² + Y² = r², maka titik tersebut berada pada lingkaran

X² + Y² < r², maka titik tersebut berada di dalam lingkaran

X² + Y² > r², maka titik tersebut berada di luar lingkaran.

Pada Persamaan Lingkaran

(x - a)² + (y - b)² = r²

Jika

(x - a)² + (y - b)² = r², maka titik tersebut berada pada lingkaran

(x - a)² + (y - b)² < r², maka titik tersebut berada di dalam lingkaran

(x - a)² + (y - b)² > r², maka titik tersebut berada di luar lingkaran.

4 + 16 ... 50

20 < 50

Titik B berada dalam lingkaran
