10. Tuliskan persamaan reaksinya dan berikan nama senyawa yang dihasilkan. a. 2-metil-2-butena + H2 b. Propuna + 2 HCI c.2-butena + Br2


Pembahasan

Ketiga persoalan di atas merupakan jenis reaksi adisi. Reaksi ini adalah reaksi penjenuhan yaitu mengubah ikatan rangkap menjadi ikatan tunggal. 

(a). 2-metil-2-butena + H₂ → 2-metil-butana

CH₃ - C = CH - CH₃ + H₂ → CH₃ - CH - CH₂ - CH₃

          |                                             |

        CH₃                                       CH₃

(b). Propuna + HCl berlangsung dalam dua tahap

Tahap Pertama: menghasilkan 2-kloro-propena

CH ≡ C - CH₃ + HCl → CH₂ = C - CH₃

                                                |

                                               Cl

Aturan Markovnikov: atom H dari HCl akan terikat pada atom C senyawa karbon berikatan rangkap yang mengikat atom H lebih banyak.

Perhatikan atom H pada C ≡ C lebih banyak pada atom C sebelah kiri. 

Tahap Kedua: 2-kloro-propena + HCl menghasilkan 2-kloro-propana 

CH₂ = C - CH₃ + HCl → CH₃ - CH - CH₃

            |                                    |

           Cl                                  Cl

Aturan Markovnikov tetap berlaku. Perhatikan atom H pada CH₂ = C tentu lebih banyak pada atom C sebelah kiri.  

(c). 2-butena + Br₂ menghasilkan 2,3-dibromo-butana

CH₃ - CH = CH - CH₃ + Br₂ → CH₃ - CH - CH - CH₃

                                                           |        |

                                                         Br      Br

Kata Kunci : reaksi adisi

A.

CH₃ - C (CH₃) = CH - CH₃ + H₂ → CH₃ + CH (CH₃) - CH₂ - CH₃

2 - metil - 2 butuna + gas hidrogen → 2 - metilbutana

Pemutusan ikatan rangkap dua menghasilkan Alkana.

B.

CH ≡ C - CH₃ + 2HCl → CH₃ - C (Cl)₂ - CH₃

Propuna + asam klorida → 2,2 - dikloropropana

Bila atom C rangkap mengikat atom H yang jumlahnya berbeda, maka atom H pada Hₓ akan terikat pada atom C yang mempunyai atom H lebih banyak.

C.

CH₃ - C ≡ C - CH₃ + Br₂ →. CH₃ - C (Br) = C (Br) - CH₃

2 - butena + bromin → 2,3 - dibromo - 2 butuna
