13. salah satu cara menyimpan workbook pada microsoft excel adalah dengan menggunakan.....A.Ctrl+BB.Ctrl+PC.Ctrl+CD.Ctrl+VE.Ctrl+S


Salah satu cara menyimpan workbook pada Microsoft Excel adalah dengan menggunakan E. Ctrl + S, Ctrl+S sendiri merupakan bentuk shortcut yang diakses dengan menekan tombol ctrl lalu s pada keyboard secara bersamaan. Ctrl+S juga sama halnya dengan perintah Save atau Save As yang dapat ditemui pada ribbon menu  File.

Pembahasan

Ctrl+S atau shortcut untuk melakukan penyimpanan ketika ditekan akan ada dua kemungkinan yang muncul. Yang pertama akan terdirect untuk melakukan save as karena workbook tersebut masih baru,  dan yang kedua, akan melakukan penyimpanan seluruh perubahan yang dilakukan di dalam workbook tersebut.

Opsi-opsi lain pada soal seperti ctrl+B, ctrl+P, ctrl+C dan ctrl+V merupakan jenis shortcut lain yang disediakan oleh Microsoft Excel untuk memudahkan pengguna dalam mengakses beberapa menu.  

Ctrl + B / Shortcut untuk Bold -> Berfungsi untuk menebalkan jenis karakter, simbol, angka dan unsur lain dalam workbookCtrl + P / Shortcut untuk print -> Shortcut ini akan mengarahkan ke menu print yang didalamnya terdapat beberapa pengaturan lebih lanjutCtrl + C / Shortcut untuk copy -> Shortcut ini berfungsi untuk melakukan penyalinan unsur  yang ada dalam workbook seperti karakter, simbol, angka, gambar, chart dan sebagainya ke clipboardCtrl + V / Shortcut untuk paste -> Shortcut ini berfungsi untuk melakukan penempelan unsur yang sebelumnya sempat berada di clipboard setelah tindakan copy atau cut ke tempat yang diinginkan pada workbook

14. bagaimana cara menyimpan ms excel dengan baik !


1. Klik Office Button, lalu pilih menu Save atau Save As.
– Save : menyimpan dokumen baru atau dokumen lama dengan nama yang sama.
– Save As : menyimpan dokumen baru yang belum pernah disimpan atau dokumen lama dengan nama baru.
2. Jika kamu menyorot ikon Save As maka akan muncul kotak pilihan format penyimpanan data. Pilih Excel Workbook atau kamu dapat langsung mengklik ikon Save As, maka akan muncul kotak dialog Save As.
3. Tentukan folder tempat penyimpanan dokumen kamu pada kotak isian Save In dengan acara klik tombol panah kecil di ujung lalu pilih folder yang diinginkan.
4. Ketikkan nama dokumen kamu pada kotak File Name.
5. Isilah kotak isian Save as type dengan Excel Workbook.
6. Klik tombol Save.

22. Jelaskan Salah satu cara menyimpan workbook pada excel adalah dengan menggunakan


Jawaban:

(1). Pertama, Klik File, Tekan simpan sebagai

(2). Kedua, pilih tempat di mana Kamu ingin menyimpan kerja. Misalnya, untuk menyimpan ke desktop atau dalam folder di komputer, klik Komputer.

(3). Ketiga, Klik Telusuri untuk mencari lokasi yang Kamu inginkan dalam folder Dokumen Kamu.

Untuk memilih lokasi lain di komputer Kamu, klik Desktop, lalu pilih tempat sesuai dengan yang Kamu inginkan untuk menyimpan buku kerja.

(4). Keempat, Dalam kotak Nama file, masukkan nama untuk buku kerja baru Kamu. Masukkan nama yang berbeda jika Kamu sedang membuat salinan dari buku kerja yang sudah ada.

(5). Kelima, Untuk menyimpan kerja dalam format file lain (seperti : xls / txt), dalam daftar Simpan sebagai tipe (di bawah kotak Nama file), pilih format yang Kamu inginkan.

(6). Keenam, Tekan Simpan.
