8. Jumlah dua bilangan adalah 1, sedangkan jumlah kuadratnya adalah 2, jumlah pangkat empat dari kedua bilangan tersebut adalah...


Jumlah dua bilangan adalah 1, sedangkan jumlah kuadratnya adalah 2, jumlah pangkat empat dari kedua bilangan tersebut adalah ...


Pembahasan :


misal :

bilangan I = a

bilangan II = b


Kalimat matematika dari pernyataan diatas


a + b = 1

a² + b² = 2


Ditanya

a⁴ + b⁴ = ... ?


Untuk menyelesaikan soal diatas kita bisa gunakan perpangkatan aljabar.


(a + b)² = a² + 2ab + b²

a² + b² = (a + b)² - 2ab


(a + b)⁴ = a⁴ + 4a³b + 6a²b² + 4ab³ + b⁴

a⁴ + b⁴ = (a + b)⁴ - 4ab (a + b)² + 2(ab)²


Penyelesaian soal


a + b = 1

a² + b² = 2


a² + b² = (a + b)² - 2ab

        2 = (1)² - 2ab

    2ab = 1 - 2

    2ab = -1

      ab = -1/2


a⁴ + b⁴ = (a + b)⁴ - 4ab (a + b)² + 2(ab)²

            = (1)⁴ - 4 (-1/2) (1)² + 2 (-1/2)²

            = 1 + 2 + 1/2

           = 3¹/₂

           = 3,5


Jadi jumlah pangkat empat dari kedua bilangan tersebut adalah 3,5

28. jumlah dua bilangan adalah 1, sedangkan jumlah kuadratnya adalah 2. jumlah pangkat empat dari kedua bilangan tersebut adalah


x+y=1
x²+y²=2
maka xy= -1/2
(x²+y2) ²=4
jadi x pangkat 4 +y pangkat 4
=7/2

semoga membantu x + y = 1
x² + y² = 2
(x + y)² - 2xy = 2
1² - 2xy = 2
2xy = 1 - 2
xy = -1/2

x⁴ + y⁴
= (x² + y²)² - 2x²y²
= 2² - 2(xy)²
= 4 - 2(-1/2)²
= 4 - 2 . 1/4
= 4 - 1/2
= 3 1/2
= 7/2 → jawaban
