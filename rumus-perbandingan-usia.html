1. Rumus mencari selisih perbandingan usia


Penjelasan dengan langkah-langkah:

n = jumlah umur yang telah diketahui

a = perbandingan pertama

b = perbandingan kedua

maka a + b = ( a + b )

perbandingan pertama = a / ( a + b ) x n  =  hasil perbandingan pertama

perbandingan kedua    = b / ( a + b ) x n = hasil perbandingan kedua

maka selisih nya

( hasil perbandingan pertama ) - ( hasil perbandingan kedua )

maka , sudah diketahui : hasil selisih dari kedua perbandingan tsb

misal :

jumlah kelereng yaqi dan mamat adalah 18, dengan perbandingan 3 : 6 hitunglah selisih jumlah kelereng mereka masing masing!

3 + 6 = 9

a ) 3/9 x 18 = 6

b ) 6/9 x 18 = 12

c ) selisih , 12 - 6 = 6

15. perbandingan usia Fira dan adiknya adalah 3:2 jika jumlah usia mereka sekarang 20 tahun,usia adik fira 5 tahun yang akan datang adalah??sama rumus/caranya​


Jawaban:

13 tahun

Penyelesaian:

Misalkan,

Fira = F

Adik = A

F : A = 3 : 2

F + A = 20

F : A : F + A

3 : 2 : 5 (diperoleh hasil dari 3 + 2)

F : A : 20

20 ÷ 5 = 4, maka F dan A dikali 4.

F = 3 × 4 = 12 tahun

A = 2 × 4 = 8 tahun

Ditanya : Usia adik Fira 5 tahun yang akan datang

A + 5 = 8 + 5 = 13 tahun

Jadi, usia adik Fira 5 tahun yang akan datang adalah 13 tahun.

(6-4)x = 4 tahun
2x = 4 tahun
x = 2 tahun

5;4;6 = 10 : 8 : 12
jumlah usia mereka = 30
misalkan :
T = Tasya
F = Fina
C = Caca
maka :
T : F : C = 5 : 4 : 6 = 5x : 4x : 6x
C = F + 4

C = F + 4
6x = 4x + 4
6x - 4x = 4
2x = 4
x = 4/2
x = 2

T + F + C = 5x + 4x + 6x
T + F + C = 5(2) + 4(2) + 6(2)
T + F + C = 10 tahun + 8 tahun + 12 tahun
T + F + C = 30 tahun
