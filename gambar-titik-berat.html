3. Hitung koordinat titik berat pada gambar !


bagi bidang menjadi 3 bagian (seperti sketsa di bawah ini)
A1 = 5 x 1    ;  x1 = 1/2 . 1 = 0,5     ; y1 = 1/2 . 5 = 2,5 
     = 5              (koodinat titik berat pada perpotongan diagonal)
A2 = 2 x 1    : x2 = 1 + 1/2 . 1    ; y2 = 4 + 1/2 . 1
     = 2                 = 1,5                       = 4,5
A3 = 1 x 1
     = 1          : x3 = 1 + 1/2 . 1     y2 = 2 + 1/2 . 1
                           = 1,5                 y2 = 2,5
Xo = [tex] \frac{A1 .x1 + A2 .x2 + A3 . x3}{A1 + A2 + A3} [/tex]
     = [tex] \frac{5 . 0,5 + 2 . 1,5 + 1 . 1,5}{5 + 2 + 1} [/tex]
     = [tex] \frac{2,5 + 3 + 1,5}{8} [/tex]
     = 7/8
Yo = [tex] \frac{A1 . y1 + A2 . y2 + A3 . y3}{A1 + A2 + A3} [/tex]
     = [tex] \frac{5 . 2,5 + 2 . 4,5 + 1 . 2,5}{5 + 2 + 1} [/tex]
     = [tex] \frac{12,5 + 9 + 2,5}{8} [/tex]
     = 24/8
     = 3
jadi, koordinat titik berat : (7/8  ,  3)

8. koordinat titik berat pada homogen seperti gambar adalah​


Xpm = (X1.A1 + X2.A2)/ΣA

= (30.1200 + 30.1200)/(1200.2)

= (36000 + 36000)/2400

= 72000/2400 ~ 30 cm ✓

Ypm = (Y1.A1 + Y2.A2)/ΣA

= (30.1200 + 70.1200)/(1200.2)

= (36000 + 84000)/2400

= 120000/2400 ~ 50 cm ✓

28. tentukan letak titik berat gambar diatas! ​


Penjelasan:

gambar terdiri dari :

1. persegi panjang, dengan koordinat (3, 1½)

2. segitiga sama kaki, dengan koordinat (4½, 4)

jika koordinat titik berat => (Xp, Yp)

Xp = (3L1 + 4½L2) / (L1+L2)

Yp = (1½L1+4L2)/(L1+L2)

L1 = 6×3 = 18

L2 = ½×3×3 = 4½

=> Xp = (3×18 + 4½×4½)/(18+4½)

= (54 + 20¼)/22½

= 74¼/22½

= 3,3

=> Yp = (1½×18 + 4×4½)/(18+4½)

= (27+18)/22½

= 45/22½ = 2

maka titik berat => (3,3 , 2)

Diketahui
- Sebuah kotak dg panjang 6 * 2 satuan. Maka
a. A' = 6 * 2 = 12 satuan
- Titik berat sebuah persegi adalah 1/2 dari tinggi persegi ( karena terletak di sumbu y ) Maka
b. y' = 1/2 * 2 = 1 satuan
Jawab :
y0=y'.A' / A'
y0=1.12 / 12
y0= 1
===========================================
Untuk menentukan koordinat titik berat nya maka Z0=(x0,y0)
Karena y0 sudah diketahui maka x0 dicari.
Diketahui x'= 2 + ( 1/2 * 8 ) = 6
Mengapa ada "2"? itu karena koordinasi bangun dimulai dari sumbu x nomor 2
Mengapa ada 4? Itu karena titik berat bangun berada di 1/2 panjang bangun
Maka jumlah keseluruhan 4 + 2 = 6
Ditanya
x0=x' * A' / A' = 6 * 12 / 12 = 6
Jadi letak titik berat di sumbu x adalah 6.
Jadi kesimpulannya, letak tepatnya titik keseimbangan
z0=(x0,y0)
z0=(6,1)
