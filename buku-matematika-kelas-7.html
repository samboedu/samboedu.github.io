Tujuan pembuatan tabel membuktikan sifat komutatif dan sifat asosiatif dalam penjumlahan.

Pembahasan

SIFAT DALAM PENJUMLAHAN BILANGAN BULAT

Dalam penjumlahan bilangan bulat memiliki beberapa sifat:

1. Tertutup dalam penjumlahan.

a + b = c

Bila a dan b bilangan bulat, maka c juga bilangan bulat.

2. Berlaku sifat komutatif.

a + b = b + a

3. Memiliki unsur identitas 0.

a + 0 = 0 + a = a

4. Memiliki invers.

a + (-a) = (-a) + a = 0

5. Berlaku sifat asosiatif.

(a + b) + c = a + (b + c)

6. Berlaku sifat distributif dengan penjumlahan dan perkalian.

(a × b) + (a × c) = a × (b + c)

Aturan tanda

Bila + dan + maka +

Bila + dan - maka -

Bila - dan + maka -

Bila - dan - maka +

Diketahui:

1. a = 1

b = - 6

c = - 11

Ditanyakan

A. a + b

B. b + a

C. (a + b) + c

D. a + (b + c)

Penjelasan:

1. a = 1  b = - 6 dan c = - 11

A. a + b = 1 + (-6) = 1 - 6 = - 5

B. b + a = (-6) + 1 = - 5

C. (a + b) + c = (-5) + (-11) = -5 - 11 = -16

D. a + (b + c) = 1 + ((-6) + (-11))

= 1 + (-6 - 11) = 1 + (-17) = 1 - 17 = - 16

2. a = 2  b = 7  c = - 12

A. a + b = 2 + 7 = 9

B. b + a = 7 + 2 = 9

C. (a + b) + c = 9 + (-12) = 9 - 12 = - 3

D. a + (b + c) = 2 + (7 + (-12)) = 2 + (7 - 12) = 2 + (-5) = 2 - 5 = -3

3. a = 3  b = 8  c = 13

A. a + b = 3 + 8 = 11

B. b + a = 8 + 3 = 11

C. (a + b) + c = 11 + 13 = 24

D. a + (b + c) = 3 + (8 + 13) = 3 + 21 = 24

4. a = - 4  b = 9  c = 14

A. a + b = - 4 + 9 = 5

B. b + a = 9 + (-4) = 9 - 4 = 5

C. (a + b) + c = 5 + 14 = 19

D. a + (b + c) = - 4 + (9 + 14) = -4 + 23 = 19

5. a = -5  b = -10  c = 16

A. a + b = -5 + (-10) = -5 - 10 = - 15

B. b + a = -10 + (-5) = -10 - 5 = -15

C. (a + b) + c = -15 + 16 = 1

D. a + (b + c) = -5 + (-10 + 16) = -5 + 6 = 1

Terlihat sifat komutatif dan sifat asosiatif berlaku dalam penjumlahan.

11. kunci jawaban buku matematika kelas 7 semester 2 hal 84​


Ayo Kita Menalar (Kelas 7 Semester 2 halaman 84)

Seorang pedagang membeli mangga sebanyak 50 kg dengan harga Rp.7.500,00 per kg. Kemudian 30 kg dijual dengan harga Rp.9.000,00 per kg, dan sisanya dijual dengan harga Rp.7.000,00 per kg.

Maka:

a. Harga pembelian dan harga penjualan adalah Rp.375.000,00 dan Rp.390.000,00.

b. Besarnya untung dari hasil penjualan tersebut adalah Rp.15.000,00.

Pendahuluan

Aritmetika sosial adalah materi matematika mengenai hal-hal yang sering terjadi dalam kehidupan sehari-hari. Contohnya seperti tabungan, perdagangan, bruto, netto, bunga, dan lain-lain. Pada permasalahan kali ini, kita akan membahas persoalan mengenai perdagangan. Beberapa hal yang perlu dipahami adalah sebagai berikut:

Perdangan akan mengalami untung jika harga jual > harga beli. Perdangan akan mengalami rugi jika harga jual < harga beli.

Dari penjelasan tersebut, mari kita selesaikan permasalahan di atas!

Pembahasan

Diketahui:

Banyak mangga = 50 kg Harga Beli = Rp.7.500,00/ kg Harga Jual (20 kg) = Rp.9.000,00/ kg Harga Jual (sisa) = Rp.7.000,00/ kg

Ditanyakan:

a. Harga pembelian dan harga penjualan.

b. Besarnya untung atau rugi dari hasil penjualan tersebut.

Jawab:

a. Harga pembelian dan harga penjualan.

Untuk menentukan harga beli:

[tex] Harga \ Beli = 50 \ kg \times Rp.7.500.00/ kg \\ Harga \ Beli = Rp.375.000,00 [/tex]

Jadi harga pembelian mangga seluruhnya adalah Rp.375.000,00.

Untuk menentukan harga jual:

[tex] Harga \ Jual = (20 \ kg \times Rp.9.000,00/kg) + ((50 \ kg \ - \ 20 \ kg) \times Rp.7.000,00/kg) \\ Harga \ Jual = Rp.180.000,00 \ + \ (30 \ kg \times Rp.7.000,00) \\ Harga \ Jual = Rp.180.000,00 \ + \ Rp.210.000,00 \\ Harga \ Jual = Rp.390.000,00 [/tex]

Jadi, harga penjualan mangga seluruhnya adalah Rp.390.000,00.

b. Besarnya untung atau rugi dari hasil penjualan tersebut.

Dari harga pembelian dam harga penjualan, kita dapat melihat bahwa harga jual > harga beli. Maka pedagang itu mendapatkan keuntungan.  

[tex] Keuntungan = Harga \ Jual \ - \ Harga \ Beli \\ Keuntungan = Rp.390.000,00 \ - \ Rp.375.000,00 \\ Keuntungan = Rp.15.000,00 [/tex]

Jadi, keuntungan yang didapatkan pedagang tersebut adalah Rp.15.000,00.

14. jawaban buku matematika kelas 7 semester 2 hal 173 no 10​


Hubungan sudut yang besarnya 105° dan sudut yang besarnya 3b° berpelurus, sedangkan sudut yang besarnya 3b° dan sudut yang besarnya 3a° luar berseberangkan. Jadi, nilai a + b pada gambar adalah 50°.

Penjelasan dengan langkah-langkah:

Diketahui:
Sudut yang besarnya 105° berpelurus dengan sudut yang besarnya 3b°.
Sudut yang besarnya 3b° luar berseberangan dengan sudut yang besarnya 3a°
Ditanyakan:
Berapa nilai a + b?
Jawab:
Sudut yang saling berpelurus jumlah besar sudutnya adalah 180°, sehingga:
105° + 3b° = 180°
3b° = 180° - 105°
3b° = 75°
b° = 25°

Sudut yang luar berseberangan memiliki besar sudut yang sama, sehingga:
3a° = 3b°
3a° = 3(25°)
3a° = 75°
a° = 25°

Menghitung nilai a + b:
a + b = 25° + 25°
a + b = 50°
Dengan demikian, nilai a + b = 50°.
