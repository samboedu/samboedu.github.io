[tex]\boxed { \: f(x) = x² − 4x \: }[/tex]

[tex]\boxed { \: Jawab \: }[/tex]

[tex]f '(x) = 2x − 4[/tex]

[tex]f(x) stasioner ⇒ f '(x) = 0[/tex]

Misalkan f(a) adalah nilai stasioner di x = a

1. f(a) adalah nilai balik maksimum, jika :

untuk x < a maka f '(x) > 0 (naik)

untuk x > a maka f '(x) < 0 (turun)

2. f(a) adalah nilai balik minimum, jika :

untuk x < a maka f '(x) < 0 (turun)

untuk x > a maka f '(x) > 0 (naik)

9. Tentukan nilai-nilai stasioner dari fungsi f(x)= x3-3x2-x+3 serta jenis-jenis nilai stasionernya


f(x) = x³ - 3x² - x + 3. utk mengetahui nilai stasionernya maka turunin deh

f'(x) = 3x² - 6x - 1
f'(x) = x² - 2x - 1/3 (udah gini faktorin deh)

[tex](x- \frac{13}{6} ) (x+ \frac{1}{6}) [/tex]
 
jadi..
[tex]x = \frac{13}{6} [/tex] dan [tex]x = -\frac{1}{6} [/tex]

untuk mengetahui jenis statisionernya maka harus di turunkan lagi

f'(x) = x² - 2x - 1/3

f''(x) = 2x - 2 (masukin deh x = 13/6 atau x = -1/6)

utk x = 13/6

2(13/6) - 2 

= 13/3 - 2

= 2,33 > 0 (jadi ini adalah nilai statisioner minimum)

utk x = -1/6

2 (-1/6) - 2

= -1/3 - 2

= -2,33 < 0 (ini adalah nilai statisioner maksimum) 

Jadi puncak maksimum pada kurva diatas akan pada saat x= -1/6
dan nilai minimum pada kurva diatas akan pada saat x = 13/6

22. jrlaskan pngertian nilai stasioner beserta contohnya​


Nilai stasioner.

Titik stasioner merupakan titik di mana fungsi "berhenti" naik atau turun.

Nilai stasioner adalah nilai yang didapat pada suatu fungsi f(x), ketika f '(x) bernilai 0, alias gradien garis singgung kurva fungsi f(x) pada titik stasioner bernilai 0.

Jika fungsi y = f(x) terdiferensial di x = a dengan f’(a) = 0, maka f(a) merupakan nilai-nilai stasioner dari fungsi f(x) di x = a.

Ada tiga jenis nilai stasioner, yaitu:

Nilai maksimum lokalNilai minimum lokalNilai belok

25. soal titik stasioner trims

• stasioner : f'(x) = 0
f'(x) = 6x² - 18x + 12
0 = 6x² - 18x + 12
0 = x² - 3x + 2
0 = (x - 2)(x - 1)
x = 2 atau x = 1

untuk x = 2
f(2) = 2(2)³ - 9(2)² + 12(2)
f(2) = 16 - 36 + 24
f(2) = 14
y = 14

untuk x = 1
f(1) = 2(1)³ - 9(1)² + 12(1)
f(1) = 2 - 9 + 12
f(1) = 5
y = 5

• fungsi naik : f'(x) > 0
f'(x) = 6x² - 18x + 12
0 < 6x² - 18x + 12
0 < x² - 3x + 2
0 < (x - 2)(x - 1)
maka, interval fungsi naiknya :
x < 1 dan x > 2

• fungsi turun : f'(x) < 0
f'(x) = 6x² - 18x + 12
0 > 6x² - 18x + 12
0 > x² - 3x + 2
0 > (x - 2)(x - 1)
maka, interval fungsi turunnya :
1 < x < 2
