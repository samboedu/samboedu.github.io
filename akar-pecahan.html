Jawaban:

• Akar

√2 / √3 + √5

= √2 / √3 + √5 x √3 - √5 / √3 - √5

= √6 - √10 / 3 - 5

= √10 - √6 / 2

2. buktikan mengapa pecahan a per akar b dapat dirasionalkan dengan mengalikan pecahan tersebut dengan akar b per akar b


Karena nilai akar b selalu positif, maka akar b per akar b = 1. Jadi perkalian a per akar b dengan akar b per akar b tidak akan mengubah nilai a per akar b, namun menyebabkan penyebut menjadi bilangan rasional.
