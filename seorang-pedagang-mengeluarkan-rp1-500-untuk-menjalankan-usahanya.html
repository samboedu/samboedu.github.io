Jadi, hasil yang diperoleh oleh pedagang adalah Rp90.000 + Rp60.000 = Rp150.000

22. Seorang pedagang sepeda ingin membeli 25 sepeda untuk persediaan. ia ingin membeli sepeda gunung dengan harga rp1. 500. 000,00 per buah dan sepeda balap dengan harga rp2. 0. 000,00 per buah. ia merencanakan tidak akan mengeluarkan uang lebih dari rp42. 0. 000,00. jika keuntungan sebuah sepeda gunung rp500. 000,00 dan sebuah sepeda balap rp600. 000,00, maka keuntungan maksimum yang diterima pedagang adalah…


Keuntungan maksimum yang diterima pedagang adalah Rp 13.400.000,00.

Untuk menyelesaikan soal di atas, kita dapat menggunakan penyelesaian Program Linear dengan Dua Variabel.

Penjelasan dengan langkah-langkah:

Diketahui:

Banyak sepeda yang beli 25 sepeda.

Harga:

          Sepeda gunung = Rp 1.500.000,00 per buah

          Sepeda balap = Rp 2.000.000,00 per buah.

Uang tidak lebih dari Rp 42.000.000,00.

Keuntungan:

      sebuah sepeda gunung Rp 500.000,00

      sebuah sepeda balap Rp 600.000,00

Ditanya:

Keuntungan maksimum yang diterima pedagang adalah…

Jawab:

Misalkan:

Sepeda gunung = x

Sepeda balap = y

x + y ≤ 25

1.500.000x + 2.000.000y ≤ 42.000.000

                            15x + 20 ≤ 420

                              3x + 4y ≤ 84

Mencari titik pojok.

x + y = 25

      x = 0; y = 25         (0, 25)

      y = 0; x = 25        (25, 0)

3x + 4y = 84

      x = 0

      3(0) + 4y = 84

                 4y = 84

                   y = 21

    (0, 21)

      y = 0

      3x + 4(0) = 84

                 3x = 84

                   x = 28

    (28, 0)

x + y = 25            (dikali 4)

3x + 4y = 84        (dikali 1)    

4x + 4y = 100

3x + 4y = 84   -

         x = 16

x + y = 25

     y = 25 - x

     y = 25 - 16

     y = 9

(16, 9)

Mencari keuntungan yang diterima: 500.000x + 600.000y

(25, 0)

  = 500.000(25) + 600.000(0)

  = 12.500.000 + 0

  = 12.500.000

(16, 9)

  = 500.000(16) + 600.000(9)

  = 8.000.000 + 5.400.000

  = 13.400.000

(0, 21)

  = 500.000(0) + 600.000(21)

  = 0 + 12.600.000

  = 12.600.000

Jadi, keuntungan maksimum yang diterima pedagang adalah Rp 13.400.000,00.
