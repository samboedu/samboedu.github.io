4. Nilai kuartil bawah dari data di atas adalah​


Jawaban:

Kuartil bawah dari data pada tabel tersebut adalah ...

Tabel data bisa dilihat pada lampiran

Kuartil adalah ukuran yang membagi data terurut menjadi empat bagian sama banyak.

Pembahasan

Jumlah data (n) = 3 + 5 + 10 + 11 + 8 + 2

= 40

Kuartil bawah (Q₁) = nilai data ke- ¹/₄ (n + 1)

= nilai data ke- ¹/₄ (40 + 1)

= nilai data ke- ¹/₄ (41)

= nilai data ke- 10,25

Nilai data ke- 10,25 terletak pada kelas interval 51 - 60.

L₁ = 51 - 0,5 = 50,5

∑f = 3 + 5 = 8

f = 10

p = 60 - 51 + 1 = 10

Menentukan kuartil bawah

n−∑f × p

(40)−8

× 10

= 50,5 + (10 - 8)

= 50,5 + 2

= 52,5

Jadi kuartil bawah dari data pada tabel tersebut adalah 52,5

2, 4, 5, 8, 9, 11, 12, 13, 14, 17, 19, 21

Banyak data (n) = 12

Cari Letak Kuartil Bawah (Q₁)
Letak Q₁ = 1/4 (n + 1)
Letak Q₁ = 1/4 (12 + 1)
Letak Q₁ = 13/4
Letak Q₁ = 3,25

30. nilai median kuartil bawah dan kuartil atas dari data berikut 8,6,5,11,80,12,​


Nilai median = 6, nilai kuartil bawah = 9,5, dan nilai kuartil atas = 12. Simak penjelasan berikut.

Pembahasan

Kuartil tengah (Q₂), disebut juga median, adalah nilai tengah dari semua data. Nilai tersebut didapatkan dengan membagi data menjadi dua bagian sama.

Kuartil bawah (Q₁) adalah nilai tengah dari data di sebelah kiri kuartil tengah.

Kuartil atas (Q₃) adalah nilai tengah dari data di sebelah kanan kuartil tengah.

Diketahui

8, 6, 5, 11, 80, 12

Ditanya

Nilai median, kuartil bawah, dan kuartil akhir

Penyelesaian

Mengurutkan data dari yang terkecil

5, 6, 8, 11, 12, 80

   Q₁   Q₂  Q₃

Nilai median = Q₁ = 6

Nilai kuartil bawah = Q₂ = (8 + 11)/2 = 9,5

Nilai kuartil atas = Q₃ = 12

Kesimpulan

Jadi, nilai median = 6, nilai kuartil bawah = 9,5, dannilai kuartil atas = 12.
