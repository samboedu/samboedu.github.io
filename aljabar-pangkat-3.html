7. (8t) pangkat 3 mtk aljabar​


[tex]{(8t)}^{3} = 512 \: {t}^{3} [/tex]

Pembahasan :

Aljabar adalah ilmu dalam matematika yang mengubah angkanya menjadi huruf/simbol tertentu. Beberapa rumus dalam aljabar :

[tex] \boxed{ {a}^{b} \times {a}^{c} = {a}^{b + c} } [/tex]

[tex] \boxed{ \frac{ {a}^{b} }{ {a}^{c} } = {a}^{b - c} } [/tex]

[tex] \boxed{ {(ab)}^{c} = {a}^{c} {b}^{c} } [/tex]

[tex] \boxed{a(b + c) = ab + ac} [/tex]

[tex] \boxed{a(b - c) = ab - ac} [/tex]

Penyelesaian soal di atas :

[tex] { (ab)}^{c} = {a}^{c} {b}^{c} [/tex]

[tex] {(8t)}^{3} = {8}^{3} {t}^{3} [/tex]

[tex] = 512 \: {t}^{3} [/tex]
