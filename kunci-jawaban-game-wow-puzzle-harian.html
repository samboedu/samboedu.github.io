3. cara membuat dynamic text nama dan usia pada perymainan game puzzle di aplikasi Adobe Flash disk


Membuat Teks Dinamis (Dynamic Text)


Nah, kalau yang ini mungkin saudaranya Static Text, tapi beda sifat. Jika Static text sifatnya statis atau tak berubah-ubah, maka beda halnya dengan dynamic text. Teks tipe ini sifatnya berubah-ubah tergantung dari perintah yang dibuat oleh programmernya di panel Action. Dynamic text digunakan pada aplikasi kalkulator, biasanya pada bagian hasil perhitungan.


Bagaimana cara menampilkannya pada stage? berikut ini langkah-langkahnya


￼Langkah-langkah membuat teks dinamis


1. Pilih Text tool


2. Lakukan Klik dan Drag pada stage dimana kamu menginginkan tulisan tersebut muncul.


3. Kemudian pada Panel Propertiesnya (Bila tidak muncul pada jendela aplikasi, pilih windows>properties) pilih Dynamic text, ada pada menu drop down. Pada saat memilih dynamic text maka akan muncul sebuah kolom baru di atasnya, kolom instance name.


4. Tuliskan sebuah nama pada kolom yang bertuliskan instance name, itu sebagai variabel untuk teks ini. disebut dengan “instance name”, tulis misalnya “hasil_teks”.


5. Atur karakter huruf, jenis huruf, ukuran, warna dan lain-lain pada option character.


6. Pada pilihan Paragraph, atur format align-nya, spasi, margin dan orientasi.


Teks dinamis buatanmu sudah jadi.



￼Hasil Akhir.
