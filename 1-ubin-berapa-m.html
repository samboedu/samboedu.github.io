■ PENDAHULUAN

〜 Pengertian 〜

Pecahan adalah bagian dari keseluruhan. Pecahan terdiri atas pembilang dan penyebut. Pembilang merupakan bilangan yang dibagi, sedangkan penyebut merupakan bilangan yang membagi. Suatu pecahan dibaca dengan menyebutkan pembilang terlebih dahulu, kemudian diikuti dengan penyebut.

[tex] \boxed{\sf{ \frac{a}{b} = \frac{a \: → \: pembilang}{b \: → \: penyebut \: }}} \\ [/tex]

〜 Macam 〜

✓ Pecahan Biasa → [tex] \sf \frac{a}{b} [/tex]✓ Pecahan Campuran → [tex] \sf a \frac{b}{c} [/tex]✓ Pecahan Desimal → [tex] \sf a,b [/tex] ✓ Pecahan Persen → [tex] \sf a\% [/tex]✓ Pecahan Permil → [tex] \sf a‰ [/tex]

〜 Operasi Pecahan 〜

Penjumlahan

Menyamakan penyebut dengan mencari KPK. Apabila sudah sama langsung operasikan ke penjumlahan saja. Sebagai contoh :

[tex] \sf \frac{a}{b} + \frac{c}{b} = \frac{a + c}{b} [/tex]

Pengurangan

Sama hal-nya dengan penjumlahan, namun hanya berbeda operasi saja. Menyamakan penyebut dengan mencari KPK. Apabila penyebut sudah sama langsung operasikan ke pengurangan saja. Sebagai contoh :

[tex] \sf \frac{a}{b} - \frac{c}{b} = \frac{a - c}{b} [/tex]

Perkalian

Berbeda dengan pengurangan dan penjumlahan, perkalian tinggal mengalikan pembilang dengan pembilang dan penyebut dengan penyebut. Sebagai contoh :

[tex] \sf \frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d} [/tex]

Pembagian

Jika pembagian pecahan, kita harus mengubah/membalik pecahan kedua atau pembagi pecahan awal menjadi terbalik ( pembilang menjadi penyebut dan sebaliknya ). Lalu kita ubah menjadi perkalian pecahan. Sebagai contoh :

[tex] \sf \frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c} [/tex]

Menyederhanakan pecahan

✓ Menyederhanakan : [tex] \sf \: \frac{2}{10} = \frac{2 \div 2}{10 \div 2} = \frac{1}{5} [/tex]

✓ Pecahan senilai : [tex] \sf \: \frac{1}{5} = \frac{1 \times 2}{5 \times 2} = \frac{2}{10 } [/tex]

✓ biasa → campuran : [tex] \sf \: 3 \frac{2}{5} = \frac{5 \times 3 + 2}{5} = \frac{17}{5} [/tex]

✓ campuran → biasa : [tex] \sf \: \frac{17}{5} = 17 \div 5 = 3 \frac{2}{5} [/tex]

Setelah memperdalami, yuk kerjakan soal atau pertanyaan dari tugas diatas. Jawaban ada pada bagian pembahasan serta kesimpulan}
