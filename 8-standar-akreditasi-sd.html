25. standar deviaasi (SD) dari 3,5,6,7,9 adalah


STAT

Standard Deviasi

dengan rumus Varian  S= √ [ n ( ∑xi²) -  {∑xi}² ] / ( n )(n-1) ]

*

data 3,5 6, 7, 9

*

n = 5

∑(xi)² = 3²+5²+6²+ 7²+ 9²= 200

∑x1 = 3 + 5+ 6 + 7 + 9 =  30 --> { ∑x1}² = 30² = 900

*

S²= [ n ( ∑xi²) -  {∑xi}² ] / ( n )(n-1)

S² = [ 5 (200) - 900 ] / (5. 4)

S² = (1.000 - 900) / 20

S²= 100/20

S²= 5

*

SD= √S² = √5
