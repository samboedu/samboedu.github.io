Misalkan pada pemrograman berbasis objek. Objek yg terlibat pada baris program pada soal di atas yaitu:

Label1: objek yg memiliki property caption

text1: objek yg memiliki property text

text2: objek yg memiliki property text

Jadi kesimpulannya, baris program di atas sudah tepat.

Yang perlu dicermati ialah saat program dijalankan: harus dipastikan bahwa text1 dan text2 diisikan data angka (bukan data teks)
