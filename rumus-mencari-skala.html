Rumus :

- Skala (S)

[tex]s = \frac{jarak \: pada \: peta}{jarak \: sebenarnya} [/tex]

[tex]s = \frac{jp}{js} [/tex]

- Jarak Pada Peta

[tex]jp = s \times js[/tex]

- Jarak Sebenarnya

[tex]js = \frac{jp}{s} [/tex]

Atau :

[tex]js = jp \times s[/tex]

26. Bagaimana rumus mencari skala?​


Penjelasan dengan langkah-langkah:

Cara Menghitung Rumus Skala Peta

Jarak Sesungguhnya = Jarak pada Peta / Skala.

Skala = Jarak pada Peta / Skala.

Jarak pada Peta = Skala x Jarak Sebenarnya.

Jawab: Skala = Jarak pada Peta : Jarak Sesungguhnya. = 8 cm : 160 km. = 8 cm : 160.000.000 cm. = 1 : 2.000.000.
