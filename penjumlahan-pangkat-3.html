6. perhatikan penjumlahan bilangan berikut ini 1 pangkat 3 ditambah 2 pangkat 3 ditambah 3 pangkat 3 ditambah 4 pangkat 3 ditambah 5 pangkat 3 ditambah titik-titik ditambah 46 pangkat 3 + 47 pangkat 3 + 48 pangkat 3 + 49 pangkat 3 + 50 pangkat 3 panjang formula yang berlaku untuk penjumlahan bilangan tersebut kemudian buktikan kebenaran formula yang kamu peroleh


Formula penjumlah tersebut adalah

[tex]1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ 46^3 + 47^3 + 48^3 + 49^3 + n^3 = \big(\frac{n(n + 1)}{2}\big)^2[/tex]

Untuk n = 50

Pembuktiannya akan dibuktikan dengan induksi matematika

Pembahasan

Induksi matematika sebenarnya merupakan satu bagian dengan logika matematika karena di dalam logika matematika ini termuat materi tentang pernyataan yang bernilai benar atau salah, ekuivalensi, ingkaran dan penarikan kesimpulan. Oke langsung aja yukk kita lihat penjabaran dari jawaban soal kali ini

Untuk n = 1 akan dibuktikan benar

[tex]\big(\frac{n(n + 1)}{2}\big)^2 = \big(\frac{1(1 + 1)}{2}\big)^2\\\big(\frac{n(n + 1)}{2}\big)^2 = \big(\frac{1(2)}{2}\big)^2\\\big(\frac{n(n + 1)}{2}\big)^2 = 1^2\\\big(\frac{n(n + 1)}{2}\big)^2 = 1[/tex]

Terbukti benar untuk n=1

Dimisalkan formula penjumlahan benar untuk n = k, diperoleh

[tex]1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ k^3 = \big(\frac{k(k + 1)}{2}\big)^2\\1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ k^3 = \big(\frac{k^2(k + 1)^2}{4}\big)\\[/tex]

Akan dibuktikan formula tersebut benar untuk n = k + 1, maka

[tex]1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ (k + 1)^3 = 1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ k^3 + (k + 1)^3\\1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ (k + 1)^3 = \big(\frac{k(k + 1)}{2}\big)^2 + (k + 1)^3\\1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ (k + 1)^3 = \frac{k^2(k + 1)^2}{4} + (k + 1)^3\\1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ (k + 1)^3 = \frac{k^2(k + 1)^2}{4} + \frac{4(k + 1)^3}{4}\\[/tex]

[tex]1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ (k + 1)^3 = \frac{k^2(k + 1)^2 + 4(k + 1)^3}{4}\\1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ (k + 1)^3 = \frac{(k + 1)^2(k^2 + 4(k + 1))}{4}\\1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ (k + 1)^3 = \frac{(k + 1)^2(k^2 + 4k + 4)}{4}\\1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ (k + 1)^3 = \frac{(k + 1)^2(k + 2)^2}{4}\\1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ (k + 1)^3 = \frac{(k + 1)^2((k + 1)+ 1)^2}{4}\\[/tex]

Jadi terbukti benar untuk n = k + 1

Jadi formula penjumlahan

[tex]1^3 + 2^3 + 3^3 + 4^3 + 5^3 +...+ 46^3 + 47^3 + 48^3 + 49^3 + n^3 = \big(\frac{n(n + 1)}{2}\big)^2[/tex]

untuk n = 50, terbukti benar
