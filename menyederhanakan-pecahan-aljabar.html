17. aljabar pecahansederhanakan


3 / (x-4)    -   x / (x² - 3x -4)

-------------------------------------------------------
Sederhanakan dulu x² - 3x -4

x² - 3x -4  = (x-4)(x+1)
--------------------------------------------------------

Maka
3 / (x-4)    -   x / (x² - 3x -4)
= 3 / (x-4)    -   x / (x-4)(x+1)   ==> samakan penyebut
= 3(x+1) / (x-4)(x+1)     -  x  / (x-4)(x+1)
= 3x+3  / (x-4)(x+1)       -  x  / (x-4)(x+1)
=  (3x + 3 - x) /  (x-4)(x+1)
= 2x+3  / (x-4)(x+1)
= 2x+ 3  /   x²-3x-4


Semoga membantu

24. Sederhanakan lah pecahan aljabar


Jawaban:

diketahui :

sederhanakan bentuk dari

[tex] \frac{ {a}^{2} - a - 20 }{ {a}^{2} - 14a + 45} [/tex]

jawab :

[tex] \frac{ {a}^{2} - a - 20 }{ {a}^{2} - 14a + 45} = \frac{ {a}^{2} + 4a - 5a - 20}{ {a}^{2} - 9a - 5a + 45 } \\ \frac{(a - 5)(a + 4)}{(a - 9)(a - 5)} = \frac{(a + 4)}{(a - 9)} [/tex]

Maka bentuk sederhananya adalah

[tex]\frac{(a + 4)}{(a - 9)} [/tex]
