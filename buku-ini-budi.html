6. perbandingan banyaknya buku adit dan budi 1:2,sedangkan perbandingan banyak buku Budi dan CACA 3:4 jika jumlah buku adit dan caca 33 buah maka banyak buku budi adalah


Adit = A, Budi = B, Caca = C
maka
A : B = 1 : 2 = 3 : 6
B : C = 3 : 4 = 6 : 8
maka
A : B : C = 3 : 6 : 8
A + C = 33

maka
B = 6/(3 + 8) · (A + C)
   = 6/11 · 33
   = 6 · 3
   = 18 buah

TErimakasih semoga membantubuku adit = a
buku budi = b
buku caca = c

misal: x = angka pembanding
a : b
1 : 2 = 1x : 2x => x + 2x

misal: y = angka pembanding
b : c
3 : 4 = 3y + 4y

a + c = 33
x + 4y = 33
x = 33 - 4y

buku budi = 2x = 3y
2x = 3y
2(33 - 4y) = 3y
66 - 8y = 3y
66 = 3y + 8y
66 = 11y
11y = 66
y = 66/11
y = 6

jadi,
3y = 3(6) = 18

banyak buku budi adalah 18 buah

12. Selisih buku Budi dan Reni 8 buku.perbandingan buku Budi dan Reni 5 : 3 berapa buku masing"?​


Jawab:

Buku Budi = 20 dan Buku Reni = 12

Penjelasan dengan langkah-langkah:

Misalkan:

Buku Budi = x

Buku Reni = y

Maka persamaan menjadi:

Selisih x - y = 8

Perbandingan x : y = 5 : 3

Selisih perbandingan x - y = 5 - 3 = 2

Selanjutnya kita membandingkan selisih dengan jumlah buku masing-masing:

[tex]\frac{selisih.perbandingan}{buku.budi} = \frac{2}{5} \\\\\frac{8}{x} = \frac{2}{5} \\\\8*5= 2x\\x=20\\x-y = 8\\20 - y= 8\\y=12[/tex]

Sehingga didapatkan x = 20 dan y = 12

19. Sebanyak 48 buku dibagikan kepada Andi, Budi, dan Doni. Buku yang diterima Andi sebanyak 6 buah. Banyak buku yang diterima Budi dua kali dari banyak buku yang diterima Doni. Banyak buku yang diterima Budi adalah....


Jawab: 28 bulky

Penjelasan dengan langkah-langkah:

Total 48 buku , 6 bulky punya Andi maka

48-6 = 42 buku untuk budi Dan doni

Budi menerima buku 2x lebih banyak maka :

Budi = 2b

Doni = b

42 buku = budi + doni

42 = 2b + b

42 = 3b

B = 14 < masukan ke b

Budi = 2b maka 2 x 14 = 28

Doni = b maka 1 x 14 = 14

Jawaban:

28buku

Penjelasan dengan langkah-langkah:

andi + budi + doni = 48 buku

6 + budi + doni = 48 buku

budi = 2 x doni

maka,

(pers. 1) 6 + x + y = 48

(pers. 2) x = 2y

pers 2 di subtitusikan ke pers 1

6 + x + y = 48

6 + 2y + y = 48

2y + y = 48 - 6

3y = 42

y = 42/3

y = 14

y = 14 disubtitusikan ke per 2

x = 2y

x = 2(14)

x = 28

maka buku yang diterima budi adalah 28 buku

note = x = buku budi

y = buku doni
