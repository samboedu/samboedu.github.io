Pembahasan:

24. Berikut merupakan kategori pesan error pada Excell, kecuali…

Terdapat beberapa pesan error yang ada pada Excell, yaitu syntax Error, Error Logika, Runtime Error, dan User Error

Jadi, pada nomor 24 yang tidak termasuk ke dalam kategori pesan error pada Excell adalah time error (c)

25. Pesan error #NAME pada sel di Excel, karena….

Berikut ini merupakan penyebab munculnya Pesan error #NAME pada sel di Excel, yaitu;

1. Terdapat kesalahan dalam pengetikan rumus ataupun fungsi Excel,

2. Terdapat kesalaham dalam mengetikan argument teks pada rumus Excel,

3. Pada saat memisahkan rentang dalam rumus tidak menyertakan tanda titik dua (:), dll

Jadi, pada nomor 25 Pesan error #NAME pada sel di Excel, karena salah penulisan rumus pada sel tersebut (a)

26. Cara memperbaiki pesan eror #VALUE pada sel di Excel adalah…

Pesan error #VALUE muncul dikarenakan terdapat kesalahan saat dalam perhitungan rumus yang dikerjakan dan kesalahan dalam mengambil data yang direferensikan. Cara memperbaikinya, yaitu dengan mengetikkan kembali referensi data atau dengan menggunakan fungsi ISERROR(value).

Jadi, pada nomor 26 cara memperbaiki pesan eror #VALUE pada sel Excel adalah dengan menampilkan kembali cel referensi dalam rumusnya (c)

27. Cara memperbaiki pesan eror #NULL pada sel di Excel adalah…

Pesan error #VALUE muncul dikarenakan terdapat kesalahan dalam menuliskan/menempatkan cell range atau rentang sel pada saat memasukan rumus di Excel. Rentang sel di dalam Excel ditandai dengan tanda titik dua (:).

Jadi, pada nomor 27 cara memperbaiki pesan eror #NULL pada sel di Excel adalah dengan memberikan tanda titik dua (:) pada rentang antara satu sel dengan sel yang lainnya (c)

28. doc, xls, ods, pdf, txt, ppt termasuk file…

doc: merupakan singkatan dari dokumen, salah satu format file yang digunakan pada microsoft word.

xls: merupakan format file yang digunakan pada microsoft excel

ods: merupakan salah satu format dari word processing, format file ini mirip dengan doc

pdf: merupakan singkatan dari portable document format, salah satu format dokumen pada adobe system

txt: merupakan salah satu format dokumen yang adap pada notepad

ppt: merupakan singkatan dari power point, salah satu format file yang digunakan pada microsoft power point

Jadi, pada nomor 28 doc, xls, ods, pdf, txt, ppt termasuk file dokumen (a)

29. File ini menunjukkan penggunaan aplikasi pengolahan data atau angka, yaitu….

Salah satu aplikasi pengolahan data atau angka, yaitu Microsoft excel dan spreadsheet, format file pada software tersebut adalah xls. Sehingga, file tersebut termasuk ke dalam file dokumen

Jadi, pada nomor 29 file ini menunjukkan penggunaan aplikasi pengolahan data atau angka, yaitu file dokumen (a)

30. Tanda yang membedakan antara satu jenis file liannya disebut…

File merupakan  sebuah data-data yang tersimpen, file juga dapat diartikan sebagai arsip ataupun berkas. File tersimpan di dalam komputer pada folder-folder terntentu, file ternagi menjadi beberapa jenis sesuai dengan ekstensinya. Ekstensi file merupakan suatu tanda yang membedakan antara satu jenis file dengan lainya.

Jadi, pada nomor 30 tanda yang membedakan antara satu jenis file liannya disebut dengan Ekstensi file (b)

31. Berikut yang tidak termasuk ke dalam jenis-jenis file adalah….

Ekstensi file merupakan suatu tanda yang membedakan antara satu jenis file dengan lainya. Berikut merupakan jenis-jenis file;

file dokumen, file system, file gambar, file suara, dan file video

Jadi, pada nomor 31 tidak ada jawaban yang benar karena pada pilihan ganda semuanya termasuk ke dalam jenis-jenis file

32. Berikut yang tidak termasuk fungsi logika dalam excel adalah…

Terdapat beberapa fungsi logika di dalam Excel, yaitu fungsi OR, fungsi AND,  fungsi NOT, dan fungsi IF

Jadi, pada nomor 32 yang tidak termasuk fungsi logika dalam excel adalah VLOOKUP (d)

33. Berikut yang tidak termasuk ke dalam jenis-jenis file adalah….

Ekstensi file merupakan suatu tanda yang membedakan antara satu jenis file dengan lainya. Berikut merupakan jenis-jenis file;

file dokumen, file system, file gambar, file suara, dan file video

Jadi, pada nomor 31 tidak ada jawaban yang benar karena pada pilihan ganda semuanya termasuk ke dalam jenis-jenis file

34. Komputasi adalah…

Komputasi merupakan salah satu cara untuk memecahkan permasalahan dari data input menggunakan algoritma berdasarkan data yang sudah diinput

Jadi, pada nomor 34 komputasi adalah algoritma untuk memecahkan masalah yang berasal dari input yang berasal dari luar system komputer (c)
