7. jawaban dialogue 1 hal 25-26 buku paket bahasa inggris kelas 9 smp​


Sebelum kita menyelesaikan dialog diatas sebaiknya kita memahami mengenai expressing purpose menggunakan in order to dan so that.

Expressing Purpose adalah kalimat yang digunakan untuk menunjukkan tujuan atau sesuatu yang dimaksudkan pada klausa kedua pada kalimat. Jika diterjemahkan secara bebas dalam bahasa Indonesia artinya 'supaya', 'agar' dan 'sehingga.'

Observing and Asking Question

Edo's Advice

Dialog 1

Dayu : "what should we avoid smoking for?

Udin : "to have stronger lungs"

Dialog 2

Beni : "What for do we need to exercise regularly?"

Udin : "so that we have a strong muscle"

Dialog 3

Siti : "What should we have to follow the safety instruction for?"

Udin : "in order to avoid muscle injury"

Lina's Advice

Dialog 1

Siti : "What for do we need to have a health check up regularly?"

Lina : "in order not to get a serious disease"

Dialog 2

Edo : "What for do we need to eat a healthy diet?

Lina : "so that we will stay healthy"

Udin's Advice

Dialog 1

Siti : "What for do we need to exercise regularly?"

Edo : "so that we will stay healthy"

Dialog 2

Dayu : "What for do we need to wash our hand before eat?"

Udin : "in order to get rid of dirt"

Dialog 3

Lina :"What should we clean our house and bathroom for?"

Edo : "so that there are no mosquito in hour house"

Dialog 4

Beni : "What for do we need to wash and cook our food well?"

Edo : " in order not to get harmful bacteria"
