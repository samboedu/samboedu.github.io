a) Kecepatan
perpindahan / waktu --> (m/s)  -> LT⁻¹

b) Percepatan
kecepatan/waktu --> (m/s) / (s)  --> (m/s²) -> LT⁻²

c) Gaya
massa x percepatan --> (kg) x (m/s²) --> (kgm/s²) -> MLT⁻²

d) Usaha
gaya x perpindahan --> (kgm/s²) x (m) --> (kgm²/s²) -> ML²T⁻²

e) Daya
usaha/waktu --> (kgm²/s²) / (s) --> (kgm²/s³) -> ML²T⁻³

f) Tekanan
gaya/luas --> (kgm/s²) / (m²) --> (kg/ms²) -> ML⁻¹T⁻²

g) Energi potensial
massa x gravitasi x tinggi --> (kg) x (m/s²) x (m) --> (kgm²/s²) -> ML²T⁻²

h) Energi kinetik
1/2 x massa x kecepatan² --> (kg) x (m/s)² --> (kgm²/s²) -> ML2T⁻²

i) Momentum
massa x kecepatan --> (kg) x (m/s) --> (kgm/s) -> MLT⁻¹

j) Impuls
gaya x waktu --> (kgm/s²) x (s) --> (kgm/s) -> MLT⁻¹
