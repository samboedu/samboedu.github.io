9. aplikasi sas 2017 bagaimana cara menjalankan nya


Sebelum melakukan penginstalan, lakukan backup SAS 2016. Backup dilakukan jika terjadi masalah dalam instalasi Aplikasi SAS 2016. (Sejauh ini masih aman).
Kemudian Klik kanan file Installer, pilih Run As Administrator, baik itu Install Aplikasi maupun Database di atas.
Ikuti petunjuknya dan langkah-langkah dalam instalasi. Jika yang dipakai adalah laptop/PC lama yang sebelumnya sudah ter install Aplikasi SAS 2016, jangan mencenteng untuk Install ODBC MySQL langsung klik Selesai, atau pilih Repair jika terlanjur melanjutkan Instalasi.
Selanjutnya Anda masuk ke aplikasi SAS menggunakan user: admin dan password: admin
Pilih Referensi I, lalu pilih Satker anda lalu ceklist untuk default. Pilih juga KPPN default tempat pembayaran anda.
Keluar dari aplikasi SAS dan masuk lagi menggunakan user dan password admin.
Lakukan Transfer Pagu DIPA 2017.
Lakukan Perekaman pada referensi I (Buka & Copas dari SAS 2016) dimulai urutannya dari Kode PPK, BPP, Buat User yg ada, No.SPP, Bank Pos, Pejabat, No.Kontrak dan Alamat Kantor.
Untuk pengisian Supplier > masuk menu Utility > Ambil Data Supplier 2016 > masuk dengan salah satu User dan Password SAS 2016 > klik Ambil.
Isi Dokumen Pengesahan jika perlu > menu Referensi II > Dokumen Pengesahan > Rekam UU APBN No. 18 Tahun 2016. (Sebagai Dasar Pengeluaran pada pembuatan SPP).

10. nilai dari 1/2017-2/2017+3/2017-4/2017...+2001/2017-2002/2017 = a. -1/2017 b. -1000/2017 c. -1001/2017 d. 1000/2017 e. 1001/2017


1/2017 - 2/2017 + 3/2017 - 4/2017 + ... + 2001/2017 - 2002/2017
= (1 - 2 + 3 - 4 + .... + 2001 - 2002) / 2017
= (-1 + (-1) + .... + (-1)) / 2017
= (-1 . 1001) / 2017
= -1001/2017

(-1) + (-1) + .... + (-1) ====> (-1) nya sebanyak (2002/2) = 1001

20. Tuliskan nama menu menu utama pada aplikasi Microsoft word 2017


MS Word adalah sebuah aplikasi pengolah kata yang dikeluarkan oleh perusahaan Microsoft sebagai bagian dari aplikasi MS Office.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan menu-menu yang terdapat di MS Word beserta fungsinya. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

Menu MS Word

File; menyajikan instruksi dasar seperti New, Save, Save As, Import from, Option dsb.

Edit; menyajikan instruksi terkait modifikasi dokumen seperti Insert dsb.

View; menyajikan instruksi terkait penampilan dokumen

Add-On; menyajikan instruksi terkait aplikasi yang dihubungkan dengan MS Word

Design; menyajikan instruksi terkait unsur desain yang dapat ditambahkan kepada dokumen

Layout; menyajikan instruksi terkait tata letak dokumen

Reference; menyajikan instruksi terkait pembuatan dan pengelolaan rujukan pada dokumen

Mailings; menyajikan instruksi terkait pembagian dokumen melalui surat elektronik

Review; menyajikan instruksi terkait pemeriksaan dokumen seperti kesalahan penulisan, tata bahasa, dan lain sebagainya.
