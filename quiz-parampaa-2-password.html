11. QUIZ MATEMATIKA Matematika Diskrit ^_^ ^_^ ^_^ Anda disuruh untuk membuat password suatu login pada sistem komputer yang panjangnya empat sampai sembilan karakter. Tiap karakter boleh berupa angka atau huruf (tidak dibedakan huruf besar dan huruf kecil). Buktikan bahwa banyak password yang dapat Anda buat untuk suatu login pada sistem komputer adalah 104.461.669.668.096 password!!!


Dengan huruf besar dan huruf kecil tidak dibedakan, maka dalam variasinya terdapat 26 bentuk alfabetik, serta tambahan berupa angka (dari 0 sampai 9) yang berjumlah 10 bentuk, sehingga dalam satu karakter memuat (26+10 = 36) kemungkinan karakter.

Untuk kasus 4 karakter, dengan tidak ada syarat khusus, banyaknya kombinasi:
n₄ = 36 x 36 x 36 x 36 = 36⁴

Hal tersebut serupa untuk 5 sampai 9 karakter, dengan:
nₓ = 36ˣ, untuk 5 ≤ x ≤ 9

Sehingga, jumlah kombinasi yang mungkin untuk sebuah password tersebut adalah:
n = n₄ + n₅ + n₆ + n₇ + n₈ + n₉
n = 36⁴ + 36⁵ + 36⁶ + 36⁷ + 36⁸ + 36⁹

Untuk mempermudah penghitungan, akan digunakan sifat deret geometri, dengan a = 36⁴, r = 36, dan n = 6, diperoleh:
n = S₆
n = 36⁴ (36⁶ - 1) / (36 - 1)
n = 1.679.616 x 2.176.782.335 / 35
n = 1.679.616 x 62.193.781
n = 104.461.669.668.096 kombinasi

14. [Quiz Rimuru] Rimuru ingin membuat sebuah password untuk laptopnya, password itu adalah bilangan 5 angka abcde yang dibentuk dari angka angka 0 sampai 9. Jika diketahui a < b dan a tidak sama dengan 0. Berapa banyak password laptop yang bisa dibuat oleh Rimuru ? Note : Jawab dengan sungguh sungguh, jika jawabannya main main akan segera Rimuru report


Banyak susunan password yang bisa dibuat Rimuru ada 36.000 kombinasi

Pembahasan:

Kaidah pencacahan adalah metode untuk menghitung banyak susunan yang memungkinkan. Salah satu metode kaidah pencacahan adalah filling slots/pengisian tempat. Prinsipnya mengalikan banyak kejadian yang mungkin dari tiap penyusunnya.

Diketahui:

Password terdiri dari 5 angka abcde

Angka yang digunakan 0-9

Angka boleh digunakan berulang

a < b dengan a ≠ 0

Ditanya:

Banyak kombinasi password yang dapat dibentuk ...

Penyelesaian:

Pertama-tama hitung ada berapa cara menyusun ab dengan a < b dan a ≠ 0.

Jika b = 1, banyak cara a = 0 (tidak ada nilai a yang memenuhi karena a tidak boleh 0)

b = 2, banyak cara a = 1 (a = 1)

b = 3, banyak cara b = 2 (a = 0, 1)

...

b = 8, banyak cara a = 7 (a = 1, 2, 3, 4, 5, 6, 7)

b = 9, banyak cara a = 8 (a = 1, 2, 3, 4, 5, 6, 7, 8)

Banyak cara a dan b yang memenuhi:

[tex]1 + 2 + 3 + ... + 7 + 8 [/tex]

[tex] = (1 + 8)( \frac{8}{2} )[/tex]

[tex] = (9)(4)[/tex]

[tex] \boxed{= 36 \: susunan}[/tex]

Sekarang kita tinggal menghitung banyak susunan pada cde.

Angka yang digunakan 0 - 9 ===> ada 10 angka

Banyak cara c = 10

Banyak cara d = 10

Banyak cara e = 10

Banyak cara menyusun cde:

[tex]10 \times 10 \times 10[/tex]

[tex] \boxed{= 1000 \: susunan}[/tex]

Banyak susunan abcde:

[tex]36 \times 1000[/tex]

[tex] \boxed{ \boxed{= 36.000 \: susunan}}[/tex]

Kesimpulan:

Jadi, banyak susunan password yang dapat dibuat Rimuru ada 36.000 susunan

17. Apa fungsi password pada Edmodo dan apa akibatnya jika orang lain mengetahui password tersebut?


PASSWORD

Fungsi passoword pada Edmodo dan akibatnya jika orang lain mengetahui password tersebut.

Jawaban

Fungsi password pada Edmodo adalah untuk melindungi akun Edmodo kita sehingga tidak disalahgunakan oleh orang yang tidak bertanggung jawab. Jika orang lain mengetahui password kita maka orang tersebut dapat melakukan akses ke dalam akun Edmodo kita dan melakukan apapun terhadap akun tersebut tanpa seizin dari pemilik akun.  

Pembahasan

Password adalah serangkaian karakter yang merupakan kata sandi yang harus dimasukkan kedalam suatu sistem agar kita dapat melakukan akses ke dalam sistem tersebut. Password bersifat rahasia. Password pada dasarnya adalah pelindung agar orang lain tidak seenaknya melakukan akses ke dalam ranah privasi kita.

Password sangatlah penting dalam melindungi hal – hal privasi kita dari orang – orang yang tidak bertanggung jawab. Misalnya saja password email. Jika ada orang yang mengetahui password email kita, maka orang tersebut dapan melakukan akses ke dalam email yang kita miliki. Mereka dapat melihat, membaca, bahkan mendapatkan akses penuh yang sama seperti kita. Bayangkan jika ada data rahasia di dalam email kita dan dicuri oleh orang – orang tersebut. Hal itu akan sangat merugikan kita.

Saat ini, di era globalisasi dan semakin canggihnya teknologi, kebutuhan akan keamanan data sangat tinggi. Password yang pada awalnya hanya berupa karakter, saat ini semakin berkembang. Beberapa cotoh pengenal identitas yang canggih seperti penggunaan scanner sidik jari dan retina, atau face recognition, adalah beberapa contoh semakin majunya teknologi untuk mencegah pencurian data.
