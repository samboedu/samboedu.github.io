16. Tono massanya 35kg meloncat dengan kecepatan 3 m/s dari sebuah perahu kecil yg diam. jika massa perahu 3kg, berapa kecepatan perahu setelah tono melompat


m₁ = 35 kg
v₁' = 3 m/s
m₂ = 3 kg
v₂' = ..........?

Penyelesaian :

Hukum Kekekalan Momentum
P sebelum = P sesudah
m₁ • v₁ + m₂ • v₂ = m₁ • v₁' + m₂ • v₂'
35 • 0 + 3 • 0 = 35 • 3 + 3 • v₂'
0 = 105 + 3v₂'
-105 = 3v₂'
v₂' = -105/3
v₂' = -35 m/s (tanda negatif berarti gerak perahu berlawanan dengan gerak Tono)

Jadi, kecepatan perahu setelah tono melompat adalah 35 m/s
