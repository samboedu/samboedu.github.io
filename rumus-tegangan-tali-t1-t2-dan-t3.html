8. sebuah benda dengan massa 2 kg digantung pada dinding menggunakan tali seperti pada gambar berikut. Tentukan nilai tegangan tali T1, T2, dan T3


Nilai tegangan tali T1, T2, dan T3 berturut-turut adalah 10√3 N, 10 N, dan 20 N.

Pembahasan

Pada kasus ini, benda berada dalam keadaan diam, baik di arah horizontal (sumbu x) maupun arah vertikal (sumbu y) sehingga

ΣFx = 0 N dan ΣFy = 0 N

Untuk mendapatkan nilai tegangan, kita perlu meninjau gaya resultan pada setiap arah satu-persatu.  

Gaya-gaya di arah horizontal (sumbu x)

Pertama kita membuat aturan terlebih dahulu, yakni bahwa bila gaya mengarah ke kiri maka dia bernilai negatif, sementara bila mengarah ke kanan maka ia bernilai positif. Sehingga,

ΣFx = 0 N

-T1. cos 60° + T2 cos 30° = 0

-T1. 0,5 + T2. 0,5√3  = 0

T1.0,5 = T2. 0,5√3  

T1 = T2√3 ... (1)

Gaya-gaya di arah vertikal (sumbu y)

Selanjutnya, untuk menghitung resultan gaya di arah vertikal (sumbu y), kita juga perlu membuat aturan terlebih dahulu, yakni bahwa bila gaya mengarah ke bawah maka dia bernilai negatif, sementara bila mengarah ke atas maka ia bernilai positif. Sehingga,

ΣFy = 0 N

-W + T3 = 0

T3 = W

T3 = m. g

T3 = 2 kg. 10 m/s²

T3 = 20 N

Selanjutnya, T3 dibagi menjadi 2, yakni T1 dan T2.

ΣFy = 20 N

T1. sin 60° + T2 sin 30° = 20 N

T1. 0,5√3 + T2. 0,5  = 20 N, bagi kedua ruas dengan 0,5,

T1√3 + T2  = 40 N  

T2 = 40 N - T1√3 ... (2)

Mencari nilai T1

Dari persamaan (1) dan (2),

T1 = T2√3 ... (1)

T2 = 40 N - T1√3 ... (2)

Maka,

T1 = (40 N - T1√3) √3  

T1 = 40√3 N – 3 T1

T1 + 3 T1 = = 40√3 N

4 T1 = = 40√3 N

T1 = 10√3 N

Mencari nilai T2

Dari persamaan (1),

T1 = T2√3 ... (1), maka

T2 = T1 / √3

T2 = 10√3 N / √3

T2 = 10 N

9. Sistem tersebut dalam keadaan seimbang. Batang XY tidak bermassa berengsek pada titik X. Jika percepatan gravitasi sebesar 10 m/s², tegangan tali T1, T2, dan T3 dinyatakan dengan hubungan...a. T1 < T2 < T3b. T1 < T3 < T2c. T2 < T1 < T3d. T2 < T3 < T1e. T3 < T1 < T2​


Tegangan tali T₁ , T₂ , dan T₃ dinyatakan dengan hubungan:

a. T₁ < T₂ < T₃

PEMBAHASAN

Momen gaya adalah efek putar dari sebuah gaya yang di definisikan dengan rumus:

τ = r . F

dimana:

τ = momen putar ( N.m )

r = lengan gaya (m)

F = gaya yang tegak lurus dengan lengan gaya (N)

Kecepatan sudut adalah besarnya sudut yang ditempuh dalam selang waktu tertentu.

ω = ΔΘ / t

dimana :

ω = kecepatan sudut ( rad/s )

ΔΘ = sudut yang ditempuh ( rad )

t = waktu tempuh ( s )

Percepatan sudut adalah besarnya perubahan kecepatan sudut yang terjadi dalam selang waktu tertentu.

α = Δω / t

dimana :

α = percepatan sudut ( rad/s² )

Δω = perubahan kecepatan sudut ( rad/s )

t = waktu tempuh ( s )

Mari sekarang kita mencoba menyelesaikan soalnya.

Mula - mula kita gambarkan gaya - gaya yang bekerja pada Batang XY dan titik simpul Z seperti terlihat pada gambar di lampiran.

Pada batang XY , jika kita meninjau titik acuan X , maka :

[tex]\Sigma \tau_X = (T_2 \times L \times \sin 90^o) - (T_3 \times L \times \sin 53^o)[/tex]

[tex]0 = (T_2 \times L \times \sin 90^o) - (T_3 \times L \times \sin 53^o)[/tex]

[tex]T_2 \times L \times \sin 90^o = T_3 \times L \times \sin 53^o[/tex]

[tex]T_2 \times 1 = T_3 \times 0,8[/tex]

[tex]T_2 = 0,8 ~ T_3[/tex]

Ini artinya T₂ < T₃

Jika kita meninjau titik simpul Z , maka:

[tex]\frac{T_2}{\sin 90^o} = \frac{T_1}{\sin 135^o}[/tex]

[tex]\frac{T_2}{1} = \frac{T_1}{1 / \sqrt{2}}[/tex]

[tex]T_2 = \sqrt{2}T_1[/tex]

Ini artinya T₁ < T₂

Kesimpulan :

T₁ < T₂ < T₃
