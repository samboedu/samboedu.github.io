Jawaban :

Harus membayar Rp 17.000,00

Penjelasan dengan langkah-langkah:

diketahui :

buku : x

pulpen : y

8x + 4y = 26.000...1

8x + 4y = 26.000...16x + 5y =22.500...2

ditanya : 5x + 3y = ?

8x + 4y = 26.000 || x5 || 40x + 20y = 130.000

6x + 5y =22.500 || x4 || 24x + 20y = 90.000

_________________ -

16x = 40.000

x = 2500

Subtitusikan x ke persamaan kedua.

6x + 5y = 22.500

6.(2500) + 5y = 22.500

15.000 + 5y = 22.500

5y = 22.500-15.000

5y = 7500

y = 1500

5x + 3y = 5.(2500) + 3.(1500)

=12.500 + 4500

= 17.000

Jadi Defikasari harus membayar Rp 17.000,00
