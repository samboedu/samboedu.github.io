17. rumus fungsi permintaan dan penawaran


Bentuk umum fungsi permintaan dengan dua variabel adalah sebagai beriut :

Qd = a - bPd    atau  Pd = -1/b ( -a + Qd)

 

dimana :

a dan b = adalah konstanta, dimana b harus bernilai negatif

b = ∆Qd / ∆Pd

Pd        = adalah harga barang per unit yang diminta

Qd       = adalah banyaknya unit barang yang diminta

Syarat, P  ≥  0, Q ≥  0, serta dPd / dQ < 0

 
Bentuk umum dari fungsi penawaran linear adalah sebagai berikut:

Qs = a + bPs

 

dimana :

a dan b = adalah konstanta, dimana b harus bernilai positif

 

b = ∆Qs/ ∆Ps

Ps = adalah harga barang per unit yang ditawarkan

Qs = adalah banyaknya unit barang yang ditawarkan

Ps≥ 0, Qs≥ 0, serta dPs/ dQs > 0

21. RUMUSKAN FUNGSI PERMINTAAN YANG DIHADAPI PERUSAHAAN


Fungsi permintaan P=200 - 3Q dengan fungsi biayanya C(Q)=75 + 80Q - Q2, dimana 0 ≤ Q ≤ 40. Maka nilai Q dan P yang akan memaksimumkan laba perusahaan dan besarnya laba maksimum tersebut adalah 1725

Jika pemerintah mengenakan pajak sebesar $4 per unit Q yang diproduksi, maka harga baru yang akan memaksimumkan keuntungan perusahaan tersebut adalah sebesar $113

"Pembahasan"

P = 200 - 3Q

C = 75 + 80Q - Q²

Nilai Q dan P yang akan memaksimumkan laba perusahaan dan besarnya laba maksimum perusahaan adalah

TR = P x Q

= (200 - 3Q) Q

= 200Q - 3Q²

MR = TR'

MR = 200 - 6Q

MC = C'

C = 75 + 80Q - Q²

MC = 80 - 2Q

Profit maksimum didapat ketika MR = MC, maka:

200 - 6Q = 80 - 2Q

200 - 80 = 6Q - 2Q

120 = 4Q

Q = 120/40

Q = 30

Fungsi permintaan perusahaan P = 200 - 3 Q

P = 200 - 3 (30)

= 200 - 90

= 110

Maka, kuantitas dan harga yang dapat memaksimumkan profit adalah

P = 110 dan Q = 30.

Laba maksimum

TR - TC

200Q - 3Q² - (75 + 80Q - Q²)

200 (30) - 3 (30)² - 75 - 80 (30) + 30²)

6.000 - 3 (900) - 75 - 2400 + 900

1725

Profit maksimum yang dapat diperoleh perusahaan dengan P dan Q adalah 1725.

Jika pajak yang ditetapkan pemerintah sebesar $4 per unit, maka:

TCt = TC + Q

= 75 + 80Q - Q² + t.Q

= 75 + 80 Q - Q² + 4Q

= 75 + 84Q - Q²

MCt = TCt'

MCt = 84 - 2Q

Untuk mendapatkan laba maksimum MR = MCt

200 - 6Q = 84 - 2Q

200 - 84 = 6Q - 2Q

116 = 4Q

Q = 29

Masukkan nilai Q ke dalam fungsi permintaan P = 200 - 3Q, maka:

P = 200 - 3 (29)

= 200 - 87

= 113

Jadi harga baru setelah ketetapan pajak 4$ /unit oleh pemerintah adalah sebesar 113$.

Harga barang dianggap sebagai faktor terpenting dan sering dijadikan acuan untuk melakukan analisis penawaran. Harga berbanding lurus dengan jumlah penawaran.

Penawaran dipengaruhi oleh beberapa faktor antara lain:

Harga barang

Tingkat teknologi

Jumlah produsen di pasar

Harga bahan baku

Harapan, spekulasi, atau perkiraan dimasa depan

Kesepakatan harga pasar terbentuk melalui proses tawar menawar antara penjual dan pembeli. Hasil tawar menawar antara penjual dan pembeli dinamakan harga pasar, biasanya dalam ekonomi disebut dengan istilah harga keseimbangan (Equilibrium Price).

Permintaan adalah sejumlah barang yang dibeli atau diminta pada suatu harga dan waktu tertentu. Sedangkan penawaran adalah sejumlah barang yang dijual atau ditawarkan pada suatu harga dan waktu tertentu


22. tuliskan rumus fungsi permintaan​


Fungsi permintaan (Qd) berbentuk

Qd a - bP

___________________________

Untuk mencari Qd, kita dapat menggunakan persamaan berikut ....

Q - Q1 / Q2 - Q1 = P - P1 / P2 - P1

Q1 = jumlah barang awal

Q2 = jumlah barang akhir

P1 = harga barang awal

P2 = harga barang akhir

____________________

Contoh Soal :

Diketahui :

P1 = Rp40/unit

Q1 = 20 unit

P2 = Rp30/unit

Q2 = 22 unit

Ditanya :

Qd = ?

Jawab :

Q - Q1 / Q2 - Q1 = P - P1 / P2 - P1

Q - 20 / 2 = P - 40 / -10

-10Q + 20 = 2P - 80

-10Q = 2P - 80 - 20

-10Q = 2P - 100

Q = -0,2P + 10

Q = 10 - 0,2P ✔️

Jadi, fungsi permintaan barang tersebut adalah Qd = 10 - 0,2P
