10. Cari tutorial atau cara membuat siluet gambar di Photoshop!


Cara Membuat Foto Siluet

Di bawah ini, cara membuat foto siluet yang akan kita bahas dengan menggunakan bantuan aplikasi berupa Photoshop. Berikut langkah-langkahnya:

Langkah pertama untuk menerapkan cara membuat foto siluet menggunakan Photoshop yaitu dengan membuka aplikasi Photoshop di PC. Jika belum punya, install dulu program aplikasi Photoshop di PC yang sobat miliki. Jika sudah, buka aplikasinya.

Jika sudah terbuka maka buatlah sebuah dokumen baru pada Photoshop dengan rincian Width 600 px, Height 600 px, Resolution 300 px dengan Color Mode RGB Color 8 bit. Kemudian klik Ok.

Nah, beranjak pada proses cara membuat foto siluet selanjutnya, pilih menu File dan Open untuk memilih foto yang nantinya kan kita olah menjadi silut.

Menginjak pada langkah selanjutnya, hal yang perlu sobat perhatikan yaitu memastikan bahwa layer fotonya aktif. Jika sudah, tekan Ctrl + Shift + L untuk membuat obyeknya sedikit gelap namun backgroundnya sedikit terang.

Kemudian, cara membuat foto siluet berikutnya, gunakan tool Quick Selection Tool (W) yang berada di bagian samping kiri dari pilihan menu edit untuk mengambil objek yang akan dijadikan siluet.

Selanjutnya, tekan Ctrl + J untuk menjadikan obyek foto yang dipilih menjadi sebuah layer baru. Jika sudah hilangkan icon mata pada layer foto yang letaknya berada di menu kanan bawah.

Cara membuat foto siluet berikutnya, Klik 2x “layer 1” untuk memunculkan jendela layer effects, kemudian pilih “Color Overlay” dan ganti warnanya menjadi hitam, setelah itu klik ok. Maka warnanya akan menjadi hitam.

Menginjak langkah berikutnya, pilih tool Pollygonal Lasso Tool (L) & Eraser Tool (E) untuk menyempurnakan bagian pinggir-pinggir foto. Edit bentuknya sesuai dengan selera. Sampai di langkah ini, cara membuat foto siluet hampir selesai.

Untuk memberikan kesan yang lebih elegan dan menari sobat bisa meletakkan foto yang sudah diedit menjadi siluet tadi ke bagian center figura dengan sedikit tulisan keterangan. Dalam hal ini, sobat bisa menghiasnya sesuai dengan selera.

14. Tuliskan 2 cara membuat dokumen baru di photoshop


Jawaban:

Berikut 2 cara yang bisa saya berikan.

1. Buka menu File - New

Saya cukup yakin bahwa cara yang pertama ini adalah yang paling populer digunakan yaitu dengan mengakses menu File - New untuk membuat dokumen baru atau dengan menggunakan tombol Ctrl + N pada keyboard.

cara membuat dokumen baru di Photoshop melalui menu

Ketika menu atau pintasan tersebut dipilih, maka pengaturan dokumennya pun akan langsung ditampilkan dan bisa disesuaikan dengan pilihan kalian mau mengaturnya seperti apa terutama terkait nama file, ukuran dan satuannya, resolusi serta pilihan warna background.

Cara mengatur dokumen baru di Photoshop

Setelah klik Ok pada pengaturan gambar diatas, maka dokumen barunya pun bisa langsung ditampilkan. Mudah, praktis, dan paling populer. Itulah apa yang menjadi kelebihan cara pertama ini.

2. Membuat dokumen baru dengan cara men-duplikat layer

Untuk cara kedua, cara menggunakannya mensyaratkan hal tertentu yaitu sudah harus ada dokumen yang terbuka sebelumnya. Baik itu membuka foto ataupun dokumen baru seperti cara pertama diatas.

Yang penting ada file yang ditampilkan.

Untuk contohnya, saya menggunakan file dokumen yang saya buat dengan cara diatas dimana langkah pertamanya adalah klik kanan pada layer Background, lalu pilih Duplicate Layer.

Tapi jangan gunakan proses duplikat layer menggunakan tombol Ctrl + J ya karena hal itu tidak akan bisa digunakan pada teknik pembuatan dokumen baru pada tahap ini.

Pokoknya klik kanan pada layer apapun termasuk layer foto kalau ada lalu pilih Duplicate Layer.

cara membuat dokumen baru Photoshop dengan duplikat layer

Setelah itu, akan muncul jendela pengaturan layer lalu lakukan perubahan pada Destination Document dengan memilih New serta ubah Name (nama) layernya sesuai pilihan kalian.

Pengaturan dokumen baru hasil Duplicate Layer Photoshop

Jika sudah dan telah memilih Ok, maka akan muncul tab dokumen baru yang berisi layer yang diduplikat tersebut.

Yang menarik dari cara yang kedua ini adalah, tidak perlu dilakukan pengaturan pada ukuran dokumen seperti cara pertama. Untuk dimensi dan resolusinya, akan mengikuti pengaturan layer pada dokumen asal sebagai referensi utamanya.

Coba saja cek dengan memilih menu Image - Image Size atau Canvas Size. Hasilnya sama dengan pengaturan pada dokumen utama yang diduplikat.

19. Bagaimana cara membuat gambar animasi/ kartun di photoshop? tutorialnya?


caranya cari di youtube semua ada disitu

Setelah Photoshop Anda aktif, silakan buat canvas baru dengan cara klik File > New. Beri nama canvas “Animasi” Width 350 pixels, Height 350 pixels dan Resolution cukup 72 pixels.

Gambar yang saya buat dalam tutorial membuat animasi ini menggunakan gambarCustom Shape Tool. Silakan Aktifkan Shape Toolnya, dan disini saya memilih gambarFlower 7.

Setelah memilih Shape seperti yang dicontohkan, sekarang buat gambar Flower di area canvas Animasi. Maka otomatis, di Pallet Layers akan bertambah sebuah Shape baru.

Aktifkan menu Animation dari Window. Animation Frames ini fungsinya untuk mengoperasikan projek Animasi Anda.

Sekarang set waktu durasi animasinya dengan cara klik pada Selects frame delay time. Cukup tentukan 0.1 seconds saja. Selanjutnya perbanyak Frames dengan mengklikDuplicates selested frames, duplikat sebanyak 5 frame.

Setelah melakukan duplikat Frame Animasi, sekarang duplikat juga Layer Animasi sebanyak 5 layer. Untuk menduplikat layer, selain dari Image > Duplicate… bisa juga dengan menekan Ctrl+J pada keyborad.

Sekarang Anda mempunyai lima buah Frame Animasi dan lima buah Layer Animasi. Langkah selanjutnya setting Indicates layer visibility (icon mata) pada setiap Frame Animasi. Maksudnya, jika Frame Animasi 1 aktif, maka Indicates layer visibility juga yang aktif harus Layer Animasi 1. Begitu selanjutnya Anda setting pada Layer animasi lainnya sesuaikan dengan posisi nomor Frame Animasi.

Contoh lain, dalam mengaktifkan Indicates layer visibility lihat gambar dibawah ini :

Jika sudah selesai melakukan pengaturan Indicates layer visibility di semua Layer Animasi dari Layer satu sampai dengan Layer 5, kita lanjutkan tutorialnya.

Aktifkan Frame Animasi 1 dan Layer Animasi 1, klik Move Tool pada Toolbox atau tekan huruf V pada keyboard. Rubah posisi derajat sudutnya menajdi 60.

Lakukan juga pada Frame Animasi 2 dan Layer Animasi 2. Rubah posisi derajat sudutnya menjadi 120. Di Frame Animasi 3 dan Layer Animasi 3 menjadi 180, Frame Animasi 4 dan Layer Animasi 4 240 dan terakhir Frame Animasi 5 dan Layer Animasi 5 menjadi 300.
