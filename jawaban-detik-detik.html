17. 1. 630 liter) detik = .....m³ ) detik 2. 24×2,5 m³/ detik = .......liter / detik 3. 3m³ / detik+1200liter / detik =.......liter/ detik . tolong jawab....


Jika liter di ubah menjadi dm^3 jadi antara 1.630 dm^3=.........m^3 jika disamakan m^3 menjadi 1,63


ingat 1 liter = 1 dm^3

1. 630 liter/detik = 630 dm^2/detik
= 630/1000 m^3/detik
= 0.63m^3/detik

2. 24x2.5 m^3/detik = 60 m^3/detik
= 60 x 1000 dm^3/detik
= 60.000 liter/detik

3. 3m^3/detik + 1200 liter/detik = ....liter/detik
= 3 x 1000 dm^3/detik + 1200 liter/detik
= 3000 liter/detik + 1200 liter/detik
= 3200 liter/detik
