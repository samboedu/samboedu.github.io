5. Dame dame artinya ......​


Dame memiliki beberapa arti, seperti jangan, gak guna, gak baik/bagus.

Pembahasan

だめ (dame)

Ditulis dengan bentuk hiragana, terdiri dari 2 huruf yaitu :

だ (da).め (me).

______

Dan kata だめ (dame) terdapat arti sebagai berikut :

Jangan.Gak guna.Gak bagus.Gak baik.

(Penggunaannya tergantung situasi) kata tersebut merupakan bentuk negatif yang dimana kata ini dipakai agar menyuruh orang lain untuk tidak melakukan apa yang tidak seharusnya dilakukan.

______

Jawaban :

だめだめ (dame dame) berarti 'jangan, jangan', 'gak bagus, gak bagus' (atau arti lainnya).

'Dame dame' artinya adalah...

Jangan, jangan Ga boleh, ga boleh Ngga, ngga

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

DAME「駄目・だめ・ダメ」

Memiliki arti "Jangan", "Ngga boleh", "Ngga bagus", "Payah", dsb.

Kanji = 駄目 Hiragana = だめ Katakana = ダメ

Kata ini jarang ditulis dalam Kanji karena terlalu sulit, dan lebih sering ditulis dalam Hiragana atau Katakana saja yang lebih mudah. Penulisan dalam Katakana memiliki kesan kasual dan informal. Sedangkan penulisan dalam Hiragana sopan atau biasa saja.

Contoh kalimat :

だめです (Dame desu) : Tidak boleh絶対ダメだ！ (Zettai dame da!) : Pokoknya ngga boleh!これはだめだね (Kore wa dame da ne) : Ini ngga bagus俺なんてダメだ (Ore nante dame da) : Aku ini payah俺じゃダメかな (Ore ja dame ka na) : Apa aku tidak cukup untukmu

Selain 'Dame', ada satu kata lagi yang memiliki arti dan penggunaan yang sama, yaitu adalah あかん (Akan). Kata ini merupakan versi dialek Kansai dari 'Dame'. Dialek Kansai digunakan di daerah barat Jepang, seperti Osaka, Kyoto, Kobe, dll.

- - - - - - - - -

Arti 'Dame dame'?

Arti dari 'Dame' sendiri sudah saya jelaskan di atas. Namun jika lebih dari satu, seperti 'Dame dame' atau 'Dame dame dame', ini lebih seperti ke ungkapan larangan. Kalau diartikan seperti "Jangan, jangan", "Ga boleh, ga boleh", dan sebagainya.

9. completez les phrases suivantes avec avoir + enviede! 1.melanie......visiter seseorang grands-parents2.nous n'.......passer notre voyagea Parisplis bantu jawab kakak abang​


Exprimer le désir - avoir envie deMelanie a envie de visiter ses grands-parents.Nous n'avons pas envie de passer de notre voyage à Paris.

ーーーーーーーーーーーー

Explication

Avoir envie de adalah ungkapan dalam bahasa Prancis yang digunakan untuk menyatakan suatu keinginan atau harapan.

Ungkapan ini terdiri dari kata kerja avoir yang berarti mempunyai, kata benda envie yang berarti harapan/keinginan, lalu diikuti preposisi de dan infinitif atau kata benda.

Contoh:

Saya ingin makan. / J'ai envie de manger. Dia ingin sebuah mainan. / Elle a envie d'un jouet.

Karena avoir adalah kata kerja, maka avoir mengalami konjugasi ketika digunakan dalam suatu kalimat.

Konjugasi avoir dalam présent:

j'ai tu as il/elle/on a nous avons vous avez ils/elles ont

20. Dame Dame Dame Yo...Artinya apa​


'Dame dame dame yo' artinya dalam bahasa Indonesia berarti jangan, jangan, jangan lho.

Pembahasan

Dame「駄目・だめ・ダメ」

Jika dalam bahasa Indonesia memiliki arti sebagai berikut ini :

Jangan.Gak.Gak bagus.Payah.Gak boleh.

Kata dame ini lebih umum ditulis dengan bentuk versi hiragana atau juga katakana, sehingga jarang menggunakan versi kanji. Kemudian untuk penggunaannya sesuai situasi tergantung dari versi penulisannya :

Versi hiragana : だめ (dame) -› informal.Versi katakana : ダメ (dame) -› sopan/halus.

yo「よ」

Partikel yang dapat berfungsi ketika ingin menegaskan atau menekankan kalimat di akhir, sehingga jika partikel yo diterjemahkan dalam bahasa Indonesia yang berarti kok, lho, atau lah.

Deskriptif teks adalah suatu teks yang menjelaskan atau mendeskripsikan orang orang, binatang atau suatu benda baik bentuknya, sifat-sifatnya, jumlahnya dan lain-lain. Tujuan dari descriptive text adalah untuk menjelaskan, menggambarkan, atau mengungkapkan seseorang atau suatu benda.

Struktur Descriptive Text (Generic Structure)

Identification (identifikasi) adalah pendahuluan, berupa gambaran umum tentang suatu topik.

Description (deskripsi) adalah berisi ciri-ciri khusus yang dimiliki benda, tempat, atau orang yang dideskripsikan. Misalnya sifat-sifat, tsmpilan fisik, dan hal lain yang dituliskan dengan spesifik.

Ciri-ciri Descriptive Text

• Verb yang digunakan yaitu attribute verb, seperti be (am, is, are)

• Tense yang digunakan yaitu simple present tense

• Hanya fokus pada satu objek tersebut.

Noun : Menggunakan kata benda yang spesifik, seperti misalnya my cat, my boyfriend, National Monument, Selain itu, sering juga menggunakan adjective (kata sifat) untuk memperjelas penggunaan noun atau kata benda, seperti a big house, a smart student, an independence woman.

Simple present tense : menggunakan kata kerja dasar atau bentuk pertama (verb 1) serta menggunakan kata kerja yang dapat menunjukkan kepemilikan atau keadaan sebuah objek. Descriptive text menggunakan simple present tense karena descriptive text menceritakan sebuah fakta dari objek yang dideskripsikan. Misalnya My office has 22 floors, Azka is pretty, dan lain-lain.

Action verbs : Menggunakan kata kerja yang menunjukkan sebuah kegiatan atau sebuah aktifitas yang bisa dilihat. Misalnya, sleep, walk, sing, dance, dll.

Figurative language : Menggunakan bahasa yang figurative atau menggambarkan sesuatu, Biasanya menggunakan sebuah metafora untuk memberikan ilustrasi kepada pembaca.

30. Vous venez d’où? Je ......... de Paris.


Présent de l'indicatif - Venir

- Vous venez d'où ?

- Je viens de Paris.

ーーーーーーーーーーーー

Explication

Pertanyaan vous venez d'où (Dari mana Anda datang) menggunakan kata kerja venir dalam bentuk présent, maka jawabannya juga menggunakan verbe venir yang dikonjugasikan dalam présent untuk subyek je.

Konjugasi venir dalam présent de l'indicatif:

je viens tu viens il/elle/on vient nous venons vous venez ils/elles viennent
