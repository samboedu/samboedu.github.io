Penjelasan dengan langkah-langkah:

[tex] \frac{100mb}{7menit} \frac{ \times70 }{ \times70 } = \frac{7.000mb}{490menit} = \frac{7gb}{8jam \: 10menit} [/tex]

Jadi,waktu yang diperlukan Tima untuk mendownload file sebesar 7GB adalah

490 menit atau setara dengan 8jam 10 menit.

Jawaban:

1. Tidak Gratis (USD $ 29,95)

2. Menggunakan Memori sekitar 9 MB waktu idle dan 20 MB saat proses download.

3. Browsing jadi lebih sering time out karena seluruh bandwidth digunakan oleh Internet Download Manager.

4. Resume capability indicator terkadang tidak akurat. Jika download dari situs yang tidak mendukung resume (seperti download dari rapidshare atau megaupload dengan free account) disitu akan ditampilkan “Unknown” atau “Yes” padahal tidak bisa diresume.

5.Lokasi (temporary) file download tidak jelas dan jika komputer mati mendadak saat proses download proses akan dimulai dari awal bahkan kadang daftar link hilang dari jendela utama Internet Download Manager.

25. Bagaimana cara menggabungkan beberapa file excel menjadi 1 file dalam microsoft excel 2010?


Cara menggabungkan beberapa file excel menjadi 1 file yaitu :

1. Buka semua file yang akan anda gabung

2. Habis itu klik di view lalu cari yang ada tulisan save workspace

3. Beri nama kepada semua file yang mau digabung dan kemudian save

4. Lalu kita buka file excel yang kita save tadi

5. Maka akan terbuka seluruh file excel tadi
