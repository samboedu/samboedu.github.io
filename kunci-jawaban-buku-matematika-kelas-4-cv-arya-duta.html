17. Arya, bagas dan desta membeli buku dan pensil sejenis di toko yang sama. arya membeli 4 buku dan 3 pensil dengan harga rp29.000,00. bagas membeli 1 buku dan 4 pensil dengan harga rp17.000,00. harga yang harus dibayar desta jika ia membeli 3 buku dan 2 pensil adalah sebesar ….


Jawaban:

Rp. 21.000

Penjelasan dengan langkah-langkah:

Diketahui,

buku : x

pensil : y

maka, dapat diubah menjadi

Arya : 4x + 3y = 29.000

Bagas : x + 4y = 17.000

Desta : 3x + 2y

Dijawab :

Menggunakan metode eliminasi, mencari y

4x + 3y = 29.000 (dikali 1)

x + 4y = 17.000 (dikali 4)

menjadi,

4x + 3y = 29.000

4x + 16y = 68.000

-------------------------- —

-13y = -39.000

y = -39.000 : -13

y = 3.000

menggunakan metode subtitusi, mencari x

x + 4y = 17.000

x + 4(3.000) = 17.000

x + 12.000 = 17.000

x = 17.000 - 12.000

x = 5.000

Maka, biaya yang harus dibayar oleh Desta adalah

3x + 2y = 3(5000) + 2(3000) = 15.000 + 6.000 = Rp. 21.000

Jawaban:

Materi : SPLDV

Diketahui :

4 buku + 3 pensil = Rp 7.000

1 buku + 4 pensil = Rp 5.000

Ditanyakan :

Harga 3 buku + 2 pensil yang harus dibayar Desta?

Jawaban :

Model matematika :

Misal :

x => buku

y => pensil

maka :

4x + 3y = 7.000

x + 4y = 5.000

Maka, kita bisa gunakan metode eliminasi caranya dengan mengeliminasi x untuk mendapatkan nilai y :

4x + 3y = 7.000 |×1 |

x + 4y = 5.000 |×4|

4x + 3y = 7.000

4x + 16y = 20.000

_________________ _

-13y = -13.000

y = (-13.000) / (-13)

y = 1.000

subsitusikan y = 1.000 ke :

x + 4y = 5.000

x + 4(1.000) = 5.000

x + 4000 = 5.000

x = 5.000 - 4.000

x = 1.000

maka,

harga 1 buku (x) Rp. 1000

harga 1 pensil (y) juga sama Rp. 1000

maka, harga yang harus dibayar Desta jika :

= 3 buku + 2 pensil

= 3x + 2y

= 3(1.000) + 2(1.000)

= 3.000 + 2000

= Rp 5.000
