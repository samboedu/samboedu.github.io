2. Tulislah struktur source code dalam salah satu IDE untuk merancang dan membangun aplikasi Android?


Salah satu contoh struktur dalam membangun aplikasi android adalah

Folder Nama AplikasiFolder ManifestFolder JavaFolder Res

Struktur source kode di atas merupakan struktur code menggunakan IDE Android studio. Pada folder manifest berisikan file xml yang berfungsi untuk mengatur layout aplikasi kita.

Sedangkan isi dari code yang kita buat berada di folder Java. Untuk file pendukung seperti gambar ikon berada di folder Res.

6. jelaskan source code


source code adalah kumpulan pernyataan atau deklarasi bahasa pemrogramman komputer yang ditulis dan dapat di baca manusia.

Source Code merupakan sebuah program yang biasanya dibuat dalam satu atau lebih file teks, kadang-kadang disimpan dalam database yang disimpan sebagai prosedur dan dapat juga muncul sebagai potongan kode.

source code(atau disebut juga source) adalah kumpulan pernyataan atau deklarasi bahasa pemrogramman komputer yang ditulis dan dapat di baca manusia. 11. apa yang di maksud dengan open source code dan close source code

close source : kode pada sebuah program yang tertutup dan dirahasiakan programmer. sehingga hanya bisa dikembangkan programmer itu sendiri open source code

14. bagaimana caranya menyembunyikan aplikasi tanpa aplikasi di handphone android ?


Cara menyembunyikan aplikasi di Android bisa dilakukan dalam 10 detik.
1. Tinggal buka aplikasi atau menu pengaturan.
2. Setelah itu, tinggal pilih “Applications”.
3. Langkah selanjutnya, kamu hanya perlu pilih “Application Manager”.
4. Pilih tab “All” dan sentuh aplikasi apa aja yang ingin kamu sembunyikan.
5. Kemudian pilih “Disable”.
