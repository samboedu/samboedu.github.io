22. Alina â€¦ song every night.alasan nya di pilih sings apa


PART = SIMPLE PRESENT TENSE - HABITUAL ACTIVITY

------------------

Jawaban:

Alina sings a song every night .

Note/discussion :

We have to use simple present tense form because there is time word which presents the habitual activity / the repetitive activity.  Harap menggunakan pola simple presnt tense karena  terdapat keterangan waktu nya "every night"  yang menunjukkan kebiasaan/ kejadian yang berulang-ulang.  

Penjelasan:

SIMPLE PRESENT TENSEmerupakan tenses yang digunakan mendeskripsikan kejadian nyata di masa lampau, saat ini dan akan tetap selalu sama  di tahun - tahun mendatang.Tenses ini juga menjabarkan fakta kebenaran umum, kebenaran yang tak terbatas dan aktivitas yang terus- menerus berulang  ( kebiasaan/habit/ hobby )

Penggunaan keterangan frekuensi ( Adverb of frequency) yang menjadi ciri khas utama dalam menyatakan habit/kebiasaan dalam simple present tense  :

Example :

Always, usually, often, everyday, seldom, rarely, daily, never, frequently, occasionally , ever, sometimes, regularly.

PATTERN

Struktur kalimat dikategorikan ke dalam 2 jenis, Verbal dan nominal.

1. Rumus Simple present tense ( verbal sentence )

Verbal sentence merupakan kalimat yang predikatnya berupa kata kerja aksi/tindakan.

(+) SUBJECT + VERB (S/ES ) + OBJECT/COMPLEMENT

(-)  SUBJECT + DO/DOES +NOT + VERB 1 + OBJECT /COMPLEMENT

(?) DO/DOES+SUBJECT + VERB 1 + OBJECT / COMPLEMENT?

Example :

(+ ) Dino often reads some books in the library   ( Habit/kebiasaan/ hobby)

( - ) Dino  doesn't often read some books in the library .

(? ) Does Dino often  read some books in the library?

2. Rumus simple present tense ( nominal sentence )

Nominal sentence merupakan kalimat yang predikatnya bukan berupa kata kerja aksi. Pada umumnya berupa kata benda ( noun), kata sifat ( adjective ), dan keterangan ( adverb )

(+ ) SUBJECT + TOBE (IS/AM/ARE)+NOUN/ADJECTIVE/ADVERB

(- ) SUBJECT + TOBE(IS/AM/ARE ) + NOT + NOUN/ADJECTIVE/ADVERB

( ? ) TOBE(IS/AM/ARE ) + SUBJECT+NOUN/ADJECTIVE/ADVERB?

Example :

( + ) Dino is smart.  ( penjabaran karakter, bagian sifat )

( - ) Dino is not smart.

( ? ) Is Dino smart ?

SUBJECT-VERB AGREEMENT Singular third person ( kata ganti orang ketiga tunggal ) : she, he , it .Sedangkan Plural third person ( kata ganti orang ketiga jamak ) : they , we, you.Jika dalam kalimat verbal dalam simple present tense, menggunakan kata ganti orang ketiga tunggal, maka kata kerja nya ditambahkan akhiran -s/es. Sedangkan kata ganti orang ketiga jamak, kata kerja nya tidak ditambahkan akhiran -s/es.Kata kerja  yang mengandung akhiran -ss, -ch, -x, -sh.  Harus ditambahkan -akhiran -es.Exp  : watches, washes, fixes, misses
