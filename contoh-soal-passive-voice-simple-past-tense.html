6. 1. Apa pengertian passive voice? 2.tulis pola (rumus) passive voice dlm present simple, present continuous,past tense 3. Berikan masing2 contohnya


Jawaban:

1. Passive voice adalah suatu bentuk kalimat dimana subjek kalimat menerima aksi, bukan melakukan aksi. Tidak seperti active voice yang fokus terhadap pihak yang melakukan aksi (doer of action), bentuk ini lebih berfokus kepada pihak atau objek yang menerima hasil dari suatu aksi tersebut (receiver of action).

2. Subject + to be (am/is/are) + past participle (pp)

Tense Rumus Passive Voice Contoh Passive Voice

Simple Present am/is/are + pp is created

Present Continuous am/is/are being + pp is being created

Simple Past was/were + pp was created

Past Continuous was/were being + pp was being created

Present Perfect has/have been + pp has been created

Past Perfect had been + pp had been created

Simple Future will be + pp will be created

Future Continuous am/is/are going to be + pp is going to be created

Future Perfect will have been + pp will have been created

2. Present simple : to be ( am,is,are) + V3 . ex : the candy are given to me

- Present continuous ; To be + V3_Ing . ex : Her phone is ringing now

- Past tense : to be ( was,were) + V3 . ex : we were visited the event to show our performance last night

16. Rumus kalimat passive voice dalam bentuk simple present tense,simple past tense, simple future tense.


Simple Present Tense

Aktif : S + do/does + V1

Pasif : S + to be (am, are, is) + V3 + by + O

Contoh:

Aktif : Students speak English

Pasif : English is spoken by students

Simple Past Tense

Aktif : S + V2 + O

Pasif : S + to be (was, were) + V3 + by + O

Contoh :

Aktif : Students spoke English

Pasif : English was spoken by students

Passive voice

1. simple present tense

S (direct object) + is/am/are + past participle +/- by … (agent)

2. simple past tense

S (direct object) + was/were + past participle +/- by … (agent)

3. simple future tense

S (direct object) +will + be + past participle +/- by … (agent)

19. What is the formula of simple past tense in passive voice? ​


Answer:

A. S + Was/were + V3 + by + O

Explanation:

Simple past tense in passive formula:

(+) S + Was/were + V3 + by + O

(-) S + Was/were + not + V3 + by + O

(?) Was/were + S + V3 + by + O

Kalimat aktif subjeknya aktif melakukan aksi dalam kalimat.

Active Voice:

1. Simple Present Tense:

(+) Subject + Verb-1 S/ES + Object

2. Present Continuous Tense:

(+) Subject + Tobe (is/am/are) + Verb-ing + Object

3. Present Perfect Tense:

(+) Subject + Have/Has + Verb-3 + Object

4. Simple Past Tense:

(+) Subject + Verb-2 + Object

5. Past Continuous Tense:

(+) Subject + Tobe (was/were) + Verb-ing + Object

6. Past Perfect Tense:

(+) Subject + Had + Verb-3 + Object

7. Simple Future Tense:

(+) Subject + Will + Verb-1 + Object

8. Future Continuous Tense:

(+) Subject + Will + Be + Verb-ing + Object

9. Future Perfect Tense:

(+) Subject + Will + Have + Verb-3 + Object

Passive Voice:

1. Passive Voice [Simple Present Tense]

-Pattern = Subject + to be (is/am/are) + Verb3

2. Passive Voice [Present Continuous Tense]

-Pattern = Subject + to be (is/am/are) + being + Verb3

3. Passive Voice [Present Perfect Tense]

-Pattern = Subject + have/has + been + Verb3

4. Passive Voice [Simple Past Tense]

-Pattern = Subject + to be (was – were) + Verb 3

5. Passive Voice [Past Continuous Tense]

-Pattern = Subject + to be (was – were) + Being + Verb3

6. Passive Voice [Past Perfect Tense]

-Pattern = Subject + had + been + Verb3

7. Passive Voice [Simple Future Tense]

-Pattern = Subject + will + be + Verb3
