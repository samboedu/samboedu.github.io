Mata Pelajaran: Matematika
Kelas: SD IV
Materi: Bilangan Bulat

a)
Maju 10 langkah → + 10 langkah
Mundur 8 langkah → - 8 langkah
Maju 3 langkah → + 3 langkah

*Kalau bilangan positif terletak di paling depan, maka tidak perlu menuliskan tanda (+)
Nyatakan dalam operasi bilangan bulat:
10 langkah + (-8 langkah) + 3 langkah
= 10 langkah - 8 langkah + 3 langkah
= 2 langkah + 3 langkah
= 5 langkah

b)
Maju → di depan posisi awal
Mundur → di belakang posisi awal
Dari hasil sebelumnya, kita ketahui bahwa posisi Alfin +5 langkah, artinya maju 5 langkah dari posisi awal.
Maka, posisi Alfin sekarang adalah 5 langkah di depan posisi awal.10-8+3=5
posisi awal 0 posisi akhir 5

Mata Pelajaran: Matematika
Kelas: SD ( IV )
Materi: Bilangan Bulat

a)
Maju 10 langkah → + 10 langkah
Mundur 8 langkah → - 8 langkah
Maju 3 langkah → + 3 langkah

*Kalau bilangan positif terletak di paling depan, maka tidak perlu menuliskan tanda (+)
Nyatakan dalam operasi bilangan bulat:
10 langkah + (-8 langkah) + 3 langkah
= 10 langkah - 8 langkah + 3 langkah
= 2 langkah + 3 langkah
= 5 langkah

b)
Maju → di depan posisi awal
Mundur → di belakang posisi awal
Dari hasil sebelumnya, kita ketahui bahwa posisi Alfin +5 langkah, artinya maju 5 langkah dari posisi awal.
Maka, posisi Alfin sekarang adalah 5 langkah di depan posisi awal.
