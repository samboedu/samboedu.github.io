Persamaan adalah bentuk dimana ruas kiri dan ruas kanan memiliki nilai yang sama, dengan kata lain ruas kiri dan ruas kanan adalah sama. Persamaan dilambangkan dengan tanda ( = ).

Bentuk bentuk persamaan yang sering dijumpai

Persamaan Linear

➠ [tex] \boxed{ax + b = 0 } [/tex]

Persamaan Kuadrat

➠ [tex] \boxed{ ax² + bx + c = 0 } [/tex]

Persamaan Nilai Mutlak

➠ [tex] \boxed{ |f(x)| = a } [/tex]

➠ [tex] \boxed{ |f(x)| = |g(x)| } [/tex]

Selain dari bentuk persamaan tersebut, ada juga persamaan logaritma, persamaan irasional, persamaan eksponen dan persamaan yang lainnya.
