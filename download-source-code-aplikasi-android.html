2. Tulislah struktur source code dalam salah satu IDE untuk merancang dan membangun aplikasi Android?


Salah satu contoh struktur dalam membangun aplikasi android adalah

Folder Nama AplikasiFolder ManifestFolder JavaFolder Res

Struktur source kode di atas merupakan struktur code menggunakan IDE Android studio. Pada folder manifest berisikan file xml yang berfungsi untuk mengatur layout aplikasi kita.

Sedangkan isi dari code yang kita buat berada di folder Java. Untuk file pendukung seperti gambar ikon berada di folder Res.

15. kenapa orang indonesia tidak mau download aplikasi android yang belum terkenal?


Mapel : Teknologi Informasi dan Komunikasi

Kategori : Dampak Positif dan Negatif Internet


Mengapa orang Indonesia tidak mau mengunduh aplikasi android yang belum terkenal ?


Saya menjawab dalam 2 sudut pandang,


Sudut pandang pengguna :

Mungkin bukan orang Indonesia saja, bisa saja dunia. Tentu setiap aplikasi memiliki fitur feedback dan rating bukan ?. Di mata pengguna, semakin tinggi nilai rating dari aplikasi tersebut, maka semakin tinggi pula penggunanya. Nah, banyak pengguna yang 'percaya' pada rating aplikasi tersebut dan beranggapan "Wah pasti aplikasinya bagus. Download ah". Banyak aplikasi dengan rating tinggi namun kontennya melanggar ketentuan bukan ?


Atau, aplikasi tersebut terkenal karena aplikasi tersebut sesuai dengan apa yang dibutuhkan masyarakat. Ditambah lagi pembaharuan aplikasi (update) membuat aplikasi tersebut semakin efisien untuk digunakan. Sehingga, yang menggunakan dapat melakukan aktivitas dengan lebih mudah dengan bantuan aplikasi tersebut.


Sudut pandang pengembang :


"Saya udah bikin aplikasi, udah terbit malah. Tapi gak laku-laku ya ?"


Mengapa demikian ?, aplikasi yang dibuat mungkin saja masih memiliki error atau bug (celah) tanpa sepengetahuan sang pengembang. Jika tidak diperbaiki, maka aplikasi tersebut memiliki nilai rating kecil, sehingga pengguna enggan memakai aplikasi tersebut. Seorang pengembang haruslah aktif, jangan sampai masukkan dari pengguna diabaikan begitu saja.


Atau, masyarakat belum membutuhkan aplikasi tersebut, makanya menjadi minim download.


Lagi, aplikasi yang dibuat bisa saja memiliki sedikit fitur daripada kompetitornya. Fitur yang lengkap, akan menjadikan aplikasi tersebut ibarat "emas".


Yang terakhir adalah dukungan. Ajaklah pengembang lain untuk bekerja sama membuat aplikasi.

close source : kode pada sebuah program yang tertutup dan dirahasiakan programmer. sehingga hanya bisa dikembangkan programmer itu sendiri open source code
