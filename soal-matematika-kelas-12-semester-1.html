Dengan metode grafik, tentukan penyelesaian dari sistem persamaan berikut untuk x, y € R! Soal: (no 9) y = x + 3 dan 2x + y + 8 = 0 (no 12) 2x - 4y = 6 dan 4x - 2y = -12 Pembahasan

Penyelesaian sistem persamaan dua variabel membutuhkan setidaknya 2 variabel persamaan dalam bilangan bulat.

Metode substitusi: Caranya dengan mengganti persamaan yang satu dan lainnya untuk mendapatkan variabel bernilai bilangan bulat.

Metode eliminasi: Caranya dengan menghilangkan salah satu variabel dengan pengurangan terhadap persamaan tersebut.

Metode grafik: Caranya dengan menentukan titik potong garis terhadap sumbu x dan sumbu y, kemudian digambarkan dalam bentuk grafik terhadap titik potong, sehingga himpunan penyelesaian dapat diketahui jika perpotongan garis terbentuk.

Metode grafik

A.

y = x + 3

2x + y + 8 = 0

B.

2x - 4y = 6

4x - 2y = -12

Menentukan titik potong sumbu x (y = 0) dan titik potong sumbu y (x = 0) untuk masing-masing persamaan.

y = x + 3 ... (1)

Untuk y = 0          Untuk x = 0

0 = x + 3               y = 0 + 3

x = -3                    y = 3

Titik koordinat adalah {x, y} yaitu {-3, 0} dan {0, 3}

2x + y + 8 = 0

Untuk y = 0          Untuk x = 0

2x + 0 + 8 = 0      2(0) + y + 8 = 0

2x = -8                  y = -8

x = -4

Titik koordinat adalah {x, y} yaitu {-4, 0} dan {0, -8}

2x - 4y = 6  ... (3)

Untuk y = 0          Untuk x = 0

2x - 4(0) = 6         2(0) - 4y = 6

2x = 6                   -4y = 6

x = 3                     y = -1,5

Titik koordinat adalah {x, y} yaitu {3, 0} dan {0, -1,5}

4x - 2y = -12  ... (4)

Untuk y = 0          Untuk x = 0

4x - 2(0) = -12       4(0) - 2x = -12

4x = -12                 -2x = -12

x = -3                     x = 6

Titik koordinat adalah {x, y} yaitu {-3, 0} dan {0, 6}

Gambar grafik menunjukkan bahwa perpotongan kedua garis adalah {x, y}, sehingga himpunan penyelesaian adalah sebagai berikut.

a. x = -3,667 dan y = -0,667, ditulis {-3,667, -0,667}

b. x = -5 dan y = -4, ditulis {-5, -4}
