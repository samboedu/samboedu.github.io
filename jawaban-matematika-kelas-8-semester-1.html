Pembahasan

15) Jika f(2x + 1) = (x – 12)(x + 13) maka nilai f(31) adalah …

Jawab

f(2x + 1) = f(31)

2x + 1 = 31

2x = 31 – 1

2x = 30

x = 15

Jadi

f(2x + 1) = (x – 12)(x + 13)

f(2(15) + 1) = (15 – 12)(15 + 13)

f(30 + 1) = (3)(28)

f(31) = 84

Jawaban D

16) Misalkan f(x) adalah fungsi yang memenuhi  

(a) untuk setiap bilangan real x dan y, maka f(x + y) = x + f(y)

(b) f(0) = 2

Nilai dari f(2016) adalah …

Jawab

Jika y = 0, maka:

f(x + y) = x + f(y)

f(x + 0) = x + f(0) ⇒ f(0) = 2

f(x) = x + 2

Jadi  

f(2016) = 2016 + 2 = 2018

17) Diketahui fungsi bilangan real f(x) = [tex]\frac{x}{1 - x}[/tex], untuk x ≠ 1. Nilai dari f(2016) + f(2015) + … + f(3) + f(2) + f([tex]\frac{1}{2}[/tex]) + f([tex]\frac{1}{3}[/tex]) + … + f([tex]\frac{1}{2015}[/tex]) + f([tex]\frac{1}{2016}[/tex]) adalah …

Jawab

Untuk menjawab soal tersebut, kita pasangkan fungsi-fungsi dengan angka yang sama yaitu

f(2) + [tex]f(\frac{1}{2})[/tex]

= [tex]\frac{2}{1 - 2} + \frac{\frac{1}{2}}{1 - \frac{1}{2}} [/tex]

= [tex]\frac{2}{-1} + \frac{\frac{1}{2}}{\frac{1}{2}} [/tex]

= –2 + 1

= –1  

f(3) + [tex]f(\frac{1}{3})[/tex]

= [tex]\frac{3}{1 - 3} + \frac{\frac{1}{3}}{1 - \frac{1}{3}} [/tex]

= [tex]\frac{3}{-2} + \frac{\frac{1}{3}}{\frac{2}{3}} [/tex]

= [tex]-\frac{3}{2} + \frac{1}{2} [/tex]

= [tex]-\frac{2}{2}[/tex]

= –1

:

f(2015) + [tex]f(\frac{1}{2015})[/tex]

= [tex]\frac{2015}{1 - 2015} + \frac{\frac{1}{2015}}{1 - \frac{1}{2015}} [/tex]

= [tex]\frac{2015}{-2014} + \frac{\frac{1}{2015}}{\frac{2014}{2015}} [/tex]

= [tex]-\frac{2015}{2014} + \frac{1}{2014} [/tex]

= [tex]-\frac{2014}{2014}[/tex]

= –1

f(2016) + [tex]f(\frac{1}{2016})[/tex]

= [tex]\frac{2016}{1 - 2016} + \frac{\frac{1}{2016}}{1 - \frac{1}{2016}} [/tex]

= [tex]\frac{2016}{-2015} + \frac{\frac{1}{2016}}{\frac{2015}{2016}} [/tex]

= [tex]-\frac{2016}{2015} + \frac{1}{2015} [/tex]

= [tex]-\frac{2015}{2015}[/tex]

= –1

Jadi penjumlahan yang menghasilkan –1 adalah 2016 – 1 = 2015 pasang

(dikurang satu karena pasangan f(1) dan f(1/1) tidak kita hitung)

Sehingga hasil dari penjumlahan soal diatas adalah

= (–1) + (–1) + (–1) + … + (–1) ⇒ sebanyak 2015

= (–1) . 2015

= –2015

18) Untuk setiap bilangan bulat x didefinisikan dengan fungsi f dengan f(x) adalah banyaknya angka (digit) dari x, contoh: f(206) = 3 dan f(2016) = 4. Nilai f(2²⁰¹⁶) + f(5²⁰¹⁶) adalah …

Jawab

Kita lihat pola f(2ⁿ) + f(5ⁿ)  

Jika n = 1

f(2¹) + f(5¹) = f(2) + f(5) = 1 + 1 = 2

Jika n = 2

f(2²) + f(5²) = f(4) + f(25) = 1 + 2 = 3

Jika n = 3

f(2³) + f(5³) = f(8) + f(125) = 1 + 3 = 4

Jika n = 4

f(2⁴) + f(5⁴) = f(16) + f(625) = 2 + 3 = 5

Jadi berdasarkan pola diatas f(2ⁿ) + f(5ⁿ) hasilnya adalah (n + 1)

berarti

f(2²⁰¹⁶) + f(5²⁰¹⁶) = 2016 + 1 = 2017

9. jawaban buku matematika paket kelas 8 semester 1 hal 30​


1. Banyak bola pada pola ke-n adalah 4n-3

2. Banyak bola pada pola ke-n adalah 2n² - 2n + 1

3. Jumlah bilangan pada garis ke-n pola bilangan Pascal tersebut adalah 2ⁿ⁻¹

Pembahasan

#1

Diketahui :

Pola bilangan 1,5,9,13

Ditanya :

suku ke-n

Jawab :

1, 5, 9, 13

 \/ \/  \/

 4  4  4

Perhatikan bahwa pola bilangan tersebut memiliki selisih antarsuku berurutan (beda) yang tetap  maka pola bilangan tersebut merupakan barisan aritmatika dan berlaku rumus :

Un = a + (n - 1) b

Un = 1 + (n - 1 ) 4

Un = 1 + 4n - 4

Un = 4n - 3

Jadi banyak bola pada pola ke- n adalah 4n - 3.

#2

Diketahui :

Pola bilangan 1,5,13,25

Ditanya :

suku ke-n

Jawab :

1, 5, 13, 25  --> a = 1

\/   \/   \/

 4  8  12  --> b = 4

  \/  \/

   4  4  --> c = 4

Perhatikan bahwa pola bilangan tersebut barisan aritmatika tingkat 2 sehingga  berlaku rumus :

Un = a + (n - 1) b + ¹/₂ x (n - 1) x (n - 2) x c

Un = 1 + (n - 1 ) 4 + ¹/₂ x (n - 1) x (n - 2) x 4

Un = 1 + (4n - 4) + 2 (n² - 3 n + 2)

Un= 1 + 4n - 4 + 2n² - 6n + 4

Un = 2n² - 2n + 1

Jadi banyak bola pada pola ke-n adalah 2n² - 2n + 1.

Un = 4n - 3

Jadi banyak bola pada pola ke- n adalah 4n - 3.

#3

Diketahui :

Pola bilangan Pascal

Ditanya :

suku ke-n

Jawab :

Mari kita tinjau pola bilangan tersebut !

          1

        1  1  

       1 2 1

     1 3  3  1

  1  4  6   4  1

1  5 10  10 5  1

U₁ = 1 = 1 = 2⁰ = 2¹⁻¹

U₂ = 1 + 1 = 2 = 2¹ = 2²⁻¹

U₃ = 1 + 2 + 1 = 4 = 2² =2³⁻¹

U₄ = 1 + 3 + 3 + 1 = 8 = 2³ = 2⁴⁻¹

U₅ = 1 + 4 + 6 + 4 + 1 = 16 = 2⁴ =2⁵⁻¹

U₆ = 1 + 5 +10 +10 + 5 + 1 = 32 = 2⁵ =2⁶⁻¹

Berarti Un = 2ⁿ⁻¹

Jadi jumlah bilangan pada garis ke-n pola bilangan Pascal tersebut adalah 2ⁿ⁻¹

Kunci Jawaban Soal Matematika Kelas 8 SMP MTs Halaman 11-12 Ayo Kita Berlatih 6,1, Gunakan Rumus Teoremma Phythagoras

Agar lebih jelasnya, berikut soal Matematika kelas 8 SMP MTs halaman 11-12 ayo berlatih 6.1 yang dilansir dari modul pembelajaran matematika kelas VIII, BSE Kemendikbud edisi

2017:

Kalian tau gak sih apa itu Teorema Phytagoras?

Menurut dalil tentang Pythagoras sebagai kuadrat panjang sisi miring suatu segitiga siku-siku (salah satu sudutnya 90°) yakni sama dengan jumlah kuadrat panjang sisi-sisi lainnya. Lebih sederhananya, teorema Pythagoras dapat dirumuskan seperti dibawah ini:

a2 + b2 = c2

Di mana “c” ialah sudut paling panjang dari segitiga siku-siku.

Sehingga, dengan rumus teorema Pythagoras, kita dapat menentukan mana yang merupakan kumpulan bilangan segitiga siku-siku.

Contohnya:

a = 3, b = 4 dan c = x, dengan cara menguadratkan sisi miring dan jumlahkan kuadrat sisi lainnya, maka kita akan mengetahui:

a2 + b2 = c2

32 + 42 = c2

9 + 16 = c2

25 = c2

C = √25

C = 5

Selain itu, dengan menggunakan rumus teorema Pythagoras, kita akan lebih mudah dalam menentukan; apakah ketiga barisan bilangan dapat membentuk segitiga siku-siku atau tidak.

22. kunci jawaban matematika kelas 8 semester 1 hal 239 uji kompetensi 5


Uji kompetensi 5 halaman 239 kelas 8 adalah kumpulan latihan soal tentang sistem persamaan linear dua variabel. Himpunan penyelesaian sistem persamaan linear dua variabel dapat diselesaikan dengan menggunakan metode eliminasi, atau metode substitusi. Disini kita akan membahas soal tersebut dari nomor 1 sampai 5 (halaman 239)


Pembahasan

1) Himpunan penyelesaian dari 2p + q = 4 jika p dan q bilangan cacah adalah ...

Jawab

2p + q = 4

q = 4 - 2p

Jika p = 0 maka q = 4 - 2(0) = 4 - 0 = 4 ⇒ (0, 4)Jika p = 1 maka q = 4 - 2(1) = 4 - 2 = 2 ⇒ (1, 2)Jika p = 2 maka q = 4 - 2(2) = 4 - 4 = 0 ⇒ (2, 0)Jika p = 3 maka q = 4 - 2(3) = 4 - 6 = -2 ⇒ bukan bilangan cacah

Jadi himpunan penyelesaiannya adalah

{(0, 4), (1, 2), (2, 0)}

Jawaban A



2) 3x + 2y - 4 = 0 ..... persamaan (1)

x - 3y - 5 = 0 ............. persamaan (2)

Dengan menggunakan metode substitusi, dari persamaan 2 diperoleh

x - 3y - 5 = 0

x = 3y + 5

Substitusikan x = 3y + 5 ke persamaan (1)

3x + 2y - 4 = 0

3(3y + 5) + 2y - 4 = 0

9y + 15 + 2y - 4 = 0

11y + 11 = 0

11y = -11

y = -1

Substitusikan y = -1 ke x = 3y + 5

x = 3(-1) + 5

x = -3 + 5

x = 2

Jadi selesaian dari sistem persamaan tersebut adalah (2, -1)

Jawaban B



3) x = a dan y = b adalah selesaian dari persamaan

2x + 3y = 12

3x + 2y = 8

------------------- +

5x + 5y = 20

x + y = 4

a + b = 4

Jawaban D



4) y = 4x - 11 kita substitusikan ke

3y = -2x - 5

3(4x - 11) = -2x - 5

12x - 33 = -2x - 5

12x + 2x = -5 + 33

14x = 28

x = 2

Substitusikan ke

y = 4x - 11

y = 4(2) - 11

y = 8 - 11

y = -3

Jadi titik potong kedua garis tersebut adalah (2, -3)

Jawaban C



5) Selesaian dari persamaan berikut kita gunakan metode gabungan  eliminasi dan substitusi

3x + y = -1 |×3|

x + 3y = 5 |×1|

----------------------

9x + 3y = -3

x + 3y = 5

------------------- -

8x = -8

x = -1

Substitusikan x = -1 ke persamaan (1)

3x + y = -1

3(-1) + y = -1

-3 + y = -1

y = -1 + 3

y = 2

Jadi selesaian dari persamaan tersebut adalah (-1, 2)

Nah sebelum mengenai fungsi biasanya akan dikenalkan terlebih dahulu mengenai relasi. Relasi merupakan aturan yang memasangkan anggota dari himpunan A yang disebut dengan domain (daerah asal) dengan himpunan B yang disebut dengan kodomain (daerah kawan) dimana tidak setiap anggota himpunan A harus punya kawan di himpunan B dan boleh terdapat anggota himpunan A yang mempunyai dua kawan atau lebih di himpunan B. Berbeda dengan fungsi dimana setiap anggota himpunan A harus punya kawan yang jumlahnya tepat satu di himpunan B. Oke langsung saja yukk kita lihat penjabaran dari jawaban soal kali ini.

Diketahui :

Fungsi f didefinisikan  dengan rumus f(x) = 5 – 3x dan diketahui daerah asalnya adalah {-2, -1, 0, 1, 2, 3}

Ditanyakan :

Buatlah tabel dan himpunan pasangan berurutan dari fungsi tersebut dan gambarlah fungsinya

Akan dicari range atau daerah hasilnya ya, atau dengan kata lain untuk x = {-2, -1, 0, 1, 2, 3} akan dicari f(x) = 5 – 3x, yaitu

f(-2) = 5 – 3(-2) = 5 + 6 = 11

f(-1) = 5 – 3(-1) = 5 + 3 = 8

f(0) = 5 – 3(0) = 5 – 0 = 5

f(1) = 5 – 3(1) = 5 – 3 = 2

f(2) = 5 – 3(2) = 5 – 6 = -1

f(3) = 5 – 3(3) = 5 – 9 = -4

penyajian dalam bentuk tabel nya ada di lampiran ya, dan himpunan pasangan berurutannya adalah {(-2, 11), (-1, 8), (0, 5), (1, 2), (2, -1), (3, -4)}
