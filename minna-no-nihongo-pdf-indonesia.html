15. Q. Bahasa Jepang dari “Belajar Bahasa Jepang” ...A. Nihongo O Manabu.B. Nihongo O Narau.C. Nihongo O Benkyou.D. Benkyou O Nihongo.​


Bahasa Jepang dari “Belajar Bahasa Jepang" adalah...

A. Nihongo o manabu.

                     

Penjelasan

Pilihan B saya rasa kurang tepat, karena kata 'narau' digunakan jika mempelajari suatu hal dari seseorang. Hal yang dipelajari biasanya juga berhubungan dengan seni atau keterampilan yang memerlukan teknik dan proses belajar secara berulang-ulang.

Contoh kalimat menggunakan 'narau' ↴

   父から空手を習います。

   = Chichi kara karate o naraimasu.

   = Belajar karate dari ayah.

Pilihan C kurang tepat, karena kata 'benkyou' merupakan kata benda. Kalimat tersebut bisa tepat bila penulisannya menjadi 'Nihongo o benkyou suru'. Kata 'suru' digunakan untuk mengubah kata benda 'benkyou' menjadi kata kerja.

Pilihan D jelas salah, karena susunan kalimatnya yang tidak tepat dan kata 'benkyou' bukanlah kata kerja. Harusnya kata 'benkyou' diganti dengan 'benkyou suru', dan pola kalimat yang tepat harusnya "[Objek] o [Predikat]".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Perbedaan 'Benkyou suru', 'Manabu', dan 'Narau' :

・Benkyou suru = "Belajar" yang lebih fokus ke hal-hal yang berhubungan dengan akademik.

・Manabu = "Belajar" yang cakupannya lebih luas dari 'benkyou suru'.

・Narau = “Belajar” yang digunakan ketika jelas ada seseorang yang mengajarkan kita akan sesuatu, dan biasa berhubungan dengan seni atau keterampilan yang memerlukan teknik dan proses belajar secara berulang-ulang.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Fungsi partikel を (O) :

Menunjukkan objekMenunjukkan tempat yang dilewatiMenunjukkan lokasi yang menjadi titik tolak dari suatu kegiatanMenunjukkan profesi

18. kalau " nihongo wo oshiemasu " berarti jadi " nihongo wo yonde kudasai?​

Jawaban:

Saya jawab pakai romaji saja ya, biar enak dibacanya..

-----------------------------------------------------------------------------------------------------------

[Perubahan K. Kerja bentuk ます menjadi K.Kerja bentuk て]

Dibagi menjadi beberapa kelompok yaitu:

1. Berakhiran huruf berikut sebelum Masu い(I), ち(Chi), り(Ri)

Maka ketiga huruf tersebut harus diganti dengan って(tte) dan ます(Masu) dihilangkan.

Contoh:

Kaimasu (Katte) Membeli

Tachimasu (Tatte) Berdiri

Machimasu (Matte) Menunggu

Hairimasu (Haitte) Masuk

-----------------------------------------------------------------------------------------------------------

2. Berakhiran huruf berikut sebelum Masu み(Mi), び(Bi), に(Ni)

Maka ketiga huruf tersebut harus diganti dengan んで(nde) dan ます(Masu) dihilangkan.

Contoh:

Yomimasu (Yonde) Membaca

Yobimasu (Yonde) Memanggil

Shinimasu (Shinde) Meninggal

*Bentuk て dari yomimasu dan yobimasu memang sama, tetapi tidak akan membuat salah paham karena kanjinya beda dan apa yang berkaitan dengan kata kerja tsb pasti beda.

-----------------------------------------------------------------------------------------------------------

3. Berakhiran dengan huruf し(Shi)

Maka cukup mengganti Masu dengan て(Te)

Contoh:

Hanashimasu (Hanashite) Berbicara

Kashimasu (Kashite) Meminjamkan

-----------------------------------------------------------------------------------------------------------

4. Kata kerja yang hurufnya bervokal え(E) sebelum Masu

Maka cukup mengganti Masu dengan て(Te)

Contoh:

Oshiemasu (Oshiete) Memberi tahu/Mengajari

Akemasu (Akete) Membuka

Tsukemasu (Tsukete) Menyalakan

Kaemasu (Kaete) Menukar

-----------------------------------------------------------------------------------------------------------

5. Berakhiran き(Ki) dan ぎ(Gi)

Maka き(Ki) dan ぎ(Gi) perlu diganti いて(Ite) dan いで(Ide) dan ます(Masu) dihilangkan.

Contoh:

Kakimasu (Kaite) Menulis

Kikimasu (Kiite) Mendengarkan

Oyogimasu (Oyoide) Berenang

-----------------------------------------------------------------------------------------------------------

6. Kata kerja bentuk Shimasu

Maka Masu pada Shimasu diganti て(Te)

Contoh:

Kekkon shimasu (Kekkon shite) Menikah

Kaimono shimasu (Kaimono shite) Berbelanja

Shokuji shimasu (Shokuji shite) Makan

-----------------------------------------------------------------------------------------------------------

Mungkin terdapat sebagian kecil K.Kerja yang memiliki "pengecualian" sehingga perubahan tidak menggunakan dasar-dasar diatas, namun dapat dibedakan dengan sering-sering membaca dan berlatih.

Contoh:

Mimasu (Mite) Melihat

Ikimasu (Itte) Pergi

Karimasu (Karite) Meminjam

Okimasu (Okite) Bangun

dan lain-lainnya..

22. arti "Sensei wa nihongo no sensei desu"​


Sensei wa nihongo no sensei desu artinya adalah guruku seorang guru bahasa Jepang.

Pembahasan

Huruf bahasa Jepangnya dari Sensei wa nihongo no sensei desu adalah 先生は日本語の先生です

Terjemahan kata-perkata :

先生 (sensei)

Artinya adalah guru.

は (wa)

Merupakan salah satu partikel yang dimana dalam konteks kalimat ini berfungsi sebagai penanda kalimat utama terhadap suatu objek dan subjek.

日本語 (nihon go)

Artinya dalam bahasa Indonesia adalah bahasa Jepang. Dimana kata 日本 (nihon) artinya adalah Jepang, 語 (go) yang berarti bahasa.

の (no)

Adalah salah satu jenis bentuk partikel yang umumnya digunakan untuk menyatakan suatu kepemilikan dan juga untuk menegaskan penegasan.

です (desu)

Merupakan salah satu partikel yang umumnya suka digunakan disetiap akhir kalimat dan juga sebagai menyatakan suatu penegasan.

30. Apa bahasa indonesia nihongo


Arti Nihongo 「日本語 」 adalah Bahasa Jepang.

Pembahasan:

Kosakata mengenai bahasa dalam Bahasa Jepang disebut dengan Gengo 「言語」. Penulisan kosakata ini disesuaikan dengan nama negara asal bahasa itu. Jika negara tersebut belum berdiri/ tidak dikenal sebelum munculnya tata penulisan Kanji, maka pada bagian negaranya ditulis dengan Katakana dan ditambah abjad Go 「語」. Contohnya sebagai berikut:

1) Bahasa Jepang:

日本 + 語 = 日本 語

(Nihon + go = Nihongo)

Semuanya ditulis dengan Kanji.

2) Bahasa Indonesia:

インドネシア + 語 = インドネシア語

(Indoneshia + go = Indoneshia-go)

Bagian Indonesia ditulis dengan Katakana.

3) Bahasa Korea:

韓国 + 語 = 韓国語

(Kankoku + go = Kankoku-go)

Semua ditulis dengan Kanji.

4) Bahasa China:

中国 + 語 = 中国語

(Chuugoku + go = Chuugoku-go)

Semua ditulis dengan Kanji.

5) Bahasa Inggris:

英 + 語 = 英語

Ei + go = Eigo

Semua ditulis dengan Kanji.

6) Bahasa Arab:

アラビア + 語 = アラビア語

Arabia + go = Arabia-go

Bagian Arabia ditulis dengan Katakana.
