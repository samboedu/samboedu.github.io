1. uang Acha banding uang Nina = 3 banding 1 sedangkan uang Acha banding uang Kiki = 3 banding 1 maka uang ada banding uang Nina banding uang Kiki adalah ​


Acha : Nina = 3 : 1

Acha : Kiki = 3 : 1

Biasanya di soal-soal kayak gini, nilai 2 perbandingannya harus disamain dulu Tapi karena di sini nilai perbandingannya Acha di perbandingan Acha & Nina, dan Acha & Kiki itu SAMA-SAMA 3, maka perbandingannya adalah; Acha : Nina : Kiki = 3 : 1 : 1

Contoh yg harus disamakan perbandingannya;

A : B = 2 : 3

B : C = 4 : 1

Karena nilai B-nya nggak sama di 2 perbandingan, maka nilai B harus kita samakan dulu.

A : B = 2 : 3 | x4 |

B : C = 4 : 1 | x3 |

         ↓

A : B = 8 : 12

B : C = 12 : 3

⇒ A : B : C = 8 : 12 : 3

23. perbandingan bertingkatuang Ali dibanding uang Budi adalah 3:5sementara uang Budi dibanding uang Cahya adalah 2:3 berapakah perbandingan uang Ali,Budi,dan Cahya jika uang mereka 155.000.00. Berapakah uang mereka masing-masing?​


Jawab:

Untuk menentukan perbandingan uang Ali, Budi, dan Cahya, kita dapat menggunakan perbandingan bertingkat.

Perbandingan uang Ali dibanding uang Budi adalah 3:5, dan perbandingan uang Budi dibanding uang Cahya adalah 2:3.

Untuk menentukan perbandingan uang Ali, Budi, dan Cahya secara keseluruhan, kita dapat mengalikan perbandingan uang Ali dibanding Budi dengan perbandingan uang Budi dibanding Cahya.

Perbandingan uang Ali, Budi, dan Cahya adalah (3/5) x (2/3) = 6/15

Untuk menentukan jumlah uang masing-masing, kita dapat menggunakan jumlah uang total yang diberikan dan perbandingan uang masing-masing.

Jumlah uang total = 155.000.000

Jumlah uang Ali = (6/15) x 155.000.000 = 52.000.000

Jumlah uang Budi = (5/15) x 155.000.000 = 83.333.333

Jumlah uang Cahya = (4/15) x 155.000.000 = 69.666.666

Jadi, uang Ali sebesar Rp. 52.000.000, uang Budi sebesar Rp. 83.333.333 dan uang Cahya sebesar Rp. 69.666.666

Semoga jawaban ini membantu.

Penyelesaian

        = 3 x 75.000

        = Rp225.000

           = 4 x 75.000

           = Rp300.000

Alifah = 5/12 x Rp900.000

          = 5 x Rp75.000

          = Rp375.000
