Lima contoh bilangan berpangkat negatif

Pembahasan

Operasi Hitung Bilangan berpangkat

Jika n bilangan bulat positif dan a bilangan real berlaku :

[tex] {a}^{n} = a \times a \times .... \times a[/tex]

Definisi bilangan berpangkat bulat positif diatas dikembangkan untuk bilangan berpangkat lainnya sebagai berikut :

a. [tex] {a}^{0} = 1[/tex], a ≠ 0

b. [tex] {a}^{ - n} = \frac{1}{ {a}^{n} } [/tex], a ≠ 0

c. [tex] {a}^{ \frac{1}{n} } = \sqrt[n]{a} [/tex]

Berikut contoh dan penyelesaian bilangan berpangkat negatif

a. [tex] {3}^{ - 1} = \frac{1}{ {3}^{1} } \\ {3}^{ - 1} = \frac{1}{ 3} [/tex]

b. [tex] {2}^{ - 3} = \frac{1}{ {2}^{3} } \\ {2}^{ - 3} = \frac{1}{8} [/tex]

c. [tex] {3}^{ - 2} = \frac{1}{ {3}^{2} } \\ {3}^{ - 2} = \frac{1}{9} [/tex]

d. [tex] {5}^{ - 2} = \frac{1}{ {5}^{2} } \\ {5}^{ - 2} = \frac{1}{25} [/tex]

e. [tex]( { {81}^{ \frac{3}{4} } })^ { - \frac{1}{3} } = ( { ({ {3}^{4} )}^{ \frac{3}{4} } })^{ - \frac{1}{3} } \\ = {3}^{4 \times \frac{3}{4} \times ( - \frac{1 }{3} ) } \\ = {3}^{ - 1} \\ = \frac{1}{ {3}^{1} } \\ = \frac{1}{3} [/tex]

3(pangkat -2) dapat diubah ke pangkat positif dengan cara

=1/3²

=1/9 diubah ke pangkat negatif lagi =9(pangkat negatif 1)
