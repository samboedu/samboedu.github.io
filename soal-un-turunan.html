27. Turunan pertama fungsi f(x) = cos² (3x + 6) adalah f '(x) =...a. -6 sin(6x + 12)b. -3 sin(6x + 12)c. -sin(6x +12)d. -3 cos(6x +12)e. -6 cos(6x + 12)tolong bantu saya yahini soal UN 2003


Turunan trigonometri adalah persamaan turunan yang melibatkan fungsi - fungsi trigonometri seperti sin, cos, tan, cot, sec dan csc.

Rumus dasar turunan trigonometri

1.  f(x) = cos x  →  f '(x) = −sin x

2.  f(x) = sec x  →  f '(x) = sec x . tan x

3.  f(x) = sin x  →  f '(x) = cos x

4.  f(x) = csc x  →  f '(x) = −csc x . cot x

5.  f(x) = tan x  →  f '(x) = sec² x

6.  f(x) = cot x  →  f '(x) = −csc² x

7.  f(x) = cos ax  →  f '(x) = −a. sin ax

8.  f(x) = sin ax  →  f '(x) = a. cos x

9. f(x) = cosⁿ (ax + c) → f'(x) = n. cosⁿ⁻¹ (ax + c). c . - sin (ax + c)

Pembahasan

Soal:

Turunan pertama fungsi f(x) = cos² (3x + 6) adalah ...

Jawab:

Gunakan rumus ↓

f(x) = cosⁿ (ax + c) → f'(x) = n. cosⁿ⁻¹ (ax + c). c . - sin (ax + c)

f(x) = cos² (3x + 6)

f'(x) = 2 cos (3x + 6) . 3. -sin (3x + 6)

     = -3 . 2 cos (3x + 6). sin (3x + 6)

ingat 2 cos a sin b = sin (a + b) - sin (a - b)

     = -3 . {2 cos (3x + 6). sin (3x + 6)}

     = -3. {sin (3x + 6 + 3x + 6) . -sin (3x + 6 - (3x + 6))}

     = -3 . {sin (6x + 12) . -sin 0}

     = -3. sin (6x + 12)

Jadi, Turunan pertama fungsi f(x) = cos² (3x + 6) adalah -3. sin (6x + 12)

Jawaban : B
