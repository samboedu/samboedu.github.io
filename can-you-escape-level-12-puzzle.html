aplikasi bangun datar persegi

banyak puzzle kuning = k

banyak puzzle  biru = ( k + 2 )

banyak puzzle Merah = ( k + 2 ) + 8

.                                    = ( k + 10 )

maka banyak puzzle seluruhnya adalah ;

= k + k + 2 + k + 10

= 3k + 12

luas seluruh puzzle = 60.000

3 k + 12 = 60.000

3 ( 400 k ) + 12 ( 400 ) = 60.000

1200k + 4800 = 60.000

1200k = 60.000 - 4.800

1200 k = 55.200

k = 55.200/1.200

k = 46

jadi banyak puzzle adalah ;

Kuning = 46 buah

biru = 48 buah

merah = 56 buah

luas puzzle ;

biru = 48 x 400

.       = 19.200 cm²

Kuning = 46 x 400

.            = 18.400 cm²
