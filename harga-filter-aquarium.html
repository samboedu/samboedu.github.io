2. apa fungsi filter dan aerator didalan aquarium


fungsi filter:menyaring air dengan proses filtrasi dengan beberapa media yang mempunyai tujuan media masing-masing, untuk menghasilkan air yg selalu bersih di akuarium.
fungsi aerator:menghasilkan (gelembung-gelembung) oksigen dlm air untuk 1)mensuplai oksigen ke air 2)menghasilkan cukup arus yang disukai ikan dan membuat air tersirkulasi 3)membawa hawa panas air kepermukaan sehingga air terhindar dari suhu panas yang berlebihan

4. Kerjakan soal berikut! 1. Harga pembelian sebuah Aquarium Rp150.000 karena merasa bosan Aquarium tersebut dijual kembali dengan harga Rp 180.000. Tentukan persentase untung atau ruginya ! ​


Jawaban:

Untuk menentukan persentase keuntungan atau kerugian, kita bisa menggunakan rumus berikut:

persentase keuntungan/kerugian = (harga jual - harga beli) / harga beli x 100%

Dalam hal ini, harga jual adalah Rp180.000 dan harga beli adalah Rp150.000. Jadi,

persentase keuntungan = (Rp180.000 - Rp150.000) / Rp150.000 x 100% = 20%.

Jadi, Aquarium tersebut dijual dengan keuntungan sebesar 20%.

29. Bodi membeli dua bungkus pupuk dasar, dua bungkus batu hias, dan satu saringan/filter dengan harga Rp 380.000, Arya membeli satu bungkus pupuk dasar dan satu saringan/filter dengan harga Rp 180.000, sedangkan Mita membeli satu bungkus pupuk dasar, tiga bungkus batu hias dan satu saringan/filter dengan harga Rp 330.000. Berapa harga satu bungkus pupuk dasar, satu bungkus batu hias, dan satu saringan filter masing-masing!


Jawaban:

pupuk dasar = 100.000

batu hias = 50.000

saringan = 80.000

Penjelasan dengan langkah-langkah:

pupuk dasar = X

batu hias = Y

saringan = Z

persamaan 1 : 2x + 2y + z = 380.000

persamaan 2 : x + z = 180.000

persamaan 3 : x + 3y + z = 330.000

metode eliminasi per 1&3

2x + 2y + z = 380.000 |×3| 6x + 6y + 3z = 1.140.000

x + 3y + z = 330.000 |×2| 2x + 6y + 2z = 660.000

____________________-

4x + z = 480.000 (per 4)

metode eliminasi per 2&4

4x + z = 480.000

x + z = 180.000

_______________-

3x = 300.000

x = 300.000 : 3

x= 100.000 (pupuk dasar)

metode substitusi per 2

x + z = 180.000

(100.000) + z = 180.000

z = 180.000 - 100.000

z = 80.000 ( saringan)

metode substitusi per 1

2x + 2y + z = 380.000

2(100.000) + 2y + (80.000) = 380.000

200.000 + 2y + 80.000 = 380.000

2y = 380.000 - 200.000 - 80.000

2y = 100.000

y = 100.000 : 2

y = 50.000 (batu hias)
