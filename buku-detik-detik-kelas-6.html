diagonal layang 1= 15+6 = 21
diagonal layang 2= 8+8 = 16 (note: angka 8 didapat dari panjang selisih sisi atas dan alas trapesium, dikali 2 untuk membentuk diagonal 2 layang layang)

luas gabungan = luas trapesium+luas layang

= ( sisi atas + sisi alas ) x t/ 2 + (diagonal 1 x diagonal 2 )/ 2
= (20+28)x15/2+ (21x16)/2
= 48x 15/2 + 168
= 360 + 168
= 528 cm^2 ( jawaban C )
