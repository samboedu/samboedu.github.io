7. 1. Tentukan persamaan sumbu simetri dan nilai optimum dari grafik fungsi y=2xpangkat2 + x -4 2.jumlah 2 buah bilangan 8 dan hasil kalinya 16 Tentukan bilangan tersebut 3. jika nilai diskriminan dari 2 x pangkat 2 negatif 3 x + c = 0 adalah 17 tentukan nilai niali C​


Persamaan sumbu simetri dan nilai optimum dari grafik fungsi y = 2x² + x - 4 adalah [tex]-\frac{1}{4}[/tex] dan [tex]-\frac{33}{8}[/tex].Jumlah 2 buah bilangan 8 dan hasil kalinya 16. Kedua bilangan tersebut adalah 4.Jika nilai diskriminan dari 2x² - 3x + c = 0 adalah 17, maka nilai c adalah -1.

Pembahasan

FUNGSI KUADRAT

Bentuk Umum: f(x) = ax² + bx + c, a ≠ 0

a : koefisien dari x²b : koefisien dari xx : variabelc : konstanta

Sumbu Simetri: [tex]x = -\frac{b}{2a}[/tex]

Nilai Optimum: [tex]y = -\frac{D}{4a}[/tex]

Titik Puncak: ([tex]x = -\frac{b}{2a}[/tex], [tex]y = -\frac{D}{4a}[/tex])

Determinan: D = b² - 4ac

Diketahui:

Nomor 1

y = 2x² + x - 4

Nomor 2

misal a = bilangan ke-1, dan b = bilangan ke-2a + b = 8 ...(1)a × b = 16...(2)

Nomor 3

D = 172x² - 3x + c = 0

Ditanyakan:

Sumbu simetri dan nilai optimumBilangan tersebutNilai c

Jawab:

NOMOR 1

y = 2x² + x - 4

a = 2b = 1c = -4

Sumbu simetri:

[tex]x= - \frac{b}{2a} [/tex]

[tex]x = - \frac{1}{2 \times 2} [/tex]

[tex]x = - \frac{1}{4} [/tex]

Nilai optimum:

[tex]y = - \frac{D}{4a} [/tex]

[tex]y = - \frac{ ({b}^{2} - 4ac)}{4a} [/tex]

[tex]y = - \frac{ ({1}^{2} - 4 \times 2 \times ( - 4)) }{4 \times 2} [/tex]

[tex]y = - \frac{(1 + 32)}{8} [/tex]

[tex]y = - \frac{33}{8} [/tex]

Atau bisa juga dengan memasukkan nilai x dari sumbu simetri ke dalam persamaan fungsi kuadrat:

[tex]y = 2 {x}^{2} + x - 4[/tex]

[tex]y = 2( { - \frac{1}{4} })^{2} + ( - \frac{1}{4} ) - 4[/tex]

[tex]y = 2( \frac{1}{16} ) - \frac{1}{4} - 4[/tex]

[tex]y = \frac{1}{8} - \frac{1}{4} - 4[/tex]

[tex]y = \frac{1}{8} - \frac{2}{8} - \frac{32}{8} [/tex]

[tex]y = - \frac{33}{8} [/tex]

Jadi, persamaan sumbu simetri dan nilai optimumnya adalah [tex]-\frac{1}{4}[/tex] dan [tex]-\frac{33}{8}[/tex].

NOMOR 2

a + b = 8

a       = 8 - b

Substitusi a = 8 - b ke persamaan (2)

a × b = 16

(8-b) × b = 16

8b - b² = 16

- b² + 8b - 16 = 0 (× (-1))

b² - 8b + 16 = 0

(b - 4)(b - 4) = 0

b = 4 b = 4

Substitusi b = 4 ke persamaan (1)

a + b = 8

a + 4 = 8

a       = 8 - 4

a       = 4

Jadi, kedua bilangan tersebut adalah 4.

NOMOR 3

2x² - 3x + c = 0

a = 2b = -3

D = 17

b² - 4ac = 17

(-3)² - 4(2)(c) = 17

9 - 8c = 17

   -8c = 17 - 9

   -8c = 8

      c = -1

Jadi, nilai c adalah -1.

jika ganjil + ganjil= genap

genap+ ganjil= ganjil
