Dua buah garis dikatakan sejajar jika posisi kedua garis tersebut tidak berpotongan dan jika kedua garis dipanjangkan maka jarak kedua garis tidak akan berubal alias selalu tetap.

=> Kordinat Kartesius

Kordinat kartesius adalah sistem koordinat yang menetapkan setiap titik secara unik dalam bidang dengan serangkaian koordinat numerik yang bisa ditentukan jaraknya dari kedua sumbu yang saling berpotongan, yaitu sumbu x atau absis ( garis horizontal ) dan sumbu y atau ordinat ( garis vertikal ).

[tex]\frac{y - 5}{0 - 5} = \frac{x - 0}{-3 - 0}[/tex]

[tex]\frac{y - 5}{-5} = \frac{x}{-3}[/tex]

1. Jumlah kelas 1 = 400

2. Jumlah kelas 2 = 300

3. Selisih kelas 3 dan 2 = 350 - 300 = 50

Penjelasan dengan langkah - langkah penyelesaian :

Quiz -

kelas 1 = a, A, A, A, a

kelas 2 = a, A, a, A

kelas 3 = A, A, a, A

a = mewakili 50

A = mewakili 100

Soal

1. jumlah kelas 1

2. jumlah kelas 2

3. selisih kelas 3 dan 2

Pembahasan :

Diketahui :

a = Mewakili 50A = Mewakili 100

Ditanya :

Jumlah kelas 1 = ... ?Jumlah kelas 2 = ... ?Selisih kelas 3 dan 2 = ... ?

Jawab :

[tex]Kelas \: 1 = a, A, A, A, a [/tex]

[tex]Kelas \: 2 = a, A, a, A [/tex]

[tex]Kelas \: 3 = A, A, a, A[/tex]

Keterangan :

a = mewakili 50

A = mewakili 100

1. Jumlah kelas 1

2. Jumlah kelas 2

3. Selisih kelas 3 dan 2

A. Jumlah kelas 1

[tex]Kelas \: 1 = a, A, A, A, a [/tex]

[tex]Kelas \: 1 = 50, 100, 100, 100, 50[/tex]

[tex]Jumlah \: kelas \: 1 : [/tex]

[tex]50 + 100 + 100 + 100 + 50[/tex]

[tex] = 400[/tex]

Jadi, jumlah kelas 1 adalah 400

B. Jumlah kelas 2

[tex]Kelas \: 2 = a, A, a, A [/tex]

[tex]Kelas \: 2 = 50, 100, 50, 100[/tex]

[tex]Jumlah \: kelas \: 2 : [/tex]

[tex]50 + 100 + 50 + 100[/tex]

[tex] = 300[/tex]

Jadi, jumlah kelas 2 adalah 300

C. Selisih kelas 3 dan 2

[tex]Kelas \: 3 = A, A, a, A[/tex]

[tex]Kelas \: 2 = a, A, a, A [/tex]

[tex]Kelas \: 3 = 100, 100, 50, 100[/tex]

[tex]Kelas \: 2 = 50, 100, 50, 100[/tex]

[tex]Kelas \: 3 = 100 + 100 + 50 + 100 [/tex]

[tex] = 350[/tex]

[tex]Kelas \: 2 = 50 + 100 + 50 + 100[/tex]

[tex] = 300[/tex]

[tex]Selisih = Kelas \: 3 - Kelas \: 2[/tex]

[tex]Selisih = 350 - 300[/tex]

[tex]Selisih = 50[/tex]

Jadi, selisih kelas 3 dan 2 adalah 50

Jawaban:

1. Jumlah kelas 1 adalah 400.

2. Jumlah kelas 2 adalah 300.

3. Selisih kelas 3 dan kelas 2 adalah 50.

Penyelesaian :

Keterangan :

a = mewakili 50

A = mewakili 100

1. Jumlah kelas 1

= 50 + 100 + 100 + 100 + 50

= 150 + 100 + 150

= 250 + 150

= 400

2. Jumlah kelas 2

= 50 + 100 + 50 + 100

= 150 + 150

= 300

3. Sebelum mencari selisih kelas 3 dan kelas 2, jumlahkan kedua kelas terlebih dahulu sesuai keterangan. ^^*

Kelas 3 : A, A, a, A

= 100 + 100 + 50 + 100

= 200 + 150

= 350

Jumlah kelas 3 adalah 350.

Kelas 2 : a, A, a, A

= 50 + 100 + 50 + 100

= 150 + 150

= 300

Jumlah kelas 2 adalah 300.

= Kelas 3 - Kelas 2

= 350 - 300

= 50.
