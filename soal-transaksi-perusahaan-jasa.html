Pembahasan

Persamaan dasar akuntansi secara umum adalah keseimbangan antara harta dengan modal dan kewajiban. Rumus dari persamaan dasar akuntansi yaitu Harta = hutang + modal. Jumlah harta suatu perusahaan harus sama dengan jumlah modal ditambah kewajibannya.

Pembelian mobil masuk dalam peralatan, bukan perlengkapan. Kenapa? Karena peralatan mempunyai sifat tidak cepat habis, atau penggunaannya bisa untuk jangka waktu yang lama. Sedangkan perlengkapan mempunyai sifat habis pakai. Untuk lebih jelasnya simak dalam lampiran ya.

- Jasa transaksi 35%

- Jasa Modal 25%

- Dana Pengurus 10%

- Dana Karyawan 15%

- Dana Pendidikan 20%

- Dana Sosial 5%

Berdasarkan data diatas, hitunglah :

a. SHU masing-masing pos anggaran

b. SHU Bu Farida

Untuk lebih jelasnya maka dapat dilihat pada penjelasan dibawah ini.

Pembahasan

Untuk menghitung SHU dari masing-masing pos anggaran kita harus menghitung satu per satu pos anggaran yang ada yaitu :

Jasa transaksi 35%  = Persentase x SHU = 35% x Rp.30.000.000 = Rp.10.500.000Jasa Modal 25%  = Persentase x SHU = 25% x Rp.30.000.000 = Rp. 7.500.000Dana Pengurus 10%   = Persentase x SHU = 10% x Rp.30.000.000 = Rp.3.000.000Dana Karyawan 15%   = Persentase x SHU = 15% x Rp.30.000.000 = Rp.4.500.000Dana Pendidikan 20%   = Persentase x SHU = 20% x Rp.30.000.000 = Rp.6.000.000Dana Sosial 5%   = Persentase x SHU = 5% x Rp.30.000.000 = Rp.1.500.000

Untuk perhitungan SHU yang ibu Farida terima, kita harus menghitung terlebih dahulu masing-masing SHU dari jasa modal dan jasa usaha yaitu :

1. SHU dari jasa modal

JM = (Simpanan anggota / SImpanan koperasi) x persentase x SHU

      = (Rp.12.000.000 / Rp.60.000.000) x 25% x Rp.30.000.000

      = Rp. 1.500.000

2. SHU dari jasa usaha

JU = (Transaksi bu farida / total transaksi koperasi) x Persentase x SHU

      = (Rp.5.000.000 / Rp.25.000.000) x 35% x Rp.30.000.000

      = Rp.2.100.000

SHU = JM + JU

        = Rp.1.500.000 + Rp.2.100.000

        = Rp.3.600.000
