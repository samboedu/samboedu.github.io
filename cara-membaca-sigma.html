7. NOTASI SIGMANo10 pake cara ya!


Sn = n/2 (a + Un)

Sigma (4k + 5) = 774
(4.1 + 5) + (4.2 + 5) + (4.3 + 5) + ... + (4n + 5) = 774
9 + 13 + 17 + ... + (4n + 5) = 774
Sn = 774
n/2 (a + Un) = 774
n/2 (9 + (4n + 5)) = 774
n/2 (4n + 14) = 774
2n^2 + 7n - 774 = 0
(2n + 43)(n - 18) = 0
n = -43/2 atau n = 18
Ambil yang positif maka n = 18

27. Notasi Sigma a-i dengan cara​


Jawaban:

[tex]a. \: \sum_{i=1}^{21} 6(i-1)[/tex]

[tex]b. \: \sum_{i=1}^{99} [3-(i+4)] = \sum_{i=1}^{99} (-i - 1)[/tex]

[tex]c. \: \sum_{i=1}^{10} 7^{(i-6)}[/tex]

[tex]d. \: \sum_{i=1}^{11} (i-11)^4[/tex]

[tex]e. \: \sum_{i=1}^{9} \frac{5(i+1)}{7} [/tex]

[tex]f. \: \sum_{i=1}^{25} \frac{i+6}{i}[/tex]

[tex]g. \: \sum_{i=1}^{13} i^i[/tex]

[tex]h. \: \sum_{i=1}^{17} (5^{i-2}+3)[/tex]

[tex]i. \: \sum_{i=3}^{66} \frac{i+1}{i-2}[/tex]

Penjelasan dengan langkah-langkah:

Bagian A

[tex]\sf{ 6(0)+6(1)+6(2)+6(3)+ ... + 6(20) = 6(1-1) + 6(2-1) + 6(3-1) + ... + 6(21-1) }[/tex]

[tex]\sum_{i=1}^{21} 6(i-1)[/tex]

Bagian B

[tex]\sf{ (3-5) + (3-6) + (3-7) + ...+(3-99)+(3-100)= (3-(1+4)) +(3-(2+4))+(3-(3+4)) +...+(3-(99+1)) }[/tex]

[tex]\sum_{i=1}^{99} [3-(i+4)] = \sum_{i=1}^{99} (-i - 1)[/tex]

Bagian C

[tex]\sf{ 7^{-5}+7^{-4}+7^{-3}+...+7^3+7^4=7^{1-6}+7^{2-6}+...+7^{10-6} }[/tex]

[tex]\sum_{i=1}^{10} 7^{(i-6)}[/tex]

Bagian D

[tex]\sf{ (-10^4)+(-9^4)+...+(-1^4)+(0^4)=(1-11)^4+(2-11)^4+...+(11-11)^4 }[/tex]

[tex]\sum_{i=1}^{11} (i-11)^4[/tex]

Bagian E

[tex]\sf{\frac{5 \times 2}{7} + \frac{5 \times 3}{7} +... + \frac{5 \times 10}{7} = \frac{5(1+1)}{7} + \frac{5(2+1}{7}+ ...+\frac{5(9+1)}{7} }[/tex]

[tex]\sum_{i=1}^{9} \frac{5(i+1)}{7} [/tex]

Bagian F

[tex]\sf{\frac{1+6}{1} + \frac{2+6}{2} +...+ \frac{25+6}{25} = \frac{1+6}{1} + \frac{2+6}{2} + ...+ \frac{25+6}{25} }[/tex]

[tex]\sum_{i=1}^{25} \frac{i+6}{i}[/tex]

Bagian G

[tex]\sf{ 2^2+3^3+4^4+...+13^{13} = 2^2+3^3+4^4+...+13^{13} }[/tex]

[tex]\sum_{i=1}^{13} i^i[/tex]

Bagian H

[tex]\sf{ (5^{-1}+3)+(5^{0}+3)+...+(5^{15} + 13) = (5^{1-2}+3)+(5^{2-2}+3)+...+(5^{17-2} + 13) }[/tex]

[tex]\sum_{i=1}^{17} (5^{i-2}+3)[/tex]

Bagian I

[tex]\sf{\frac{3 + 1}{3-2} + \frac{4+1}{4-2} +...+ \frac{66+1}{66-2} =\frac{3 + 1}{3-2} + \frac{4+1}{4-2} +...+ \frac{66+1}{66-2} }[/tex]

[tex]\sum_{i=3}^{66} \frac{i+1}{i-2}[/tex]
