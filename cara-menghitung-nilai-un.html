12. Soal matematika, 1. Diketahui, suku terakhir dari barisan aritmatika = 398, a= 2, b = 4, hitung nilai n ? 2. Suku ketiga dari suatu barisan aritmatika adalah 10 sedangkan suku ke 8 adalah 25, berapa nilai suku ke 13 3. Diketahui barisan aritmatika, 3, 6, 9,.... Un, hitung nilai suku ke 100? Kerjakan soal-soal MTK diatas dg menggunakan cara..​


NOMOR 1.

Un = a + (n - 1)•b398 = 2 + (n - 1)•(4)398 = 2 + 4n - 4398 = 4n + 2 - 4398 = 4n - 24n = 398 - (-2)4n = 398 + 24n = 400n = 400/4n = 100

NOMOR 2.

Menentukan nilai dari beda (b) terlebih dahulu.

U3 = a + 2b = 10U8 = a + 7b = 25 _>>>>>> -5b = -15>>>>>> b = (-15)/(-5)>>>>>> b = 3

Menentukan nilai dari suku pertama (a) dengan mensubstitusikan nilai dari beda (b) = 3, ke salah satu persamaan tersebut.

U3 = a + 2b = 10>>>> a + 2(3) = 10>>>> a + 6 = 10>>>> a = 10 - 6>>>> a = 4

Menentukan nilai dari rumus suku ke-n (Un) dengan mensubstitusikan nilai dari suku pertama (a) = 4 dan nilai dari beda (b) = 3, ke bentuk persamaan umum tersebut.

Un = a + (n - 1)•bUn = 4 + (n - 1)•(3)Un = 4 + 3n - 3Un = 3n + 4 - 3Un = 3n + 1

Menentukan nilai dari suku ke-13 (U13) dengan menggunakan nilai dari rumus suku ke-n (Un) yang telah ditemukan.

Un = 3n + 1U13 = 3(13) + 1U13 = 39 + 1U13 = 40

NOMOR 3.

Pola barisan aritmetika = 3, 6, 9, ....

Suku pertama (a) = U1 = 3Beda (b) = 6 - 3 = 3

Nilai dari rumus suku ke-n (Un) adalah

Un = a + (n - 1)•bUn = 3 + (n - 1)•(3)Un = 3 + 3n - 3Un = 3n + 3 - 3Un = 3n + 0Un = 3n

Jadi, nilai dari suku ke-100 (U100) adalah

Un = 3nU100 = 3(100)U100 = 300

23. Cara menghitung rata rata nilai UN


Cara menghitung rata rata nilai UN

Rata-rata Nilai UN = [tex]\frac{Jumlah~semua~nilai}{Banyak~mapel~yang~diujikan}[/tex]

Pembahasan

Mean adalah rata" dari beberapa data.

Mean dapat dihitung atau ditentukan dengan cara membagi jumlah nilai data  dengan banyak data

Rumus untuk mencari Mean

= ∑f.x/∑f

Contoh Soal

Nilai UN kita

Bahasa Indonesia = 8,40

Bahasa Inggris = 9,60

Matematika = 9,25

Ekonomi = 6,50

Sosiologi = 7,60

Geografi = 6,40

Rata-rata = [tex]\frac{8,40+9,60+9,25+6,50+7,60+6,40}{6}[/tex]

               = [tex]\frac{47,75}{6}[/tex]

               = 7,958

Jadi rata-rata nilai UN nya = 7,958.

30. Cara menghitung nilai un/ipk untuk melamar kerja


Jawab:

Cara menghitung nilai UN

Jumlahkan semua nilai tiap mata pelajaran.

Contoh Nilai UN:

B Indonesia 10.00

B Inggris 8.50

Matematika 7.50

IPA 9.00

Maka total nilai UN nya 10+8.5+7.5+9 = 35.00

Cara menghitung IPK

Jumlahkan seluruh nilai IP dari semester satu hingga akhir, kemudian dibagi jumlah semester.

Contoh : 3,4 + 3,6 + 3,2 + 2,6 + 3,2 + 3 + 2,8 + 3,6= 25,4 : 8 = 3,175 maka nilai IPK nya adalah 3,175
