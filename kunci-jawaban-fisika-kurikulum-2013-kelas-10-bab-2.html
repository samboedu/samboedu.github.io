1. Gelombang bunyi dari suatu sumber memiliki cepat rambat 340 m/s. Jika frekuensi gelombang bunyi adalah 650 Hz, tentukan panjang gelombangnya!

Diketahui

v = 340 m/s

f = 650 Hz

Ditanya

Panjang gelombang, λ

Penyelesaian

v = λf

340 = λ (650)

λ = 340 / 650

λ = 34/65 meter

b) periode sumber bunyi

340 = 8f

Kabel Listrik dipasang agak kendur agar kabel tidak putus saat suhu turun dan terjadi penyusutan.Koefisien muai panjang besi sebesar 12 x 10⁻⁵/°C atau  1,2 x 10⁻⁶/°CPanjang Baja pada suhu 70°C adalah 40,04 cm

Pembahasan

Pemuaian

Saat benda dikenai kalor, maka akan terjadi pemuaian pada benda tersebut yang akan mengakibatkan pertambahan ukuran pada benda. dan sebaliknya saat kalor dilepaskan dari benda, akan terjadi penyusutan, besarnya pemuaian atau penyusutan tergantung pada koefisien muai benda dan perubahan suhu.

Pada Pemuaian Panjang berlaku persamaan :ΔL = Lo x α x Δt  Lt = Lo + ΔL  

atau  

Lt = Lo + (Lo x α x Δt)  

Keterangan :  

Lo = Panjang awal = m  

Lt = Panjang akhir = m  

ΔL = Pertambahan Panjang = m  

α = Koefisien muai panjang

Pada Pemuaian Luas berlaku persamaan :ΔA = Ao x β x Δt  

dan  

At = Ao + ΔA  

Atau  

At = Ao + (Ao x β x Δt)  

Keterangan :  

ΔA = Pertambahan Luas  

Ao = Luas awal  

At = Luas Akhir  

β = Koefisien Muai Luas  

β = 2 α  

Pada Pemuaian Volume berlaku persamaan :ΔV = Vo x γ x Δt  

dan untuk Volume akhir adalah :  

Vt = Vo + ΔV  

atau  

Vt = V0 + (V0 x γ x Δt)  

Keterangan  

ΔV = Pertambahan Volume  

γ = Koefisien Muai Ruang atau Volume  

besarnya γ = 3α  

Δt= Perubahan Suhu  

Vo= Volume awal  

Vt = Volume akhir

Penyelesaian :

ΔL = Lo x α x Δt

Ditanyakan :

Lt = Panjang akhir Baja = ...

Penyelesaian :

Lt = Lo + (Lo x α x t)

Lt = 40 + (40 x 2 x 10⁻⁵ x 50)

Pembahasan :

1) |x - 2| = 6

• x - 2 = 6
=> x = 8

• -(x - 2) = 6
=> -x + 2 = 6
=> -x = 4
=> x = -4

HP = {-4, 8}



2) |3x - 5| = 7

• 3x - 5 = 7
=> 3x = 12
=> x = 4

• -(3x - 5) = 7
=> -3x + 5 = 7
=> -3x = 2
=> x = -2/3

HP = {-2/3, 4}



3) |x| + |x - 5| = 7
|x| = x jika x ≥ 0
|x| = -x jika x < 0
|x - 5| = (x - 5) jika x ≥ 5
|x - 5| = (-x + 5) jika x < 5

• jika x < 0
|x| + |x - 5| = 7
-x + (-x + 5) = 7
-2x = 2
x = -1 ..... (memenuhi syarat x < 0)

• jika 0 ≤ x < 5
|x| + |x - 5| = 7
x + (-x + 5) = 7
5 = 7
(Salah)

• jika x ≥ 5
|x| + |x - 5| = 7
x + (x - 5) = 7
2x = 12
x = 6 ...... (memenuhi syarat x ≥ 5)

Jadi HP = {-1, 6}


4) |2x - 2| + |3x - 8| = 5
|2x - 2| = (2x - 2) jika x ≥ 1
|2x - 2| = (-2x + 2) jika x < 1
|3x - 8| = (3x - 8) jika x ≥ 8/3
|3x - 8| = (-3x + 8) jika x < 8/3

• jika x < 1
|2x - 2| + |3x - 8| = 5
(-2x + 2) + (-3x + 8) = 5
-5x + 10 = 5
-5x = -5
x = 1 ...... (tidak memenuhi syarat x < 1)

• jika 1 ≤ x < 8/3
|2x - 2| + |3x - 8| = 5
(2x - 2) + (-3x + 8) = 5
-x + 6 = 5
-x = -1
x = 1 ..... (memenuhi syarat 1 ≤ x < 8/3)

• jika x ≥ 8/3
|2x - 2| + |3x - 8| = 5
(2x - 2) + (3x - 8) = 5
5x - 10 = 5
5x = 15
x = 3 ...... (memenuhi syarat x ≥ 8/3)

Jadi HP = {1, 3}



5) |x - 1| + |2x| + |3x + 1| = 6
|x - 1| = (x - 1) jika x ≥ 1
|x - 1| = (-x + 1) jika x < 1
|2x| = (2x) jika x ≥ 0
|2x| = (-2x) jika x < 0
|3x + 1| = (3x + 1) jika x ≥ -1/3
|3x + 1| = (-3x - 1) jika x < -1/3

• jika x < -1/3
|x - 1| + |2x| + |3x + 1| = 6
(-x + 1) + (-2x) + (-3x - 1) = 6
-6x = 6
x = -1 ...... (memenuhi syarat x < -1/3)

• jika -1/3 ≤ x < 0
|x - 1| + |2x| + |3x + 1| = 6
(-x + 1) + (-2x) + (3x + 1) = 6
2 = 6
(salah)

• jika 0 ≤ x < 1
|x - 1| + |2x| + |3x + 1| = 6
(-x + 1) + (2x) + (3x + 1) = 6
4x + 2 = 6
4x = 4
x = 1 ...... (Tidak memenuhi syarat 0 ≤ x < 1)

• jika x ≥ 1
|x - 1| + |2x| + |3x + 1| = 6
(x - 1) + 2x + (3x + 1) = 6
6x = 6
x = 1 ...... (memenuhi syarat x ≥ 1)

Jadi HP = {1, -1}
