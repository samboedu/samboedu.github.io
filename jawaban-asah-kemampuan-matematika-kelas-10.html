5. tolong jawab asah kemampuan diri integral​


1. a Fungsi f(x) nya adalah x³ + 3x² + 5

1. b Fungsi f(x) nya adalah x⁴ - 4x³ + 15x + 7

2. Nilai c nya adalah 3

Pembahasan

Integral adalah lawan dari turunan (diferensial). Integral mempunyai rumus dasar yaitu:

[tex]\int\limits ax^n \, dx = \frac{a}{n+1} x^{n+1} + c[/tex]

Sekarang akan kita bahas satu per satu persoalan tersebut:

1. a. f'(x) = 3x² + 6x dan f(-3) = 5

[tex]\int\limits 3x^2 + 6x \, dx = \frac{3}{2+1} x^{2+1} + \frac{6}{1+1} x^{1+1} + c[/tex]

[tex]\int\limits 3x^2 + 6x \, dx = \frac{3}{3} x^{3} + \frac{6}{2} x^{2} + c[/tex]

[tex]\int\limits 3x^2 + 6x \, dx = x^{3} + 3x^{2} + c[/tex]

f(x) = x³ + 3x² + c

f(-3) = 5

x³ + 3x² + c = 5

(-3)³ + 3(-3)² + c = 5

-27 + 27 + c = 5

0 + c = 5

c = 5

f(x) = x³ + 3x² + 5

1. b. f''(x) = 12x² - 24x, f'(3) = 15 dan f(-2) = 25

[tex]\int\limits 12x^2 - 24x \, dx = \frac{12}{2+1} x^{2+1} - \frac{24}{1+1} x^{1+1} + c[/tex]

[tex]\int\limits 12x^2 - 24x \, dx = \frac{12}{3} x^{3} - \frac{24}{2} x^{2} + c[/tex]

[tex]\int\limits 12x^2 - 24x \, dx = 4x^{3} - 12x^{2} + c[/tex]

f'(x) = 4x³ - 12x² + c

f'(3) = 15

4x³ - 12x² + c = 15

4(3)³ - 12(3)² + c = 15

4(27) - 12(9) + c = 15

108 - 108 + c = 15

0 + c = 15

c = 15

f'(x) = 4x³ - 12x² + 15

[tex]\int\limits 4x^3 - 12x^2 + 15 \, dx = \frac{4}{3+1} x^{3+1} - \frac{12}{2+1} x^{2+1} + 15x + c[/tex]

[tex]\int\limits 4x^3 - 12x^2 + 15 \, dx = \frac{4}{4} x^{4} - \frac{12}{3} x^{3} + 15x + c[/tex]

[tex]\int\limits 4x^3 - 12x^2 + 15 \, dx = x^{4} - 4x^{3} + 15x + c[/tex]

f(x) = x⁴ - 4x³ + 15x + c

f(-2) = 25

x⁴ - 4x³ + 15x + c = 25

(-2)⁴ - 4(-2)³ + 15(-2) + c = 25

16 + 32 - 30 + c = 25

48 - 30 + c = 25

18 + c = 25

c = 25 - 18

c = 7

f(x) = x⁴ - 4x³ + 15x + 7

2. F'(x) = 4x + 4, F(2) = 19, tentukan nilai c!

[tex]\int\limits 4x + 4 \, dx = \frac{4}{1+1} x^{1+1} + 4x + c[/tex]

[tex]\int\limits 4x + 4 \, dx = \frac{4}{2} x^{2} + 4x + c[/tex]

[tex]\int\limits 4x + 4 \, dx = 2x^{2} + 4x + c[/tex]

F(x) = 2x² + 4x + c

F(2) = 19

2x² + 4x + c = 19

2(2)² + 4(2) + c = 19

2(4) + 8 + c = 19

8 + 8 + c = 19

16 + c = 19

c = 19 - 16

c = 3
