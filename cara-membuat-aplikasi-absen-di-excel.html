2. bagaimana cara membuat aplikasi microsoft excel?


Masuk Ms. Excel dan buat 3 buah sheet dengan masing masing nama sheet "Utama, Main, Stock, dan Laba". Untuk sheet utama jangan diisi data apapun namun beri warna background agar lebih menarik.

Pada sheet Main, buat tabel transaksi seperti terlihat pada gambar berikut

Pada cells F7 ketikan =IF(C7="","",VLOOKUP(C7,stock!$B$4:$C$23,2,FALSE))
pada cell G7 ketikan =IF(D7="","",VLOOKUP(C7,stock!$B$4:$F$24,3,FALSE))
pada cells H7 ketikkan =IF(E7="","",VLOOKUP(C7,stock!$B$4:$E$23,4,FALSE))
pada cells I7 ketikkan =IF(D7="","",G7*D7+H7*E7)
pada cells J7 ketikan =IF( E7="","",IF(K7-D7-E7*L7<0,"stock kurang","ada"))
pada cells I20 ketikkan =IF(I7="","",SUM(I7:I19)) dan
pada cell I22 ketikkan =IF(I20="","",I21-I20)

Pada sheets Stock buat database barang seperti berikut
Pada sheet Laba buat tabel seperti pada gambar berikut

Setelah tabel laba terbuat ketikan rumus di cell C3 berikut=IF(main!C7="","",VLOOKUP(main!C7,stock!$B$4:$H$23,7,FALSE)) dan di cells D3 ketikan =IF(main!I7="","",(main!I7-((main!D7)+(main!E7*main!L7))*C3))

12. jelaskan langkah langkah membuat aplikasi menu pada microsoft excel


Langkah untuk membuat menu sederhana pada aplikasi microsoft excel adalah sebagai berikut

Buka lembar kerja excel kitaPada lembar kerja, kita tambahkan beberapa objek dengan menu insertKemudian berikan link kepada objek-objek tersebutBerikan link pada objek kepada sheet lain atau dokumen lain

Panduan di atas merupakan panduan yang paling mudah dalam membuat menu sederhana pada microsoft excel.

Pembahasan

Microsoft excel merupakan software yang digunakan untuk mengolah angka. Dalam menggunakan excel kita juga bisa memunculkan menu-menu yang belum ada secara default ketika menginstall microsoft excel. Cara untuk memunculkan menu tersebut adalah

Masuk ke dalam menu excel optionsPilih menu CustomizePilih menu/toolbar yang akan ditambahkanKemudian klik OK

Dengan memasukkan toolbar-toolbar yang belum ada, maka kita akan bisa mengakses menu-menu tersebut lebih cepat, sehingga waktu pengerjaan akan lebih cepat.

Pelajari lebih lanjut

Demikian pembahasan mnegenai cara untuk memberikan menu-meu pada software excel, kita bisa membuat hyperlink pada objek yang kita sisipkan. Selain itu kita bisa menambahkan menu melalui excel option. Untuk mempelajari lebih lanjut mengenai excel dapat dibaca pada link berikut

13. Jelaskan cara menambahkan border pada aplikasi microsoft excel


Jawaban:

Misalkan di Ms. Excel 2013, caranya:

langkah 1: sorot/ blok sel yg ingin ditambahkan border

langkah 2: klik kanan bagian yg tersorot dan klik Format Cells...

langkah 3: tunggu hingga tampil kotak dialog Format Cells

langkah 4: klik bagian Border & lakukan pengaturan sesuai selera

langkah 5: klik tombol OK

Penjelasan tambahan:

Amatilah lampiran:

→ langkah 1: ditunjukkan tanda panah merah

→ langkah 2: ditunjukkan tanda panah biru

→ langkah 3: kotak dialog Format Cells ditunjukkan tanda panah hijau

→ langkah 4: ditunjukkan tanda panah hitam

→ langkah 5: ditunjukkan tanda panah ungu

Beberapa pengaturan border (pada langkah keempat), antara lain:

Outline : untuk memberi border di bagian luar

Inside : untuk memberi border di bagian dalam
