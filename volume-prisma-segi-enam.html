16. Perhatikan gambar di samping! Gambar di samping menunjukkan prisma segi enam (tak beraturan) yang di dalamnya terdapat prisma segitiga. Hitunglah: a. Volume prisma segi enam, b. Volume bangun ruang.


Bangun prisma adalah bangun ruang yang memiliki alas berbentuk segi-n. Rumus volume nya adalah :

Volume prisma = luas alas x tinggi

Luas alas prisma tergantung dengan bentuk alas nya.

=> Alas prisma jika berbentuk trapesium :

    Luas alas = [tex] \frac {1} {2}[/tex] x jumlah sisi sejajar x tinggi

=> Alas prisma jika berbentuk segitiga :

    Luas alas = [tex] \frac {1} {2}[/tex] x alas x tinggi

Sedangkan untuk rumus luas permukaan prisma adalah :

Luas permukaan prisma = 2 x luas alas + jumlah luas sisi-sisi tegak prisma.

Pembahasan

Volume prisma segienam

Perhatikan gambar terlampir.

Pada gambar terlampir, itu adalah bagian alas prisma segienam. Hitung terlebih dahulu luas alas ABCDEF.  

Luas ABCDEF terdiri dari persegi panjang dan trapesium sama kaki.

Luas persegi panjang ABCF, yaitu :

= panjang x lebar

= 34 cm x 8 cm  

= 272 cm²

Luas trapesium sama kaki, yaitu :

= [tex]\frac{1}{2}[/tex] x jumlah sisi sejajar x tinggi  

= [tex]\frac{1}{2}[/tex] x ( 18 cm + 34 cm ) x 6 cm

= [tex]\frac{1}{2}[/tex] x 52 cm x 6 cm

= [tex]\frac{1}{2 : 2}[/tex] x 52 cm x ( 6 : 2 ) cm

= [tex]\frac{1}{1}[/tex] x 52 cm x 3 cm

= 156 cm²

Luas bangun datar ABCDEF, yaitu :

= luas persegi panjang + luas trapesium sama kaki

= 272 cm² + 156 cm²

= 428 cm²

Volume prisma segienam adalah :

= luas alas x tinggi

= luas ABCDEF x tinggi prisma

= 428 cm² x 20 cm

= 8.560 cm³

Volume Bangun Ruang

Volume prisma segitiga, yaitu :

= luas alas x tinggi

= ( [tex]\frac{1}{2}[/tex] x alas x tinggi ) x tinggi prisma  

= ( [tex]\frac{1}{2}[/tex] x 16 cm x 9 cm ) x 20 cm

= ( [tex]\frac{1}{2 : 2}[/tex] x ( 16 : 2 ) cm x 9 cm ) x 20 cm

= ( [tex]\frac{1}{1}[/tex] x 8 cm x 9 cm ) x 20 cm

= 1.440 cm³

Volume bangun ruang tersebut adalah :

= volume prisma segienam - volume prisma segitiga

= 8.560 cm³ - 1.440 cm³

= 7.120 cm³

Sehingga diperoleh bahwa :

a. volume prisma segienam = 8.560 cm³

b. volume bangun ruang tersebut = 7.120 cm³
