4. Quiz Fikri Yahya .....1. Apa itu Bilangan Berpangkat ?2. Contoh Soal Bilangan berpangkat, jawabannya, dan pengerjaannya​


Jawaban:

bilangan berpangkat merupakan bilangan yang memiliki pangkat,apabila bilangan A adalah bilangan asli,maka bilangan bulat merupakan bilangan pangkatnya.

contohnya A pangkat N.

Penjelasan dengan langkah-langkah:

latihan soal:

5²=5.5=25

30³=30.30.30=27.000

89²=89.89.89=7.921

719³=719.719.719=2.157

Perpangkatan adalah operasi matematika untuk perkalian berulang dengan bilangan yang sama sebanyak pangkatnya.

✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤

[tex]{\boxed{\bold{ \pink{Bentuk \: Umum\: }}}}[/tex]

[tex]\begin{gathered}\text{a}^ \text{n} = \underbrace{\text{a × a × a×...×a}}_{\text{n faktor}} \\ \end{gathered}[/tex]

[tex]{{\bold{ { a \times a \times a... = {a}^{n} \: }}}}[/tex]

Keterangan :

a = basis

n = pangkat

[tex]{\boxed{\bold{ \pink{Syarat \: Bilangan \: Berpangkat\: }}}}[/tex]

[tex]{{\bold{ { {a}^{m} \div {a}^{n} = {a}^{m - n} \: }}}}[/tex]

[tex]{{\bold{ { {a}^{m} \times {a}^{n} = {a}^{m + n} \: }}}}[/tex]

[tex]{{\bold{ {( {a}^{m}) ^{n} = {a}^{m \times n} \: }}}}[/tex]

[tex]{{\bold{ { \sqrt[n]{ {a}^{m} } = a \frac{m}{n} \: }}}}[/tex]

[tex]{{\bold{ { (ab) ^{n} = {a}^{n} {b}^{n} \: }}}}[/tex]

[tex]{{\bold{ {a ^{ - n} = \frac{1}{ {a}^{n} } \: }}}}[/tex]

[tex]{{\bold{ { {a}^{0} = 1 \: }}}}[/tex]

[tex]{{\bold{ { {a}^{1} = a, \: a ≠ 0 \: }}}}[/tex]

a,b,m,n ∈ B

a^n x a^m = a^(n+m)

contohnya :
2^5 x 2^4 = .... ?

pembahasan :
2^5 x 2^4 = 2^(5+4)
                 = 2^9
              
Sifat yang pertama a^m : a^n=a^m-n
Contoh soal dari sifat di atas
1.5^2 : 5^-3=5^-2(-3)=5^-2+3=5^1=5

Sifat yang kedua (a^m)^n=a^m.n
Contoh soal dari sifat di atas
1.(2^3)^2=2^3×2=2^6=64

6. Tolong jawab soal-soal berikut : 1. Bilangan berpangkat itu apa?2. Cara menghitung bilangan berpangkat positif, nol, dan negatif !3. Sifat-sifat operasi pada bilangan berpangkat !


1. Bilangan berpangkat merupakan bilangan berulang dari "a" sebanyak n faktor.

3. sifat-sifat
1) a^m × a ^n = a^m+n
2) a^m : a^n = a^m-n
3) (a^m) ^n = a^m×n
4) a^n × b^n = (ab)^n
5) a^n : b^n = (a/b)^n
6) a^n + a^m = a^n (1+a^m-n) dgn m>n
7) a^n - a^m = a^n (1-a^m-n) dgn m>n
8) a^-n = 1/a^n
9) a^0 = 1

^ itu artinya pangkat. Contoh: 3^2 itu sama aja sama 3²
