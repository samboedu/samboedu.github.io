Ada 4 garis diagonal yang terdiri dari 4 segitiga siku-siku 

Keempat diagonal tersebut kita masukan kedalam rumus pythagoras yaitu c² = a² + b², maka PA² = a² + b²

PD² = b² + c²

  4² = b² + c²

  b² = 4² - c²

PC² = c² + d²

 8² = c² + d²

 c² = 8² - d²

PB² = a² + d²

 7² = a² + d²

 a² = 7² - d²

Subtitusikan

PA² = a² + b²

PA² = (7² - d²) + (4² - c²)

PA² = 7² - d² + 4² - (8² - d²)

PA² = 7² - d² + 4² - 8² + d²

PA² = 7² + 4² - 8² 

PA² = 49 + 16 - 64

PA² = 65 - 64

PA² = 1

PA = √1

PA = 1 cm

Cara cepat :

PA² + PC² = PD² + PB²

PA² + 8² = 4² + 7²

       PA² = 4² + 7² - 8²

       PA² = 16 + 49 - 64

       PA² = 65 - 64

       PA² = 1

       PA = 1 cm

Jadi panjang PA adalah 1 cm
