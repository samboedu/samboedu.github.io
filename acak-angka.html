10. Dari angka-angka 1,2,3 . . .,6 akan dipilih dua angka secara acak. Peluang terpilihnya dua angka faktor dari 15 adalah ...​


Jawaban:

[tex] \frac{1}{5} [/tex]

Penjelasan dengan langkah-langkah:

Dari angka-angka 1, 2, 3, 4, 5, dan 6 yang merupakan faktor dari 15 adalah 1, 3, dan 5.

Banyak cara memilih 2 angka dari 1, 3, dan 5 adalah

[tex] \binom{3}{2} = 3[/tex]

Sedangkan banyak cara memilih 2 angka dari 1, 2, 3, 4, 5, dan 6 adalah

[tex] \binom{6}{2} = 15[/tex]

Sehingga peluang terpilihnya 2 angka dari 1, 2, 3, 4, 5, dan 6 yang merupakan faktor dari 15 adalah

[tex] \frac{3}{15} = \frac{1}{5} [/tex]

var angkaTebakan = Math.floor(Math.random() * 100);

30. Bagaimana peluang terambilnya bilangan bulat positif tiga angka pada pengambilan acak dengan satu atau lebih angka 7?


Jawab:

[tex]1- \frac{648}{900}[/tex] atau 1 - (648/900)

Penjelasan dengan langkah-langkah:

Pertama, perhatikan dari sifat komplemen peluang didapat peluang

p = #{x | x  setidaknya satu angka 7 dan 100≤ x ≤ 999 }/#{100,101, ... ,999 }  

  = 1 - #{x | x  tidak ada angka 7 dan 100 ≤ x ≤ 999 } /#{100,101, ... ,999 }  

  = 1 -  #{x | x  tidak ada angka 7 dan 100 ≤ x ≤ 999 } /(999 - 100 + 1)

  =  1 -  #{x | x  tidak ada angka 7 dan 100 ≤ x ≤ 999 } /900

#{x | x  tidak ada angka 7 dan 100 ≤ x ≤ 999 } dapat dicari dengan memerhatikan

kita bisa memilih 8 digit ketiga {1,2,3,4,5,6,8,9} untuk menghindari memilih angka 7kita bisa memilih 9 digit kedua {0,1,2,3,4,5,6,8,9} untuk menghindari memilih angka 7kita bisa memilih 9 digit pertama {0,1,2,3,4,5,6,8,9} untuk menghindari memilih angka 7

maka berdasarkan aturan perkalian dalam kaidah pencacahan nilai dari

#{x | x  tidak ada angka 7 dan 100 ≤ x ≤ 999 } adalah 8*9*9 = 648.

maka didapat peluangnya adalah

p =  1 -  #{x | x  tidak ada angka 7 dan 100 ≤ x ≤ 999 } /900 = 1 - [tex]\frac{648}{900}[/tex]
