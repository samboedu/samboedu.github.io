Pembahasan

Operasi penjumlahan dan pengurangan pada bentuk aljabar dapat dilakukan dengan ketentuan, yaitu :

pada suku-suku sejenis.mengumpulkan suku-suku sejenis.

Suku-suku sejenis dapat diartikan sebagai suku yang memiliki variabel dan pangkat yang sama. variabel biasanya dilambangkan sebagai a, b, c, ...., z

3] Tentukan hasil pengurangan bentuk aljabar berikut

a) (42n + 35m + 7) − (−50m − 20n + 9)

    Penyelesaian :

    = (42n + 35m + 7) − (−50m − 20n + 9) → perkalian tanda

    = 42n + 35m + 7 + 50m + 20n - 9 → gabungkan suku sejenis

    = 42n + 20n + 35m + 50m + 7 - 9 → Jumlahkan suku sejenis

    = 62n + 85m - 2


b] (5x + 3) − (x − 1) =

   Penyelesaian :

   = (5x + 3) − (x − 1) → Perkalian tanda

   = 5x + 3 - x + 1      → gabungkan suku sejenis

   = 5x - x + 3 + 1      → Jumlahkan suku sejenis

   = 4x + 4


c] (2y + 15z) − (4y − 8)

   Penyelesaian :

   = (2y + 15z) − (4y − 8) → Perkalian tanda

   = 2y + 15z - 4y + 8      → gabungkan suku sejenis

   = 2y - 4y + 15z - 8       → jumlahkan suku sejenis

   = -2y + 15z - 8

---------------------------------------------------------------------------------------------------

4] Tentukan Hasil pengurangan 5z + 3 oleh 2z - 7

Penyelesaian :

 = (5z + 3) - (2z - 7) → Perkalian tanda

 = 5z + 3 - 2z + 7     → gabungkan suku sejenis

 = 5z - 2z + 3 + 7     → jumlahkan suku sejenis

 = 3z + 10

---------------------------------------------------------------------------------------------------

5] Tentukan hasil pengurangan 6x + 4 oleh x - y

Penyelesaian :

= (6x + 4) - (x - y) → perkalian tanda

= 6x + 4 - x + y     → gabungkan suku sejenis

= 6x - x + y + 4     → jumlahkan suku sejenis

= 5x + y + 4

---------------------------------------------------------------------------------------------------

4. jawaban buku paket matematika kelas 8 halaman 228​


Jawab:

Untuk menentukan penyelesaian sistem persamaan linier dua variabel, maka harus mencari nilai dari variabel-variabelnya. Jika persamaan tersusun atas dua buah variabel (misal: x dan y), maka nilai x dan y dapat ditentukan dari dua buah persamaan matematis.

Penjelasan dengan langkah-langkah:

soal aSusun persamaan agar ruas kiri mengandung bilangan konstanta dengan variabel (x dan y) dan ruas kanan hanya tersusun atas bilangan konstanta.

2x - y = 0

3x - 2y = -3

Buat nilai salah satu variabel (x atau y) sama dengan 0. Sebagai contoh, dipilih variabel y. Cara membuat variabel y menjadi bernilai 0 adalah dengan cara menyamakan nilainya.

2x - y = 0 ----- kedua ruas dikali (-2) agar suku y bernilai (+2y)

3x - 2y = -3 ----- kedua ruas dikali (1) agar suku y bernilai tetap (-2y)

Persamaan yang baru menjadi:

-4x + 2y = 0  

3x - 2y = -3

Jumlahkan kedua persamaan:

(-4x+3x) + (2y+(-2y)) = 0 + (-3)

-x + 0 = -3

-x=-3 ---- kedua ruas dikali (-1)  dan diperoleh x=3

mencari nilai y dari salah satu persamaan dengan mensubstitusikan nilai x. Jangan lupa gunakan persamaan awal:

2x - y = 0 ----- 2(3) - y = 0

6 - y = 0 ---- angka 6 pindah ruas ke kanan

-y = -6 ---- kedua ruas dikali dengan (-1) dan diperoleh y = 6

cek jawaban akhir dengan mensubstitusikan nilai x dan y yang diperoleh ke dalam persamaan awal:

2x - y = 0 ----- 2(3) - (6) = 6 - 6 = 0 -------------- COCOK

3x - 2y = -3 ----- 3(3) - 2(6) = 9 - 12 = -3 -------------- COCOK

karena jawaban sudah cocok, maka untuk soal a, penyelesaian sistem persamaannya adalah: x=3 dan y=6.

Soal b hingga d bisa dikerjakan sesuai dengan urutan cara di atas. Berikut ini panduan pengerjaannya:

soal bmenyamakan variabel

-2x + 3y = 7 --- kedua ruas dikali (5) agar suku y menjadi (-10y)

persamaan menjadi:  -10x + 15y = 35

5x + 8y = -2 --- kedua ruas dikali (2)  agar suku y menjadi (+10y)

persamaan menjadi 10x + 16y = -4

Persamaan yang baru kemudian dijumlah sehingga variabel x bernilai = 0

(-10x+10x) + (15y+16y) = 35 + (-4)

(0x) + 31y = 31

31y = 31 ---- kedua ruas dibagi dengan 31 sehingga y=1

Substitusi nilai y=1 ke dalam persamaan awal untuk mencari nilai x:

-2x + 3y = 7

-2x + 3(1) =7

-2x + 3 = 7

-2x = 4  ---- kedua ruas dibagi dengan (-2) sehingga x=-2

Cek jawaban pada kedua persamaan awal:

-2x + 3y = 7 ----- -2(-2)+3(1) = 4+3 = 7  -------------- COCOK

5x + 8y = -2 ----- 5(-2)+8(1) = -10+8 = -2 -------------- COCOK

karena jawaban sudah cocok, maka untuk soal b, penyelesaian sistem persamaannya adalah: x=-2 dan y=1.

soal cmenyamakan variabel dan menyeragamkan ruas kiri serta ruas kanan

3x + 3 = 3y  ----- 3x - 3y = -3 --- kedua ruas dikali (2) agar suku x menjadi (6x)

persamaan menjadi:  6x - 6y = -6

2x - 6y = 2  ----- 2x - 6y = 2 --- kedua ruas dikali (3) agar suku x menjadi (6x)

persamaan menjadi:  6x - 18y = 6

Karena suku x pada kedua persamaan yang baru bernilai positif, maka kedua persamaan dikurangi satu sama lain agar variabel x bernilai = 0

(6x - 6x) + (-6y - (-18y)) = -6 - 6 ------ variabel x =0

-6y + 18y = -12

12y = -12 ---- kedua ruas dibagi dengan (12) sehingga y = (-1)

Substitusi nilai y= (-1) ke dalam persamaan awal untuk mencari nilai x:

3x + 3 = 3y

3x + 3 = 3(-1)

3x = (-3) - 3 = -6

3x = -6 ---- kedua ruas dibagi 3 sehingga x = -2

Cek jawaban pada kedua persamaan awal:

3x - 3y = -3 ----- 3(-2) - 3(-1) = -6 - (-3) = -6 + 3 = -3  -------------- COCOK

2x - 6y=2 ----- 2(-2) - 6(-1) = -4 - (-6) = -4 + 6 = 2 -------------- COCOK

karena jawaban sudah cocok, maka untuk soal c, penyelesaian sistem persamaannya adalah: x = (-2) dan y = (-1).

soal dmenyamakan variabel dan menyeragamkan ruas kiri serta ruas kanan

5x = 4y + 8  -----5x - 4y = 8 --- kedua ruas dikali (3) agar suku x berubah menjadi (15x)

persamaan menjadi:  15x - 12y = 24

3y = 3x - 3  ----- 3x - 3y = 3 --- kedua ruas dikali (5) agar suku x menjadi (15x)

persamaan menjadi:  15x - 15y = 15

Karena suku x pada kedua persamaan yang baru bernilai positif, maka kedua persamaan dikurangi satu sama lain agar variabel x bernilai = 0

(15x - 15x) + (-12y - (-15y)) = 24 - 15 ------ variabel x =0

-12y + 15y = 9

3y = 9 ---- kedua ruas dibagi dengan (3) sehingga y = 3

Substitusi nilai y= 3 ke dalam persamaan awal untuk mencari nilai x:

5x - 4y = 8

5x - 4(3) = 8

5x = 8 + 12 = 20

5x = 20 ---- kedua ruas dibagi 5 sehingga x = 4

Cek jawaban pada kedua persamaan awal:

5x - 4y = 8 ----- 5(4) - 4(3) = 20 - 12 = 8  -------------- COCOK

3x - 3y = 3 ----- 3(4) - 3(3) = 12 - 9 = 3 -------------- COCOK

karena jawaban sudah cocok, maka untuk soal d, penyelesaian sistem persamaannya adalah: x = 4 dan y = 3.Detil Tambahan

Koordinat Cartesius.

Dalam matematika, sistem koordinat Cartesius digunakan untuk menentukan letak setiap titik dalam sebuah bidang dengan menggunakan jarak titik tersebut terhadap dua sumbu, yaitu sumbu x (absis) yang horizontal dan sumbu y (ordinat) yang vertikal, baik ke kanan atau ke kiri dari sumbu y, maupun ke atas atau ke bawah dari sumbu x.

Untuk mendeskripsikan suatu titik tertentu dalam sistem koordinat dua dimensi, nilai x ditulis terlebih dahulu, lalu diikuti dengan nilai y. Dengan demikian, format yang dipakai selalu (x , y) dan urutannya paten (tidak bisa bisa dibalik).

Agar lebih jelasnya, simak pembahasan soal berikut.

Jawaban Buku Paket Matematika hal.34 Kelas VIII.1. Jika pola tersebut terus berlanjut, banyak batang korek api pada susunan ke - 10 adalah 33 batang.2. Pernyataan yang tepat untuk mendapatkan bilangan kedua dari pasangan bilangan pertama pada pola tersebut adalah dikalikan 3 kemudian dikurangi 3.

Ada dua nomor soal pilihan ganda yang terdapat pada Buku Paket Matematika hal.34 Kelas VIII. Nomor 1 adalah tentang pola bilangan yang direpresentasikan dalam bentuk sejumlah korek api, sedangkan nomor 2 adalah tentang hubungan operasi hitung yang terjadi antara 3 pasang bilangan.

Agar lebih jelasnya, kita simak langsung pembahasan soalnya.

PEMBAHASAN :

1. Diketahui beberapa batang korek api disusun dengan pola tertentu. Pola pertama menggunakan 6 batang korek api, pola kedua menggunakan 9 batang korek api dan pola ketiga menggunakan 12 batang korek api. Dari sini kita dapat melihat bahwa susunan tiga pola korek api tersebut membentuk barisan aritmatika yang suku awalnya (U1 atau a) adalah 6. Dari 6 ke 9 memiliki selisih yang sama dengan 9 ke 12, yaitu 3, sehingga beda antar sukunya (b) adalah 3.

Lalu, buatlah rumus Un baru untuk barisan tersebut dengan rumus umum Un barisan aritmatika.

Un = a + (n - 1)b

Un = 6 + (n - 1)3

Un = 6 + 3n - 3

Un = 3n + 3

Jika pola tersebut terus berlanjut, banyak batang korek api pada susunan ke - 10 dapat ditentukan dengan rumus Un yang baru.

Dengan demikian,

Un = 3n + 3

U10 = 3.10 + 3

U10 = 30 + 3

U10 = banyak batang korek api pada susunan ke - 10 = 33.

2. Diketahui 3 pasang bilangan yang terdiri dari bilangan pertama dan bilangan kedua, yaitu (3 , 6) ; (6 , 15) ; (8 , 21). Antara kedua bilangan tersebut ada sebuah operasi bilangan yang tepat sehingga melalui operasi tersebut, bilangan pertama dapat menghasilkan bilangan kedua.

Agar kita dapat menentukan operasi hitung yang tepat, kita akan mencoba satu per satu.

• Ditambah 3

Untuk (3 , 6) >>> 3 + 3 = 6 MEMENUHI

Untuk (6 , 15) >>> 6 + 3 = 9 TIDAK MEMENUHI

Untuk (8 , 21) >>> 8 + 3 = 11 TIDAK MEMENUHI

• Dikalikan 2

Untuk (3 , 6) >>> 3 × 2 = 6 MEMENUHI

Untuk (6 , 15) >>> 6 × 2 = 18 TIDAK MEMENUHI

Untuk (8 , 21) >>> 8 × 2 = 16 TIDAK MEMENUHI

• Dikalikan 2 kemudian ditambah 3

Untuk (3 , 6) >>> 3 × 2 + 3 = 9 TIDAK MEMENUHI

Untuk (6 , 15) >>> 6 × 2 + 3 = 15 MEMENUHI

Untuk (8 , 21) >>> 8 × 2 + 3 = 19 TIDAK MEMENUHI

• Dikalikan 3 kemudian dikurangi 3

Untuk (3 , 6) >>> 3 × 3 - 3 = 6 MEMENUHI

Untuk (6 , 15) >>> 6 × 3 - 3 = 15 MEMENUHI

Untuk (8 , 21) >>> 8 × 3 - 3 = 21 MEMENUHI

Dengan demikian, operasi hitung yang sesuai untuk tiga pasang bilangan tersebut adalah dikalikan 3 kemudian dikurangi 3.
