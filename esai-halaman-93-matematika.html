1. Mtk kelas 8 esai halaman 93 lengkapi tabel berikut


Mtk kelas 8 esai halaman 93 lengkapi tabel berikut

Pada soal ini kita masih membahas soal Lingkaran, yaitu tentang sudut pusat, panjang busur dan Luas Juring

Sudut pusat adalah sudut yang dibentuk oleh dua jari-jari yang berpotongan pada pusat lingkaran

Busur adalah garis lengkung yang menghubungkan dua titik sembarang pada lengkungan lingkaran

Juring adalah daerah di dalam lingkaran yang dibatasi oleh dua jari-jari lingkaran dan busur lingkaran dihadapan sudut pusat yang dibentuk oleh kedua jari-jarinya

Rumus :

Luas Juring = [tex]\frac{\theta}{360}[/tex] L

Busur = [tex]\frac{\theta}{360}[/tex] K

Dimana :

[tex]\theta[/tex] = sudut juring dalam derajat

K = keliling lingkaran (K = 2πr )

L = Luas lingkaran (L = πr² )

Pembahasan :

1. a. Diketahui :

[tex]\theta[/tex] = 90°

r = 7cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex]  x 7

= 44 cm

Busur = [tex]\frac{\theta}{360}[/tex] K

= [tex]\frac{90}{360}[/tex] x 44

= 11 cm

b. Diketahui :

[tex]\theta[/tex] = 60°

r = 21cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex]  x 21

= 132 cm

Busur = [tex]\frac{\theta}{360}[/tex] K

= [tex]\frac{60}{360}[/tex] x 132

= 22 cm

c. Diketahui :

[tex]\theta[/tex] = 120°

Panjang Busur = 88 cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex] K

88 = [tex]\frac{120}{360}[/tex] x K

88 = [tex]\frac{1}{3}[/tex]  x K

K = 264 cm

Keliling = 2πr

264 = 2 x [tex]\frac{22}{7}[/tex]  x r

264 = [tex]\frac{44}{7}[/tex]  x r

r = 42 cm

d. Diketahui :

r = 100 cm

Panjang Busur = 31,4 cm

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

Keliling = 2πr

= 2 x 3,14 x 100

= 628 cm

Busur = [tex]\frac{\theta}{360}[/tex] K

314 = [tex]\frac{\theta}{360}[/tex] x 628

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{314}{628}[/tex]

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{1}{2}[/tex]

[tex]\theta[/tex] = [tex]\frac{360}{2}[/tex]

[tex]\theta[/tex] = 180°

e. Diketahui :

[tex]\theta[/tex] = 72°

Panjang Busur = 1.256 cm

π = 3,14

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex] K

1.256 = [tex]\frac{72}{360}[/tex] x K

1.256 = [tex]\frac{1}{5}[/tex] x K

K = 6.280 cm

Keliling = 2πr

6.280 = 2 x 3,14 x r

6.280 = 6,28r

r = 1000 cm

2. a. Diketahui :

[tex]\theta[/tex] = 100°

r = 6cm

π = 3,14

Ditanya : Luas Juring ?

Dijawab :

L = πr²

L = 3,14 x 6 x 6

L = 113,04 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex] L

= [tex]\frac{100}{360}[/tex] x 113,04

= 31,4 cm²

b. Diketahui :

[tex]\theta[/tex] = 25°

Luas juring = 31,4 cm²

π = 3,14

Ditanya : r ?

Dijawab :

Luas Juring = [tex]\frac{\theta}{360}[/tex] L

31,4 = [tex]\frac{25}{360}[/tex] x L

31,4 = [tex]\frac{5}{72}[/tex] x L

L = 452,16 cm²

L = πr²

452,16 = 3,14 x r²

r² = 144

r = 12 cm

c. Diketahui :

r = 90 cm

Luas juring = 8.478 cm²

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

L = πr²

L = 3,14 x 90 x 90

L = 25.434 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex] L

8.478 = [tex]\frac{\theta}{360}[/tex] x 25.434

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{8.478}{25.434}[/tex]

[tex]\theta[/tex] = 120°

23. jawaban matematika kelas 8 semester 2 kurikulum 2013 halaman 52 uji kompetensi 6 esai no 9


Jawaban matematika kelas 8 semester 2 kurikulum 2013 halaman 52 uji kompetensi 6 esai no 9

Pada soal ini kita masih membahas tentang Teorema Pythagoras

Bunyi Teorema Pythagoras adalah kuadrat sisi miring sama dengan jumlah kuadrat kedua sisi penyikunya

Pembahasan :

Gambar dibawah ini merupakan balok ABCD.EFGH dengan panjang 10 dm, lebar 6 dm dan tinggi 4 dm. Titik PQ berturut-turut merupakan titik tengah AB dan FG. Jika seekor laba-laba berjalan dipermukaan balok dari titik P ke titik Q, tentukan jarak terpendek yang mungkin ditempuh oleh laba-laba!

Diketahui :

panjang = 10 dm

lebar = 6 dm

tinggi = 4 dm

Ditanya :

Jarak PQ ?

Dijawab :

Lihat gambar ilustrasi.

Saya tambahkan titik R di tengah garis BC

Pertama-tama kita cari dahulu panjang PR

AB = 10 dm

AP = PB = 5 dm

BC = 6 dm

BR = CR = 3 dm

PR² = PB² + CR²

PR² = 5² + 3

PR² = 25 + 9

PR² = 34

PR = √34 dm

Kemudian kita cari Panjang PQ

QR = tinggi = 4 dm

PQ² = PR² + QR²

PR² = (√34)² + 4²

PR² = 34 + 16

PR² = 50

PR = √50 = 5√2 dm

Jadi jarak terpendek PQ adalah 5√2 dm

26. jawaban halaman 93 pai kls 7​


Jawaban:

Jawablah pertanyaan berikut ini!

1. Jelaskan kandungan Q.S. ar-Rahman/55: 33!

Surat Ar-Rahman ayat 33 memiliki isi kandungan tentang pentingnya ilmu pengetahuan bagi kehidupan umat manusia. Dengan ilmu pengetahuan, manusia dapat mengetahui benda-benda langit. Dalam ilmu pengetahuan,manusia dapat menjelajahi angkasa raya.Dengan ilmu pengetahuan, manusia mampu menembus sekat-sekat yang selama ini belum terkuak.

2. Jelaskan kandungan Q.S. al-Mujadalah/58: 11!

Surah al-Mujadalah/58 ayat 11 menjelaskan keutamaan orang-orang beriman dan berilmu pengetahuan. Maka ayat ini menegaskan bahwa orang yang beriman dan berilmu pengetahuan akan diangkat derajatnya oleh Allah Swt.

3. Salinlah hukum bacaan Al Syamsiyah dan Al Qamariyah yang terdapat pada dua ayat tersebut!

Hukum alif lam syamsyiah pada surah Ar rahman ayat 33 adalah السَّمَاوَاتِ. Hukum alif lam syamsyiah pada surah Al mujadallah aat 11 adalah الَّذِينَ, اللَّهُ الَّذِينَ , dan وَالَّذِينَ. Hukum alif lam qamariyyah pada surah Ar rahman ayat 33 adalah الْجِنِّ , الْإِنْسِ dan وَالْأَرْضِ. Hukum alif lam qamariya pada surah Al mujadallah aat 11 adalah الْمَجَالِسِ.

4. Q.S. ar-Rahman/55: 33!

Surah Ar-Rahman 33 adalah ayat yang membicarakan mengenai keutamaa dalam menuntut ilmu. Pada ayat tersebut Allah SWT menegaskan bahwa jamaah manusia maupun jin tak akan bisa melintasi langit dan bumi apabila tak memiliki kekuatan. Kekuatan dalam hal ini ditafsirkan ulama sebagai ilmu pengetahuan.

Berikut bunyi dari surah Ar-Rahman ayat 33:

يَا مَعْشَرَ الْجِنِّ وَالْإِنْسِ إِنِ اسْتَطَعْتُمْ أَنْ تَنْفُذُوا مِنْ أَقْطَارِ السَّمَاوَاتِ وَالْأَرْضِ فَانْفُذُوا ۚ لَا تَنْفُذُونَ إِلَّا بِسُلْطَانٍ

Adapun terjemahnnya sebagai berikut:

Hai jamaah jin dan manusia, jika kamu sanggup menembus (melintasi)

30. matematika hal 93 kelas 8 kurikulum 2013


Lengkapi tabel berikut halaman 93 Buku Matematika kelas 8 smp semester 2

Pada bab ini kita masih membahas soal Lingkaran, yaitu tentang sudut pusat, panjang busur dan Luas Juring

Sudut pusat adalah sudut yang dibentuk oleh dua jari-jari yang berpotongan pada pusat lingkaran

Busur adalah garis lengkung yang menghubungkan dua titik sembarang pada lengkungan lingkaran

Juring adalah daerah di dalam lingkaran yang dibatasi oleh dua jari-jari lingkaran dan busur lingkaran dihadapan sudut pusat yang dibentuk oleh kedua jari-jarinya

Rumus :

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

Busur = [tex]\frac{\theta}{360}[/tex]K

Dimana :

[tex]\theta[/tex] = sudut juring dalam derajat

K = keliling lingkaran (K = 2πr )

L = Luas lingkaran (L = πr² )

Pembahasan :

1. a. Diketahui :

[tex]\theta[/tex] = 90°

r = 7cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex] x 7

= 44 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

= [tex]\frac{90}{360}[/tex] x 44

= 11 cm


b. Diketahui :

[tex]\theta[/tex] = 60°

r = 21cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex] x 21

= 132 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

= [tex]\frac{60}{360}[/tex] x 132

= 22 cm


c. Diketahui :

[tex]\theta[/tex] = 120°

Panjang Busur = 88 cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex]K

88 = [tex]\frac{120}{360}[/tex] x K

88 = [tex]\frac{1}{3}[/tex] x K

K = 264 cm

Keliling = 2πr

264 = 2 x [tex]\frac{22}{7}[/tex] x r

264 = [tex]\frac{44}{7}[/tex] x r

r = 42 cm


d. Diketahui :

r = 100 cm

Panjang Busur = 31,4 cm

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

Keliling = 2πr

= 2 x 3,14 x 100

= 628 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

314 = [tex]\frac{\theta}{360}[/tex] x 628

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{314}{628}[/tex]

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{1}{2}[/tex]

[tex]\theta[/tex] = [tex]\frac{360}{2}[/tex]

[tex]\theta[/tex] = 180°


e. Diketahui :

[tex]\theta[/tex] = 72°

Panjang Busur = 1.256 cm

π = 3,14

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex]K

1.256 = [tex]\frac{72}{360}[/tex] x K

1.256 = [tex]\frac{1}{5}[/tex] x K

K = 6.280 cm

Keliling = 2πr

6.280 = 2 x 3,14 x r

6.280 = 6,28r

r = 1000 cm


2. a. Diketahui :

[tex]\theta[/tex] = 100°

r = 6cm

π = 3,14

Ditanya : Luas Juring ?

Dijawab :

L = πr²

L = 3,14 x 6 x 6

L = 113,04 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

= [tex]\frac{100}{360}[/tex] x 113,04

= 31,4 cm²


b. Diketahui :

[tex]\theta[/tex] = 25°

Luas juring = 31,4 cm²

π = 3,14

Ditanya : r ?

Dijawab :

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

31,4 = [tex]\frac{25}{360}[/tex] x L

31,4 = [tex]\frac{5}{72}[/tex] x L

L = 452,16 cm²

L = πr²

452,16 = 3,14 x r²

r² = 144

r = 12 cm


c. Diketahui :

r = 90 cm

Luas juring = 8.478 cm²

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

L = πr²

L = 3,14 x 90 x 90

L = 25.434 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

8.478 = [tex]\frac{\theta}{360}[/tex] x 25.434

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{8.478}{25.434}[/tex]

[tex]\theta[/tex] = 120°
