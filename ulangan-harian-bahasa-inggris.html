2. Ulangan harian bahasa inggris​


Jawaban:

1. in order to

2. in order to

3. so that

4. in order to

5. so that

6. in order to

7. in order to

8. so that

9. in order to

10. so that

Penjelasan:

Kalau ada subjek keterusannya pasti so that, kalau gak ada subjek (jadi cuma ada kata kerja, predikat, objek, keterangan) keterusannya in order to

Contoh so that

I need to work out so that I can lose some weight ("I" //aku// itu termasuk subjek dari kalimat tersebut)

Contoh in order to

Let's work out in order to stay healthy during this pandemic

Untuk penggunaan tanda petik (“...”) pada kalimat tersebut digunakan pada penggalan kata yang diucapkan oleh tokoh. Kalau misalkan pada percakapan, akan menjadi :
