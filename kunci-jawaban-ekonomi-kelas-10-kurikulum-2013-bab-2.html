18. Ada yang bisa Matematika kurikulum 2013 uji kopetensi 2.2 bab 2 kelas 10 /X


Tentukan himpunan penyelesaian persamaan berikut !

Pembahasan :

1) |x - 2| = 6

• x - 2 = 6
=> x = 8

• -(x - 2) = 6
=> -x + 2 = 6
=> -x = 4
=> x = -4

HP = {-4, 8}



2) |3x - 5| = 7

• 3x - 5 = 7
=> 3x = 12
=> x = 4

• -(3x - 5) = 7
=> -3x + 5 = 7
=> -3x = 2
=> x = -2/3

HP = {-2/3, 4}



3) |x| + |x - 5| = 7
|x| = x jika x ≥ 0
|x| = -x jika x < 0
|x - 5| = (x - 5) jika x ≥ 5
|x - 5| = (-x + 5) jika x < 5

• jika x < 0
|x| + |x - 5| = 7
-x + (-x + 5) = 7
-2x = 2
x = -1 ..... (memenuhi syarat x < 0)

• jika 0 ≤ x < 5
|x| + |x - 5| = 7
x + (-x + 5) = 7
5 = 7
(Salah)

• jika x ≥ 5
|x| + |x - 5| = 7
x + (x - 5) = 7
2x = 12
x = 6 ...... (memenuhi syarat x ≥ 5)

Jadi HP = {-1, 6}


4) |2x - 2| + |3x - 8| = 5
|2x - 2| = (2x - 2) jika x ≥ 1
|2x - 2| = (-2x + 2) jika x < 1
|3x - 8| = (3x - 8) jika x ≥ 8/3
|3x - 8| = (-3x + 8) jika x < 8/3

• jika x < 1
|2x - 2| + |3x - 8| = 5
(-2x + 2) + (-3x + 8) = 5
-5x + 10 = 5
-5x = -5
x = 1 ...... (tidak memenuhi syarat x < 1)

• jika 1 ≤ x < 8/3
|2x - 2| + |3x - 8| = 5
(2x - 2) + (-3x + 8) = 5
-x + 6 = 5
-x = -1
x = 1 ..... (memenuhi syarat 1 ≤ x < 8/3)

1. Gelombang bunyi dari suatu sumber memiliki cepat rambat 340 m/s. Jika frekuensi gelombang bunyi adalah 650 Hz, tentukan panjang gelombangnya!

Diketahui

v = 340 m/s

f = 650 Hz

Ditanya

Panjang gelombang, λ

Penyelesaian

v = λf

340 = λ (650)

λ = 340 / 650

λ = 34/65 meter

340 = 8f
