5. gunakanlah buku tabel bilangan akar pangkat tiga dan untuk memeriksa jawabanmu 343= akar pangkat 3 dari 343 = 1000= akar pangkat 3 dari 1000= 2197= akar pangkat 3 dari 2197= 4913= akar pangkat 3 dari 4913= 10648= akar pangkat 3 dari 10648=


Kelas : VII (1 SMP)
Materi : Bilangan Bulat
Kata Kunci : bilangan, akar, pangkat, tiga

Pembahasan :
Akar pangkat tiga merupakan operasi kebalikan (invers) dari operasi pangkat tiga.
 
Untuk menghitung akar pangkat tiga dari bilangan a, dapat ditentukan dengan menggunakan sifat berikut.
∛a = p ⇔ p³ = a,
dengan p bilangan bulat.
(∛a dibaca akar pangkat tiga dari a).

Mari kita lihat soal tersebut.
Tentukan akar pangkat tiga dari
1. 343.
2. 1000.
3. 2197.
4. 4913.
5. 10648.

Jawab :
1. ∛343 = ∛7³ = 7.
2. ∛1000 = ∛10³ = 10.
3. ∛2197 = ∛13³ = 13.
4. ∛4913 = ∛17³ = 17.
5. ∛10648 = ∛22³ = 22.

Semangat!

Kelas : VII (1 SMP)
Materi : Bilangan Bulat
Kata Kunci : bilangan, akar, pangkat, tiga

Pembahasan :
Akar pangkat tiga merupakan operasi kebalikan (invers) dari operasi pangkat tiga.
 
Untuk menghitung akar pangkat tiga dari bilangan a, dapat ditentukan dengan menggunakan sifat berikut.
∛a = p ⇔ p³ = a,
dengan p bilangan bulat.
(∛a dibaca akar pangkat tiga dari a).

Mari kita lihat soal tersebut.

∛27                           ∛27 = ∛3³ = 3;
∛64                           ∛64 = ∛4³ = 4.
