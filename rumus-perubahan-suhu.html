2. Rumus Perubahan Suhu = ...


Rumus perubahan suhu sebagai berikut:

C:celsius

F:Fahrenheit

R:Reamur

K:Kelvin

C:R=R=(4/5)C

C:K=K=C + 273

C:F=F=(9/5)C + 32

F:C=C=5/9(F - 32)

F:R=R=4/9 (F-32)

F:K=K=5/9 (F - 32)+273

R:C=C=(5/4)R

R:K=K=C+273=(5/4)R + 273

R:F=F=(9/4)R+32

K:C=C=K - 273

K:R=R=4/5 (K - 273)

K:F=F=9/5(K - 273) + 32

Jadi itulah Rumus Perubahan Suhu,semoga bermanfaat:)

RUMUS PERPINDAHAN KALOR:

Q = mc ∆ t

keterangan:

Q: banyaknya kalor yang diterima atau dilepas suatu benda.

m: benda yang menerima melepas kalor.

c: kalor jenis zat ( J/ kg°C ).

∆t: perubahan suhu ( °C ).

RUMUS KALOR JENIS:

Q = mc∆T --> c = Q/ m∆T

Untuk keterangannya, sama dengan gambar rumus perpindahan kalor di atas, ya! Jadi, satu rumus tersebut bisa kita gunakan untuk menghitung dua hal, yakni perpindahan kalor dan kalor jenis.

RUMUS KAPASITAS KALOR:

C = mc

keterangan:

C: kapasitas kalor ( J/ kg°C ).

19. Rumus mencari perubahan suhu pada permulaan panjang


yang aku tau hanya pemuaian panjang bukan permulaan panjang
Penambahan panjang = ΔL
Panjang awal = L₀
Panjang akhir = L
Suhu awal = T₀
Suhu akhir = T
α = koefisien muai panjang


Cara Pertama

ΔL = L₀ . α . ΔT

ΔT = ΔL / [L₀ . α]

T - T₀ = ΔL / [L₀ . α]

∴ Suhu akhir T = T₀ + { ΔL / [L₀ . α] }


Cara Kedua

L = L₀.[1 + α.ΔT]

1 + α.ΔT = L / L₀

α.ΔT = [L / L₀ ] - 1

α.ΔT = [L - L₀] / L₀

ΔT = [L - L₀] / [L₀ . α]

T - T₀ = [L - L₀] / [L₀ . α]

∴ Suhu akhir T = T₀ + { [L - L₀] / [L₀ . α] }
