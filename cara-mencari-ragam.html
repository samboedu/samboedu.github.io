12. carilah a) letak data (Q1, Q2, Q3, H, Qd, D3, P24) b) ukuran penyebaran data (simpangan rata-rata, ragam, simpangan baku) cara jawab :a) Q1=1/4(7+1)=2Q1= ......lanjutkan hehe


Data setelah diurutkan dari terkecil :

2 3 3 3 4 4 5

Jumlah data ( n ) = 7

Rumus letak Qᵢ (kuartil) pada data tunggal :

[tex] \boxed{Letak \: \: Q_i = \frac{i(n + 1)}{4}}[/tex]

Maka :

[tex]Letak \: \: Q_1 = \frac{1 \times (7 + 1)}{4} = \frac{8}{4} = 2[/tex]

[tex]\boxed{\boxed{Q_1 = 3}}[/tex]

[tex]Letak \: \: Q_2 = \frac{2 \times (7 + 1)}{4} = \frac{16}{4} = 4[/tex]

[tex]\boxed{\boxed{Q_2 = 3}}[/tex]

[tex]Letak \: \: Q_3 = \frac{3 \times (7 + 1)}{4} = \frac{24}{4} = 6[/tex]

[tex]\boxed{\boxed{Q_3 = 4}}[/tex]

[tex]H : jangkauan \: \: interkuartil[/tex]

[tex]H = Q_3 - Q_1 = 4 - 3 \: \: > > \: \: \boxed{ \boxed{H = 1}}[/tex]

[tex]Q_d :jangkauan \: \: semi \: \: interkuartil[/tex]

[tex]Q_d = \frac{1}{2} \times H = \frac{1}{2} \times 1 \: \: >> \: \: \boxed{ \boxed{Q_d = \frac{1}{2} }}[/tex]

Rumus letak Dᵢ (desil) pada data tunggal :

[tex] \boxed{Letak \: \: D_i = \frac{i(n + 1)}{10}}[/tex]

Maka :

[tex]Letak \: \: D_3 = \frac{3 \times (7 + 1)}{10} = \frac{24}{10} = 2,4[/tex]

[tex]D_3 = X_2 + (0,4 \times (X_3 - X_2))[/tex]

[tex]D_3 = 3 + (0,4 \times (3 - 3)) = 3 + (0,4 \times 0) \: \: >> \: \: \boxed{ \boxed{D_3 = 3}}[/tex]

Rumus letak Pᵢ (persentil) pada data tunggal :

[tex] \boxed{Letak \: \: D_i = \frac{i(n + 1)}{100}}[/tex]

Maka :

[tex]Letak \: \: P_{24} = \frac{24 \times (7 + 1)}{100} = \frac{192}{100} = 1,92[/tex]

[tex]P_{24} = X_1 + (0,92 \times (X_2 - X_1))[/tex]

[tex]P_{24} = 2 + (0,92 \times (3 - 2)) = 2 + (0,92 \times 1) = 2 + 0,92 \: \: > > \: \: \boxed{ \boxed{P_{24} = 2,92}}[/tex]

Rata-rata ( Xᵣ ) :

[tex]X_r = \frac{2 + 3 + 3 + 3 + 4 + 4 + 5}{7} = \frac{24}{7} = 3,4286[/tex]

Rumus Simpangan Rata-rata ( SR ) pada data tunggal :

[tex] \boxed{SR = \frac{1}{n} \: \sum \limits^{n}_{i = 1} | \: X_i \: - \:X_r\: |}[/tex]

Maka :

[tex]SR = \frac{1}{7} \: \sum \limits^{7}_{i = 1} | \: X_i \: - \:3,4286\: |[/tex]

[tex]SR = \frac{1}{7} \: \times \: ( | \:2\: - \:3,4286\: | + | \:3\: - \:3,4286\: | + | \:3\: - \:3,4286\: | + | \:3\: - \:3,4286\: | + | \:4\: - \:3,4286\: | + | \:4\: - \:3,4286\: | + | \:5\: - \:3,4286\: |)[/tex]

[tex]SR = \frac{1}{7} \: \times \: ( 1,4286\: + 0,4286\: + 0,4286\: + \:0,4286\: + \:1,4286\: + \:1,4286\: + \:2,4286\:)[/tex]

[tex]SR = \frac{1}{7} \: \times \: (8,0002)[/tex]

[tex] \boxed{ \boxed{SR = 1,14289}}[/tex]

Rumus Ragam / varians ( S² ) pada data tunggal :

[tex] \boxed{{S}^{2} = \frac{1}{n} \: \sum \limits^{n}_{i = 1} {(X_i \: - \:X_r)}^{2} }[/tex]

Maka :

[tex]{S}^{2} = \frac{1}{7} \: \sum \limits^{7}_{i = 1} \: {(X_i \: - \:3,4286)}^{2} [/tex]

[tex]{S}^{2} = \frac{1}{7} \: \times \: ({(2\: - \:3,4286)}^{2} \: + \:{(3\: - \:3,4286)}^{2} \: + \:{(3\: - \:3,4286)}^{2} \: + \:{(3\: - \:3,4286)}^{2} \: + \:{(4\: - \:3,4286)}^{2} \: + \:{(4\: - \:3,4286)}^{2} \: + \:{(5\: - \:3,4286)}^{2})[/tex]

[tex]{S}^{2} = \frac{1}{7} \: \times \: (2,0409\: + \:0,1837\: + \:0,1837 \: + \:0,1837\: + \:2,0409\: + \:2,0409\: + \:5,8981)[/tex]

[tex]{S}^{2} = \frac{1}{7} \: \times \: (12,5714)[/tex]

[tex] \boxed{ \boxed{{S}^{2} = 1,7959}}[/tex]

Rumus Simpangan Baku / standar deviasi ( S ) pada data tunggal :

[tex] \boxed{S = \sqrt{{S}^{2}} }[/tex]

Maka :

[tex]S = \sqrt{1,7959} [/tex]

Ragam hias (ukir): Talawang

Talawang berasal dari daerah Kalimantan Tengah

Pada umumnya ukiran Talawang bermotif burung Tinggang

Cara membuat Talawang:

1.Membeli kayu Endef ditempat pangkalan kayu. Mengapa bukan kayu Ulin ? Dikarena harga dari kayu Endef lebih murah dari kayu Ulin.

2.Mencetak gambar motif Talawang yang ingin dibuat dengan printer.

3.Hasil print gambar motif Talawang tadi kemudian di gambar di plywood menggunakan pensil/spidol yang nanti akan digunakan sebagai mal.

4.Ukir kayu Endef tadi dengan mal yang sudah dibuat.

5.Setlah diukir, bersihkan dengan amplas.

6.Kemudian dicat dan tunggu sampai kering catnya.

7.Talawang selesai dibuat.
