16. rumus luas permukaan prisma trapesium


Rumus luas permukaan prisma trapesium adalah

LP Prima = ( 2 × La ) + ( Ka × t )LP Trapesium = ( Jumlah sisi sejajar × tinggi trapesium ) + ( Keliling trapesium × Tinggi prisma )LP Trapesium ( sesuai gambar adalah ) =  (( ab + cd ) × bd ) + (( ab + bc + cd + da ) × ae )

Pembahasan

Secara singkat cara mencari luas permukaan trapesium adalah

[tex]\boxed{\text{LP = (Jumlah sisi sejajar * t) + (Ka * T)}}[/tex]

[tex]\boxed{LP = \text{(( ab + cd ) * bd ) + (( ab + bc + cd + da ) * ae )}}[/tex]

Dalam mencari nilai luas permukaan prima trapesium diperlukan mencari

Luas Alas

Didapatkan dari menjumlahkan sisi sejajar dan membaginya dengan 2 serta mengalikkan dengan tinggi

[tex]\boxed{\text{Luas alas = } \frac{\text{Jmlh sisi sejajar * t}}{2}}[/tex]

Keliling Alas

Didapatkan dengan menjumlahkan seluruh sisi trapesium.

Tinggi Prisma

Setelah itu tinggal menggunakan rumus ini

[tex]\boxed{\text{(2 * la)+(ka * t)}}[/tex]

a² + b² = c²

9 + 16 = c²

ac = 5

Selanjutnya, langsung jawab soalnya dengan rumus ini , ( hf = bc = 3cm ), ( ae = bf = 10 cm )

[tex]\boxed{LP =\text{(( ab + cb ) * bd ) + (( ab + bc + cd + da ) * ae )}}[/tex]

LP = (( ab + cd ) * bd ) + (( ab + bc + cd + da ) * ae )

LP = (( 14 + 10 ) * 3 ) + (( 10 + 3 + 14 + 5 ) * 10 )

LP = ( 24 * 3 ) + ( 32 * 10 )

21. diketahui prisma ABFE berbentuk trapesium . luas permukaan prisma adalah ... cm²


Diketahui prisma ABFE berbentuk trapesium. Luas permukaan prisma adalah 107.200 cm². Kemungkinan ada kesalahan satuan pada soal tersebut yaitu AB lebih panjang dari EF jadi tidak mungkin 1 cm > 40 cm, jadi seharusnya panjang AB adalah 1 m, begitu juga dengan BC, BC adalah sisi terpanjang, jadi tidak mungkin panjangnya hanya 3 cm, seharusnya 3 m. Rumus pada prisma:

Volume prisma = luas alas × tinggiLuas permukaan = 2 × luas alas + keliling alas × tinggi

Pembahasan

Diketahui  

Prisma trapesium dengan ukuran

Tinggi prisma = BC = 3 m = 300 cmTinggi trapesium = AE = 80 cm

Sisi yang sejajar pada trapesium

EF = 40 cmAB = 1 m = 100 cm

Ditanyakan

Luas permukaan prisma = … ?

Jawab

Luas alas

= luas trapesium

= ½ × jumlah sisi sejajar × tinggi

= ½ × (AB + EF) × AE

= ½ × (100 cm + 40 cm) × 80 cm

= ½ × (140 cm) × 80 cm

= 5.600 cm²

Kemudian untuk mencari keiling trapesium, kita harus mencari sisi miring dari trapesium tersebut yang dengan rumus Pythagoras. Misal garis tinggi trapesium dari titik F adalah FO, sehingga

FO = AE = 80 cmAO = EF = 40 cmOB = AB - AO = 100 cm - 40 cm = 60 cm

Sisi miring trapesium ABFE adalah

FB = [tex]\sqrt{FO^{2} + OB^{2}}[/tex]

FB = [tex]\sqrt{80^{2} + 60^{2}}[/tex]

FB = [tex]\sqrt{6.400 + 3.600}[/tex]

FB = [tex]\sqrt{10.000}[/tex]

FB = 100

Keliling alas

= keliling trapesium

= FB + EF + AE + AB

= 100 cm + 40 cm + 80 cm + 100 cm

= 320 cm

Luas permukaan prisma tersebut adalah

Lp = 2 × luas alas + keliling alas × tinggi

Lp = 2 × 5.600 cm² + 320 cm × 300 cm

Lp = 11.200 cm² + 96.000 cm²

Lp = 107.200 cm²
