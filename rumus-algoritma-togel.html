16. buatlah algoritma bahasa natural dan pseudocode rumus keliling persegi


Algoritmanya adalah
1. Mulai program
2. Inputkan berapa sisinya
3. Lakukan perhitungan dengan rumus s*s
4. Outputkan hasil dari perhitungan kamu
5. Selesai

Pseudocoode dalam bahasa c++
include<bits/stdc++.h>
using namespace std;
int main()
{
float s;
cin>>s;
cout<<"keliling persegi adalah " <<s*s;
return 0;
}
