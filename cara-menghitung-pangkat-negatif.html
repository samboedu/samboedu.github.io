21. 1.Sederhanakan bentuk 9³ X 2³ X 1³ =2.hitunglah hasil 81 X 3 pangkat negatif 3=Bantu jawab ykk,jangan lupa pake cara


1. Bentuk sederhana dari 9³ X 2³ X 1³ adalah 3⁶ x 2³

2. Hasil dari 81 X 3 pangkat negatif 3 adalah 3

Pembahasan

Pangkat atau eksponen adalah perkalian berulang dari suatu bilangan.

Misalnya :

2³ = 2 x 2 x 2

3⁴ = 3 x 3 x 3 x 3

(¹/₂)⁵ = ¹/₂ x ¹/₂ x ¹/₂ x ¹/₂ x ¹/₂

Untuk menyelesaikan soal di atas, mari kita ingat kembali beberapa sifat bilangan berpangkat berikut :

[tex]\bold{a.~a^0=1,~a\in{R}}[/tex]

[tex]\bold{b.~a^{-1}=\frac{1}{a}}[/tex]

[tex]\bold{c.~a^{-n}=\frac{1}{a^n}}[/tex]

[tex]\bold{d.~a^n~.~a^m = a^{(m + n)}}[/tex]

[tex]\bold{e.~\frac{a^m}{a^n}=a^{(m-n)}}[/tex]

[tex]\bold{f.~a^m~.~b^m = (a.b)^m}[/tex]

[tex]\bold{g.~(a^m)^n = a^{(m\times{n})}}[/tex]

=======================================

Mari kita terapkan pada soal!

1.  9³ x 2³ X 1³ = (3 x 3)³ x 2³ x 1

   9³ x 2³ X 1³ = (3²)³ x 2³ x 1

   9³ x 2³ x 1³ = 3²⁽³⁾ x 2³

   9³ x 2³ x 1³ = 3⁶ x 2³

2.  81 x 3⁻³ = (3 x 3 x 3 x 3) x 3⁻³

    81 x 3⁻³ = 3⁴ x 3⁻³

    81 x 3⁻³ = 3⁴⁻³

    81 x 3⁻¹ = 3¹

    81 x 3⁻¹ = 3

1. Bilangan berpangkat merupakan bilangan berulang dari "a" sebanyak n faktor.

3. sifat-sifat
1) a^m × a ^n = a^m+n
2) a^m : a^n = a^m-n
3) (a^m) ^n = a^m×n
4) a^n × b^n = (ab)^n
5) a^n : b^n = (a/b)^n
6) a^n + a^m = a^n (1+a^m-n) dgn m>n
7) a^n - a^m = a^n (1-a^m-n) dgn m>n
8) a^-n = 1/a^n
9) a^0 = 1
